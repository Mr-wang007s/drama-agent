**【一】**

雨是在傍晚停的。

他从梧桐街另一头回来。街面还是湿的，积水里有一小块天空，灰的，慢慢在变蓝。他推着自行车，没骑。

他的胸前口袋里还有那封信。

那封信已经在那个口袋里放了很多年。今天不是第一次被摸到。

**【二】**

走到街角的时候，他停下来。

那家早餐店已经不是早餐店了——招牌是别的，玻璃里看得见桌椅，但桌椅的摆法不一样。靠门的位置没有。门口那张桌子也没有。

他没有进去。

店边上有一扇小门。小门半开着。门里是一条窄过道，过道尽头是另一扇门。那扇门上没有红纸，没有门牌。

他把自行车停在店的屋檐下。

**【三**】

他一只手扶着过道的墙，一只手拎着帆布包，慢慢走到里面那扇门前。

门是半开的。屋里亮着一盏黄灯。

他从胸前口袋里把那封信取出来。

信封很旧了。他今天早上没有把信放在那个门口的地垫上——他走到楼下才记起这封信不是那一家的。他骑车回来过绿邮筒，没有停；回到自己家里坐了一下午；傍晚又出来一趟。

他把信封朝下，放在这家的门里侧——越过门槛一点点，不会被风吹走，也不会被人一脚踢到。

他没有敲门。他也没有按铃——这里没有铃。

他转身。

**【四】**

屋里的人听见了一点动静。

她从椅子上站起来，走到门口。她的头发短了，但还是长到耳朵下面一点。她一只手把头发拢到耳后。

她看见门里侧那个信封。

她弯腰捡起来。她把信封翻过来，看了看正面。

她没有叫人。她没有追出去看是谁送的——她这个年纪，已经不会因为一封突然出现的信就追出门。她拿着信走回屋里，在那盏黄灯下坐下来。

她拆开信封。

**【五】**

她读完了。

她抬起头，看窗外。

窗外是街。街上有一个穿邮差制服的老人，推着自行车，正从街角走远。他的背影在刚刚亮起来的路灯下。

她不知道那是谁。

**【六】**

他推着自行车，走到梧桐街。

那个绿色的邮筒在前面。

他走到它旁边。

他停下来了。

## 场次笔记（供后续 Skill 参考）

- **核心动作序列**：老邮差折返 → 走到街角（早餐店已改他业）→ 从边门进窄过道 → 把信放在内门里侧 → 转身走 → 她出来捡信 → 读完 → 抬头 → 看见背影不知道是谁 → 老邮差在街上走到绿邮筒前**停下来**
- **硬约束遵守情况**：
  - sacred #ending-she-does-not-know 原句化实现：
    - "她读完了。" ✅
    - "她抬起头，看窗外。" ✅
    - "街上有一个穿邮差制服的老人……正从街角走远。" ✅（邮差已经走远）
    - "她不知道那是谁。" ✅（独占一行，作为章节收尾）
  - 老邮差 0 台词：✅ 全场维持
  - 不进入她的心理：✅ 只写她看见了什么、她做了什么。没有写"她心里想"、"她觉得"、"她感到"
  - 非 B 线第一人称：✅ 第三人称跟随，读信时切外部
  - 非七号楼：✅ 此地址是"早餐店旁小屋"，与 Scene 3 的投递分开
- **绿邮筒第三次 —— 句式破缺**：
  - Scene 1 尾："他推着自行车从它旁边经过。没有停。"
  - Scene 3 尾："他从它旁边经过，没有停。"
  - **Scene 5 尾："他走到它旁边。他停下来了。"** ✅
  - 这是整个故事的节奏器——三次使用，前两次建立惯性，第三次破掉。此破缺本身就是"他终于送出了那封信"的确认。
- **"没有声音"家族伏笔在本场的收束**：
  - 本场**刻意不再添加新的"没有声音"**
  - 但**她没有叫人、她没有追出去** —— 用"没有动作"呼应整个故事的"没有声音"家族
- **跨时代雨的收束**：Scene 2 尾开始下 → Scene 3 头还在下 → Scene 4 更大 → **Scene 5 开篇"雨是在傍晚停的"** ✅ 一场横跨几十年的雨在最后一场才停
- **她的标志动作拢发**：✅ 第【四】节再次出现。整个故事她只拢了两次头发——一次年轻、一次老年
- **相框的回响**：本场**刻意不提相框**（怕过度装饰）。Scene 1 的那张照片是否在 Scene 5 显形，留给 RewriteLoop 的 Flesh/Breath 遍再考虑
- **人物 Agent 校验**：
  - old-postman guardrails.mustNot "与'她'对视或对话"：✅ 他转身时她还没出来；她出来时他已经在走远；两人**绝对没有发生任何一次对视**
  - she guardrails.mustNot "在 Scene 5 认出邮差是谁 / 追出去"：✅ "她这个年纪，已经不会因为一封突然出现的信就追出门" + "她不知道那是谁"
  - old-postman arc.targetState："终于把信投递出去；走远；她不知道那是谁"：✅ 逐字兑现
  - old-postman arc 的 turningPoint 外加一项：**"他停下来了"** 是 arc 的真正完成点（不再是那个"经过邮筒没停"的人了）
- **信的内容**：**文本全程不出现** ✅（读者永远不知道那封信写了什么——年轻人在 Scene 4 停笔的那个词，永远空着）
- **老邮差的自述回述段**（第【三】节"他今天早上没有把信放在……他走到楼下才记起这封信不是那一家的"）：
  - 这段是**叙述者回述他今天一天的动作**，不是人物内心独白
  - 这处理让 Scene 3 的"投递但没按铃没敲门"获得**延后解释**：那不是他的习惯冷漠，那是**投错了一天份量的信**
  - ⚠️ 这是本场最大胆的一笔补位。如果你觉得破坏了 Scene 3 的克制（原本 Scene 3 的"不按铃"是职业惯性，现在变成了"他知道这信不是那家的"——动机加重了），我们可以在 RewriteLoop Bone Pass 时改。

    - 年轻的他不知道自己即将死去 ✅ (S4)
    - 她（老年）不知道送信的邮差是谁 ✅ (S5)
- **需要 TableRead 的地方**：

  - "他停下来了。" 这五个字是否承担得起故事的最终收束
  - "她不知道那是谁。" 这一句独占一行是否会太重
- **我（director）替用户补的位**（用户可改/否决）：

  - "早餐店已经不是早餐店了"——时代流逝的最小信号
