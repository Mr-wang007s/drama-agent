## 二、故事骨架（从 intent 的 coreSkeleton 原样承接）

- **结构**：双线并行，同一人的两个时间点。A 线 = 几十年后的老年邮差；B 线 = 年轻时的死亡瞬间。
- **核心遗憾（B 线动机）**：年轻的他临死前想寄一封信给"没来得及喜欢的人"。
- **收信人**：那个"没来得及喜欢的人"还活着，是街角的老人。
- **结尾时刻**（sacred #ending-she-does-not-know）：她读完信，抬头看窗外，邮差已经走远；她不知道那是谁。
- **贯穿秘密**：听见死人说话，是老邮差一辈子的秘密。
- **身份谜底**：A 线的"他"就是 B 线那个"我"——这个认识在人称切换消失时发生（sacred #person-shift-is-death）。

---

## 三、不设限清单（Producer 阶段）

以下边界是**软的**，写作过程中可随时更新：

- 章节数 / 场次数：暂定 5 场（scene-01 至 scene-05），可增减
- 每场字数：不设硬指标，由场次内在节奏决定
- 是否出现"死人说话"的直接文本：**倾向不出现**（`intentionalDeviations.stylistic[1]` 延迟揭露）
- B 线时代感：未锁定（文艺青年、看书、早餐店——可能是 80-90 年代中国小城，但不必明写）
- A 线时代感：现代（电动车、手机），但刻意保留模糊感

## 四、已在 Scene 1-3 中建立的"默认设定"（后续 Skill 不得擅改）

| 元素 | 值 | 出处 |
|---|---|---|
| **邮筒颜色** | 绿色 | Scene 1 重复 2 次 + Scene 3 再现 1 次 |
| **A 线街道** | 梧桐街（两侧高梧桐） | Scene 1、Scene 3 |
| **A 线收件地址** | 梧桐街尽头 / 七号楼 / 三楼东户 / 铁门两遍绿漆 / 门牌底下有红纸翘角 | Scene 3 |
| **老邮差的投递习惯** | 信封朝下放地垫，不塞门缝，不按铃 | Scene 3 |
| **相框中的照片** | 一条石板路老街 + 一扇门 + 一个邮筒（绿色），没有人 | Scene 1 |
| **"没有声音"基调** | 电动车无声（Scene 1、Scene 3）；早餐店门铃在"陌生男人"进来时没响（Scene 2） | 跨场伏笔 |
| **跨时代雨** | Scene 2 结尾下雨 → Scene 3 开篇"雨从早饭后开始下"（一场雨横跨几十年） | Scene 2/3 |
| **B 线女主角动作** | 一只手把头发拢到耳后 | Scene 2 |
| **信的物理位置** | 老邮差胸前口袋 | Scene 3 |
