# 场景调度手艺（scene-staging）

> 这是 Director 的"选角 + 走位"能力册。回答的是："这个场景该由谁来演？他们怎么分布？谁先开口？"
>
> 本册不是规范，是**手艺**——读完你应该有"导演直觉"而不是"SOP 流程"。

---

## 场景的三种形态

每个场景在你接手时先判断它是哪种形态。不同形态的调度规则完全不同。

### 形态 A · 独幕戏（1 人 + world-manager）

**出现条件**：主角独处、没有对手角色、时间短（≤5 分钟故事时间）

**典型场景**：
- 清晨醒来（林墨场景 1）
- 独自看信/看照片
- 一个人在车上、床上、镜子前

**调度难点**：**没有对手 = 压力全靠世界**。world-manager 此时不是"辅助"，是**唯一施压源**。

**调度策略**：
1. world-manager 每 2 回合注入 1 个**物理事件**（不是心理提示）——光变化 / 声音 / 物品位置 / 其他人的存在（走廊脚步声）
2. 主角的每个回合尽量是**动作 + 极短观察**，不是大段内心——否则退化为独白
3. 结束条件：**物理变化驱动**——主角离开空间（开门出去 / 站起来 / 被电话打断）

**失败陷阱**：
- ❌ 让主角在独幕里"想"大段内心（读者会走神）
- ❌ world-manager 只写环境不施压（氛围文不是戏）

**合法写法**：
- 独幕戏里每 30-50 字必须有一个**可观察事件**（动作/声音/视觉变化）

---

### 形态 B · 对手戏（2 人 + world-manager）

**出现条件**：两人在同一空间，有**不等权力关系**（医生/病人、师兄/师弟、母亲/儿子、导师/学生）

**典型场景**：
- 告别（林墨 × 李医生）
- 探视（林墨 × 母亲）
- 交接（周伯 × 沈砚之）

**调度难点**：**谁先开口、谁主动、谁被动**决定了整场戏的走向。

**调度策略**：
1. **先判断"谁的场"**——这个场景的物理空间是谁的地盘？是谁的地盘就是他的主场（医生在诊室，母亲在会客室她带的东西上，周伯在办公室）
2. **主场者先开口**——主场者通常以问句开场，压力放到客场者身上
3. **客场者的前 3 回合必须是"身体反应"为主**——客场者刚进这个场地，他在观察环境、调整状态，不该立刻就雄辩
4. **情绪反差 = 权力博弈**——弱者越克制压力越大（林墨对母亲的不纠正）

**对手戏节奏模板**（推-拉-转-收）：
- **推**（前 3 回合）：主场者发起压力（问句 / 物品呈现 / 眼神）
- **拉**（中 3-5 回合）：客场者反抗或转移（沉默 / 自嘲 / 无关话题）
- **转**（关键 2-3 回合）：主场者加压或客场者投降/反击——整场的"戏眼"在这里
- **收**（最后 2 回合）：一个物理节点（签字、起身、物品被拿起/放下）收束

**失败陷阱**：
- ❌ 两人节奏同步（都快 or 都慢 = 没有张力）
- ❌ 两人都说话很多（对手戏的高级手法是"其中一人用行动说话"）
- ❌ 强制"50-50 对白分配"（真正的对手戏通常是 70-30 或 80-20）

---

### 形态 C · 群戏（3+ 人 + world-manager）

**出现条件**：3 人以上同台，有**不同利益**的角色在场

**典型场景**：
- 办公室三人（林墨 + 周伯 + 张教授）
- 服务区交接（林墨 + 周伯 + 沈砚之 + 张教授）
- 工地现场（考古队 + 九歌司 + 方士会）

**调度难点**：**谁是主视角、谁是沉默第三方、谁是调停者**——群戏里不是每个人都该说同样多的话。

**调度策略**：
1. **1 主 + 1 副 + N 旁观**原则——任何一场群戏都把角色分为 3 档：
   - **主角**（1 人）：贯穿全场，视角锚点
   - **副角**（1-2 人）：和主角有对话/动作交集
   - **旁观**（N 人）：在场但不说话，用"看""点头""不接话"参与
2. **旁观者是最重要的**——他们的"在场但不出声"本身是压力（EP01 场景 4 张教授的沉默）
3. **每个回合 world-manager 只激活 1-2 个角色**——不能全员都说话，否则场面失控
4. **调度序列**：主 → 副 → 旁观（一个动作）→ 主 → 副 → 旁观（一个动作）…… 旁观每隔 3-4 回合做一个"可观察微动作"即可

**失败陷阱**：
- ❌ 让每个人都说同样多的话（戏散）
- ❌ 旁观者成"透明人"——没有他们的眼神/动作反应
- ❌ 四人以上同时有台词（读者跟不上）

## 视角锚选择

**铁律：每个场景只能有一个主视角**。

### 怎么选主视角

问三个问题：
1. **这个场景谁的信息差最大？**（他最不知道在发生什么 → 读者代入他去"发现"）
2. **这个场景谁的情绪变化最大？**（他的起点和终点差距最大 → 读者跟着他"被改变"）
3. **这个场景谁的决定最重要？**（他的一个选择推动下一场 → 读者等他"做决定"）

三个问题指向同一人 = 主视角确认。
指向不同人 = 要么改场景结构，要么接受这是一场"散点戏"（需要更高水平的调度）。

### 主视角的权限

- 他的**内心独白**可以用自由间接引语嵌入叙事（≤ 20 字独立运用）
- 他的**身体反应**优先被白描
- 其他角色的内心**不能**直接透视——只能通过他的观察间接感知

### 视角切换

一个场景**原则上不切换视角**。除非：
- 出现**双线同时进行**的物理状态（周伯桌下观察林墨的食指 + 林墨自己没察觉）——此时可以短暂切到观察者视角 ≤ 2 段，然后切回主视角
- 主视角**物理离开**场景（开门出去）但场景还要继续——可以切到留下的人

## 进出场节拍

### 进场（角色到达场景的前 30 秒）

**第一个进场动作必须是"调整"**——不是说话。
- ✅ 林墨进周伯办公室：目光在红塔山上停 0.3 秒
- ✅ 李医生开场：左手按在病历右上角（保持六秒）
- ❌ 角色进场立刻说"我来了"——AI 味

**进场的三选一**：
- **他先观察环境**（他扫了一眼房间，物品/人的位置）
- **他先做一个习惯动作**（他放下包 / 他摘眼镜 / 他扶一下眼镜）
- **他先对环境做一个反应**（屋子冷，他把围巾紧了一下）

### 出场（角色离开场景）

**出场必须有一个物理节点**：
- ✅ 起身 → 走到门口 → 回头 → 说一句 → 推门 → 门关上
- ❌ "他走了。"（飞快出场 = 读者没跟上）

**出场的"回头"很关键**——这是本场戏的情感收束点：
- 回头看谁？看的是什么？
- 说一句话还是不说？
- 说了不完整的半句（"那个青铜柜子——"）是本场的余韵种子

## 快速自检（Director 选完角色走位后问自己）

- [ ] 这是独幕/对手/群戏哪一种？（明确形态）
- [ ] 主视角是谁？（选三问答案一致）
- [ ] 主场是谁？（地盘 = 先开口权）
- [ ] 客场角色的前 3 回合是身体反应为主而不是对白？
- [ ] 对手戏的节奏是推-拉-转-收四段式吗？
- [ ] 群戏分了主 1 / 副 1-2 / 旁观 N 吗？
- [ ] 旁观者有可观察的微动作吗（不是透明人）？
- [ ] 进场第一动作是"调整"不是说话？
- [ ] 出场有物理节点 + 一次"回头"吗？

九问全过 → 场景调度合格 → 可以 spawn Team。
