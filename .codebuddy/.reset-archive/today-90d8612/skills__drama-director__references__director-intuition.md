# 导演直觉七问（director-intuition）

> 这是 Director 在**读完 beat-sheet 准备启动 Phase 2** 之前的"开机前自问"。
> 不是流程清单，是直觉训练。读完这 7 个问题你对一个场景的感觉会完全变。
>
> 场景设计是一门可学的手艺。下面七问让你从"照 beat-sheet 执行"升级到"懂这场戏"。

---

## 使用时机

- Phase 1 完成（brief + beat-sheet 已写）
- Phase 1.5 Writing-Coach 8 问预检已过
- **Phase 2 启动 Team 之前**

每个场景问完七问，如果任一问答不上，说明这个场景还不成熟——回去改 beat-sheet，不要硬开 Team。

---

## 第 1 问：这个场景的"门"在哪里？

"门"= **主视角的期待 vs 实际发生**之间的反差点。

好场景的门很清晰：
- 林墨以为今天是普通的一天 → 桌上多了一张纸（门开了）
- 林墨以为母亲会和平常一样 → 她叫他"糕"（门开了）
- 林墨以为能拒绝周伯 → 他的身体先答应了（门开了）

**一个没有"门"的场景是什么样的**：
- 两人见面、聊天、告别（无期待 = 无落差 = 无戏）
- 林墨去吃了一顿饭、回来（没有"不对"的瞬间）

### 实战用法

读 beat-sheet 问自己："这场戏里，主视角走进来时以为会发生什么？实际发生了什么？两者差多远？"

差得越远，戏越好。差得很近 → 回去改 beat-sheet，加一个"不对劲"的触发物。

---

## 第 2 问：谁带着"门"进来？

"门"不是凭空开的，是**某个角色带着它走进场景**。

- 场景 1 的门 = 那张纸。那张纸不是林墨带来的——**谁放在桌上的？** → 这个"谁"（隐形的操盘人）就是场景的隐藏主角
- 场景 3 的门 = 母亲叫"糕"。谁带着"糕"来的？**不是她主动带来的，是她身上的褪色带来的** → 褪色是隐形主角
- 场景 4 的门 = 陈队长录像。谁带来的？**周伯** → 周伯是本场的推动者

**场景的隐藏主角** = 带着"门"来的那个人（或力量）。他不一定是主视角，但他是让戏发生的人。

### 实战用法

给每个场景标出：
- **表面主角**（主视角）
- **隐藏主角**（带门的人）

两者如果是同一个人 → 独幕戏 / 内省戏
两者不同 → 对手戏 / 被动戏

隐藏主角决定了**world-manager 的施压方向**——world-manager 要把戏推向隐藏主角的预设方向。

---

## 第 3 问：谁想把"门"关上？

每个场景里一定有一个角色**不希望这件事发生**——他在物理上/言语上试图阻止"门"的打开。

- 场景 3：母亲不希望发现自己在忘（她的潜意识会启动合理化叙事 → "这个耳钉挺好看的"）
- 场景 4：林墨不希望承认他的身体背叛了理性（他说"陈队长敲到第四下停"而不说自己也敲到第四下停）
- 场景 5：林墨不希望看见沈砚之影子淡一度（他快速把视线收回来，塞进"今天不命名"）

**关门者的抵抗 = 本场戏的张力来源**。

### 实战用法

问自己："这场戏里谁最想这件事没发生过？他会用什么方式否认/掩盖/绕开？"

如果你答不上来 → 这场戏没有反作用力 → 戏会轻得像气球 → 回去补反作用力。

---

## 第 4 问：最没话说的人在干什么？

群戏中**最沉默的那个人**往往承载了最多的戏剧权重。

- 场景 4 张教授全场只说一句话（"那我陪林老师走这一趟"）。他不说的时候在做什么？**他在看林墨的右手**——他和周伯对了一眼都没出声。这个"不出声的目击"是戏的核心证词。
- 场景 3 母亲走后林墨没追她。他不追的时候在做什么？**他坐着没动，咬栗子糕咬了六下才咽**。这个沉默的咀嚼是本场最深的一刀。

**铁律**：沉默者必须有**可观察的行为**，否则他就是透明人。

### 实战用法

列出场景在场的所有角色。对每个**回合数 ≤ 2 次**的角色标出他们在这场戏里的**物理存在痕迹**：
- 他坐在哪里？
- 他手在做什么？
- 他的目光落在哪里？
- 他什么时候调整了一次身体姿态？

这些细节不一定全写进 novel，但你必须知道——否则 spawn Team 时这个角色会退化成背景板。

---

## 第 5 问：镜头离得多近？

不同场景需要不同镜头距离：

- **特写**（< 1 米）：看见手指的细节、耳钉的金属反射、茶杯口的水汽——用于情感爆点
- **中景**（1-3 米）：全身动作、对手戏标准距离——用于对话场景
- **远景**（> 3 米）：空间结构、人与人相对位置——用于群戏开场
- **跨空间**（不在场）：被提到但不出场的人——用于制造"不在场的压力"

**一个场景原则上在两种距离之间切换，不要来回乱切**。

### 实战用法

读 beat-sheet 问自己：

- 这场戏的"戏眼"是什么（第 1 问的门）？戏眼需要什么距离？
- 戏眼前的铺垫用什么距离？

如果戏眼是"母亲的手停在半空" → 戏眼需要特写 → 铺垫可以用中景（两人坐着）+ 戏眼时切近 → 戏眼过后回到中景继续

### 镜头语言的具体写法

- **特写** = 叙事句子聚焦到**单一物件/身体部位**："她的手伸到半空停住了。"
- **中景** = 叙事句子聚焦到**人整体 + 对白**："他合上病历。'林医生。'"
- **远景** = 叙事句子写**空间 + 多人位置**："张教授坐在沙发一角。阿墨冲他颔首，走到桌前的椅子坐下。"

---

## 第 6 问：留白在哪一拍？

一场戏不能全是密度。必须有**至少一个"无事发生"的瞬间**让读者呼吸。

- 场景 2 留白：李医生等到脚步声完全听不见才起身
- 场景 3 留白：林墨一个人咬栗子糕咬了六下才咽
- 场景 4 留白：林墨在门口把手伸进空口袋愣了半秒
- 场景 5 留白：服务区夕阳下两条影子在地上

**留白是戏的"换气"**——没有它，戏会让读者窒息。

### 留白的三种类型

**类型 1 · 物理静止**：角色做了一个无目的的动作（咬糕咬六下 / 把纸翻过来扣在桌上 / 把眼镜又擦一遍）

**类型 2 · 环境独占**：一段只有环境没有人的句子（"门关得很轻。门轴没响。"）

**类型 3 · 对白后的停顿**：对白之后 3-5 个字的停顿描写（"他停了。"）

### 实战用法

每个场景至少设计 1-2 处留白点。标注在 beat-sheet 里。

没有留白 → 戏会变成"事件列表" → 读者被事件压迫但没情感落点。

---

## 第 7 问：这场戏结束后，哪个物品改变了位置？

戏剧结束需要一个**物理锚点**作为收束。问自己：

场景开始时，A 物品在 X 位置；场景结束时，它应该在 Y 位置。

- 场景 1：那张纸从"桌上没见过"到"被扣在桌上"
- 场景 2：李医生的卡片从"抽屉里"到"林墨日记本第二页"
- 场景 3：保温饭盒从"母亲手里"到"椅子上"到"前台"
- 场景 4：聘书从"周伯抽屉"到"林墨签字"；红塔山烟盒从"桌上"到"抽屉最里头"
- 场景 5：林墨从"服务区地面"到"沈砚之副驾"

**物品的移动轨迹就是场景的物理证据**。戏结束时物品必须移动过——否则读者会觉得"什么都没发生"。

### 实战用法

给每场戏画**物品移动图**：

| 物品 | 开始位置 | 结束位置 |
|---|---|---|
| 那张纸 | 桌上（覆盖） | 被扣在桌上 |
| 日记本 | 原地 | 多了一页"有人来" |
| 耳钉 | 林墨左耳 | 林墨左耳（响了一下） |

**至少 2 个物品发生位置变化** = 戏有物理推进。

---

## 七问全过之后

七问全过 = 你对这场戏的"形状"有了直觉。这时候才动手 spawn Team。

Agent 的 prompt 里就可以告诉它：
- 这场戏的"门"在哪（让他知道什么时候该反应）
- 谁是关门者（让他知道怎么抵抗）
- 镜头距离（让他决定给多少身体细节）
- 留白拍（让他知道哪里该沉默）
- 物品轨迹（让他知道该对哪些物品有反应）

七问答案注入到 world-manager 的 prompt 里，world-manager 就会知道怎么施压。

---

## 七问的最终功能

七问**不是让你想得更多**——是让你**知道得更少**。

知道得少 → 你给 Agent 的空间就大 → Agent 才能真正演起来
知道得多 → 你会忍不住替 Agent 做决定 → Team 退化为独白拼图

> 好的导演知道自己不该知道的是什么。
> 坏的导演什么都想掌控，结果什么都演不出来。
