#### 老陈（编剧 · 结构与节奏）

**核心方法论**
- 每集必须有 1 个"反转点"（观众以为 A 实则 B）放在 60%-75% 位置
- 反派/对手的行动必须与主角"反相位"——主角沉默时对手出击，主角行动时对手布局
- 场景转换必须有"情绪落差"——前一场高能后一场必须低开（禁止连续高能场）
- 对话密度建议：独角戏 0% → 双人戏 30-40% → 多人戏 50-60%

#### Q 老师（悬疑小说家 · 诡计与线索）

**核心方法论**
- 悬疑三铁律（已落地 writing-craft.md C 级）
- "线索三明治"：每个关键线索必须被 2 个无关信息夹在中间（防读者过早锁定）
- 钩子分级管理（A = 主线 / B = 支线 / C = 氛围），A 级钩子回收周期 ≤5 集
- 读者与主角的信息差管理：至少 1 处"读者知道但主角不知道"（制造紧张），1 处"主角知道但读者不知道"（制造好奇）

**待执行建议**
- [ ] hooks-ledger.md 每集更新，A 级超 5 集未回收标红
- [ ] EP08 起 beat-sheet 标注每场景的"信息差方向"（读者>主角 / 主角>读者 / 对等）

#### 许教授（文学教授 · 意象与语言）

**核心方法论**
- 意象闭环三阶段：引入（首次出现建立感官锚点）→ 挑战（被颠覆/变形/对立使用）→ 完成（在新语境中重生）
- 核心意象数量：每个故事 6-10 个，每集激活 2-3 个（不能全部同时在场）
- "身体诗学"：每个角色有 1-2 个专属身体动作（替代心理描写的外化标记）
- 语言节奏：长句（≥25字）和短句（≤8字）的交替比建议 1:2 到 1:3

**待执行建议**
- [ ] imagery-ledger.md 每集更新，追踪 8 个核心意象当前阶段
- [ ] EP08 起在 beat-sheet 标注"本场景激活意象"

#### K 教练（写作教练 · 角色与对话）

**核心方法论**
- 角色语言指纹 5 要素：句长 / 口头禅 / 标点习惯 / 比喻来源域 / 沉默模式
- Jaccard < 0.25 是底线，理想值 < 0.15
- "独有称呼"保护：角色 A 对角色 B 的称呼如果是独有的（如"墨弟"），其他角色**绝对不能使用**
- 破防戏五铁律（已落地 writing-craft.md）
- Team 5 tips：
  1. world-manager 每 3 轮必须注入 1 个"计划外微事件"（打断角色的舒适区）
  2. Agent 的第一句回复禁止直接回应上一句（必须有 0.5-2 秒的"身体缓冲"）
  3. 角色碰撞时至少 1 次"说错话"（说完立刻后悔的瞬间）
  4. 沉默 ≥ 对话的 20%（用行动/环境描写填充沉默，不是跳过）
  5. 每个角色每场戏有 1 个"私人议程"（他来这个场景真正想要什么）

**待执行建议**
- [ ] EP08 起 SOUL.yaml 补 `unique_markers` 字段，dialogue-jaccard.js 持续追踪
- [ ] 每集 beat-sheet 标注每个出场角色的"私人议程"
