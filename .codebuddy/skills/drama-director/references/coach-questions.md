# Writing-Coach 预检（Phase 1.5）

> Director 在完成 beat-sheet 后、进入 Phase 2 演绎前的自我审问。
> 8 个问题 + 6 条打回判定。任中 1 条 → 回 Phase 1 修订 beat-sheet。

---

## 8 问清单

### Q1：核心冲突
> 本集的核心冲突是什么？能用一句话说清吗？

判定标准：必须是一句 ≤30 字的陈述，含"谁"+"想要什么"+"被什么阻碍"。
❌ "林墨的内心挣扎" → 太模糊
✅ "林墨想见父亲遗体，但周文渊拦住他不让进太平间" → 有人物+目标+阻碍

### Q2：情绪弧线
> 主视角角色这一集的情绪起点和终点分别是什么？

判定标准：起点和终点必须不同（有变化）。用数字标注（1-10）更佳。
❌ 起点=紧张，终点=紧张 → 没有弧线
✅ 起点=麻木(3)，终点=恐惧(7) → 有清晰上升

### Q3：爽点
> 哪个场景是"爽点"——读者读到会兴奋/紧张/感动？

判定标准：必须能指出具体哪个 beat 是爽点，并说明为什么"爽"。
❌ "整体氛围营造得很好" → 这不是爽点
✅ "Beat 4：林墨第一次反问周文渊——打破了之前 3 集的被动，读者终于等到他发力"

### Q4：钩子经济
> 本集回收了哪些前集钩子？释放了哪些新钩子？

判定标准：回收数 ≥1 且释放数 ≥1。引用 hooks-ledger.md 中的编号。
❌ 回收 0 个 → 读者积累的好奇心得不到满足
✅ 回收 H03(A级)：父亲遗物的第二层含义被揭开；释放 H16(B级)：新角色的来历

### Q5：破防戏判定
> 有破防戏吗？如果有，满足 R1-R5 五铁律吗？

判定标准：如果 beat-sheet 标记了"破防戏/breakdown"，逐条对照 R1-R5。
- R1: 前 200 字过度平静？
- R2: 触发物 ≤5 字？
- R3: 第一个动作不是情绪动作？
- R4: 无关者目击？
- R5: 清醒 ≥ 2× 崩溃？

无破防戏则标注"本集无破防戏，pass"。

### Q6：悬疑三铁律
> 悬疑三铁律（判断错/第三方物理反应/A级钩子）能满足吗？

判定标准（仅适用于 genre 含 mystery/thriller/crime）：
- 判断错 ≥1
- 第三方物理反应 ≥1
- A 级钩子 ≥1（每 2 集释放 1 个新 A 级）

非悬疑类故事标注"非悬疑类，pass"。

### Q7：私人议程
> 每个出场角色的"私人议程"是什么？

判定标准：每个角色必须有"他来这个场景真正想要什么"的一句话描述。
❌ "林墨在场" → 没有议程
✅ "林墨：想从周伯的反应里确认父亲是否还活着（但不能直接问）"

### Q8：翻页欲
> 如果你是读者，读完这集会想立刻看下一集吗？为什么？

判定标准：必须给出具体原因（不能只说"会"）。原因必须与本集结尾的悬念/期待相关。
❌ "会，因为故事还没完" → 废话
✅ "会，因为结尾林墨收到一条短信但没有打开——我想知道是谁发的"

## 6 条打回判定

以下任中 1 条 → beat-sheet 不合格，回 Phase 1 修订：

| 编号 | 条件 | 说明 |
|------|------|------|
| R1 | Q1 说不清一句话核心冲突 | 没有核心冲突的集 = 流水账 |
| R2 | Q3 无法指出明确爽点 | 没有爽点 = 读者走神 |
| R3 | Q4 回收钩子 = 0 | 不回收钩子 = 辜负读者积累的好奇心 |
| R4 | Q6 悬疑三铁律一条都不满足（悬疑类） | 悬疑故事丧失悬疑属性 |
| R5 | Q8 回答为"不会"或模棱两可 | 不想看下集 = 翻页欲归零 |
| R6 | beat-sheet 字数预算总和 < 6000 中文字符 | 低于下限一定写不出足够密度 |

## 迭代约束

- 打回后 Director 修订 beat-sheet 再次自问 8 问
- 最多 2 轮预检（第 3 轮强行通过但 wrap-report 标注"预检勉强通过"）
- 如果第 2 轮仍不过，Director 记录哪些条件未满足及原因，继续推进

## 与 Director 的调用关系

Phase 1.5 由 Director 自行执行（不 spawn 外部 Agent）。
Director 对着 beat-sheet 自问自答 8 问，判定通过后才进入 Phase 2。
如果需要更严格的外部预检，可 spawn writing-coach Skill 的 Task Agent。
