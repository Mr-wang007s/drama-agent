# 人物学 · Characterology

> 给角色注入灵魂的专业方法论。
> 本文件是**表演指导**的核心手册，也是**编剧**写角色、**责编**审人物一致性的事实源。

---

## 为什么需要这份手册

写一个"有血有肉"的角色——这句话在创作圈被说了一百年，但几乎没人能告诉你"血"是什么、"肉"是什么。

本文件给出一套可操作的定义：

- **血** = 创伤链 + 动机结构（人物为什么是现在这样）
- **肉** = 身体标记 + 语言指纹（人物如何被读者感知）
- **灵魂** = 秘密 + 恐惧 + 未说出口的渴望（人物在沉默时的样子）

## 一、Stanislavski 体系的落地——"魔术如果"

### 什么是魔术如果（Magic If）

Stanislavski 在《演员的自我修养》里提出的核心方法：

> 演员不是"假装成"某个角色，而是问自己："**如果我是这个处境下的这个人，我会怎么做？**"

翻译到写作：**作者不是在"设计"角色会怎么做，作者是在"感受"这个角色在此刻的身体冲动。**

### Agent 实操：如何让 Task Agent 进入"魔术如果"

spawn 一个角色 Agent 时，prompt 的前三行决定了这个 Agent 能不能"入戏"：

```
❌ 死板写法：
你是林墨，32岁，刚从病院出来。你要和周文渊对话。

✅ 魔术如果写法：
你是林墨。你刚在病院住了三年。今天早上六点你从病院门口被人接走，
车里坐着周文渊——三年前你父亲的葬礼上就是他推着你走出灵堂的。
现在你坐在他的书房里，他让你坐下。
你感觉到的是什么？你的身体现在是什么状态？你的手放在哪？
```

差别：后者激活了 Agent 的**具身情境**。Agent 不是在"演林墨"，Agent 是**此刻就是**林墨。

### 三个激活开关

每次 spawn 角色 Agent，prompt 必须包含：

1. **此刻的时间** — "现在是凌晨两点" / "你刚经历了..."
2. **此刻的空间** — "你坐在..." / "你站在..." / "你的背靠着..."
3. **此刻的身体** — "你的手..." / "你的呼吸..." / "你的重心..."

缺一个，Agent 就会滑回"上帝视角描述"。

## 二、Uta Hagen 9 问——角色此刻的全景

### 9 个必答问题

Uta Hagen 在《尊重表演艺术》里给演员的自问清单。任何一场戏开始前，角色 Agent 都必须对着这 9 问过一遍：

| 编号 | 问题 | 为什么重要 |
|---|---|---|
| Q1 | **我是谁？** | 不是"名字+年龄"，是"我此刻是什么状态的我" |
| Q2 | **我在哪？** | 地理位置 + 空间感受（安全 / 压抑 / 陌生） |
| Q3 | **什么时间？** | 年月日时 + 这个时间对我意味着什么（深夜=独处, 清晨=未醒, 黄昏=疲惫） |
| Q4 | **我要什么？** | 场景目标：我来这里要得到什么（具体到可达成的东西） |
| Q5 | **为什么要？** | 场景动机的深层理由（连到我的创伤/秘密/渴望） |
| Q6 | **怎么得到？** | 策略：我打算用什么方法（说服/威胁/诱骗/沉默/观察） |
| Q7 | **阻碍是什么？** | 外部阻碍（对手、环境）+ 内部阻碍（我的恐惧、我的懦弱） |
| Q8 | **如果得不到怎么办？** | 失败后的退路（这定义了角色愿意付出多少代价） |
| Q9 | **我的身体在告诉我什么？** | 此刻的身体状态——这是最重要的一条，身体不撒谎 |

### 表演指导的 9 问注入协议

**表演指导 Agent 在 Phase 3 演绎前，必须对每个角色 Agent 发送一条 9 问消息**（可以简化为"你现在要什么？阻碍是什么？身体在告诉你什么？"三问最小集）：

```
[表演指导 → 林墨]
进场前对自己说一遍：
- 我要什么？（周文渊嘴里那句没说完的话）
- 我怎么得到？（装作不在乎，让他自己说）
- 阻碍？（我确实不在乎——三年病院把我磨得只剩沉默）
- 身体？（手指在口袋里摸烟盒的塑封但没撕开）
```

这段 prompt 一旦发出，Agent 的下一个回合就会"有身体"。

## 三、创伤链理论 · Trauma Chain

### 创伤是什么

**创伤不是"过去发生的坏事"，创伤是"现在仍然在身体里响着的过去的坏事"。**

这是本节最重要的一句话。创伤的核心特征：

1. **不受意志控制** — 你不想想起，但某个气味/声音/光线会自动把你拽回去
2. **身体先反应** — 大脑还没处理，心跳/呼吸/出汗先动
3. **具体而非抽象** — 不是"我怕被抛弃"，是"我怕凌晨三点的电话铃声"

### 创伤三型

根据 Judith Herman《创伤与复原》的分类，写作中实用的三型：

| 类型 | 核心伤口 | 典型触发 | 身体标记 |
|---|---|---|---|
| **丧失型**（Loss） | 重要之人/物的消失 | 遗物 / 相似的背影 / 特定日期 | 呼吸变浅 / 指尖发冷 / 视线焦点失焦 |
| **背叛型**（Betrayal） | 被信任的人所伤 | 那个人的声音 / 再次的承诺 | 胸口发紧 / 下意识后退半步 / 笑容僵在脸上 |
| **羞辱型**（Humiliation） | 在他人面前的尊严被摧毁 | 类似的人多场景 / 被凝视 | 耳朵发烫 / 低头 / 手藏起来 / 声音变小 |

### 创伤链的结构

创伤不是单点。**创伤是一串因果链**：

```
原始事件 → 身体印记 → 避免行为 → 替代反应 → 二次伤害
```

样例（林墨）：

```
父亲死于方士会的审讯室（原始事件）
  ↓
林墨此后一听到金属门锁咔哒声会心率骤升（身体印记）
  ↓
他开始拒绝进入任何有金属门的房间（避免行为）
  ↓
他在病院用"配合治疗"换取被安排在木门病房（替代反应）
  ↓
但这让他三年没见过母亲——她只能隔着金属门探视（二次伤害）
```

写角色之前，**每个 S 级和 A 级角色必须有至少一条完整的 5 段式创伤链**写在 SOUL.yaml 的 trauma 字段里。

### Agent 实操：创伤触发的自动化

表演指导在场景中注入事件时，应**精准触发**角色的创伤链的"身体印记"环节：

```
❌ 作者替 Agent 感受：
[场景描述] 林墨听到金属门的声音，感到恐惧。

✅ 触发创伤链让 Agent 自己反应：
[表演指导 → 世界管家] 请注入事件：隔壁房间传来金属柜门关上的 "咔哒" 声。
[世界管家 → 全场] 隔壁房间传来一声"咔哒"。很轻，但很清楚。
[林墨的下个回合] 他的手指在桌面上停了一下，又继续搓指节——他没抬头。

↑ Agent 基于自己的创伤链自动产生了身体反应，作者不需要写"他感到恐惧"。
```

## 四、动机三层——表层欲望 / 深层需求 / 无意识驱力

### 模型

每个角色的每个场景，同时运行三层动机：

```
     表层欲望（Want）        ← 角色嘴里说的 / 行动上追求的
         ↕
     深层需求（Need）        ← 角色内心真正想要的（可能自己都不清楚）
         ↕
     无意识驱力（Drive）     ← 角色的创伤决定的"默认反应模式"
```

### 样例（林墨 · EP01 见周伯）

```
表层欲望：想从周伯那里听到父亲最后一句话是什么
  ↕
深层需求：想被一个"父亲形象"承认他三年没垮
  ↕
无意识驱力：一见到权威男性就本能地保持 "我不需要你" 的距离感
    （父亲死后的身体默认）
```

**好戏的标志**：三层动机互相矛盾。

- 林墨嘴里问"我爸最后说了什么"（表层）
- 但心里想要的是周伯一句"你三年没垮，不容易"（深层）
- 而他的身体在下意识地往椅背上靠、不给周伯靠近的机会（无意识驱力）

结果：周伯一旦试图靠近说安慰的话，林墨会**本能地反击**——哪怕他"深层需求"正是这个。

**这就是有灵魂的人物。**

### 编剧实操：三层动机标注法

编剧在 beat-sheet v3 中，每个出场角色的**每个场景**必须标注三层动机：

```yaml
scene_1:
  agents_motivation:
    林墨:
      want: "听周伯说出父亲最后一句话"
      need: "被周伯以父辈身份承认"
      drive: "对权威男性的本能反抗"
    周文渊:
      want: "判断林墨是否被病院重新塑造了"
      need: "确认自己当年把林墨送进病院是对的"
      drive: "作为幸存者的愧疚驱使的过度控制"
```

这 6 行，是整场戏的引擎。

## 五、身体诗学——让身体替心理说话

### 核心原则

**不写心理活动，写身体动作。**

这不是文风建议，是**定律**。读者通过身体感知人物，不是通过心理描写。

### 对比样本

```
❌ 心理描写型（AI 味重灾区）：
  林墨感到一种说不清的焦虑。他知道自己不应该这样，但他无法控制。
  他想起了父亲死的那天，心里涌起一股复杂的情绪。

✅ 身体动作型：
  林墨把烟盒在手里转了两圈，没撕开。他把它放回口袋，又拿出来。
  三秒后他发现自己的右手在抖。他把手插进口袋。烟盒卡在他的虎口上。
```

差别：

- 坏版本 90% 是"他知道/他感到/心里涌起"——**读者没有任何画面**
- 好版本 100% 是可以被摄像机拍下来的**动作**——**读者自己产生情绪**

### 身体标记库——30 种经典替代

每个角色在 SOUL.yaml 应当有 1-2 个**专属身体标记**。以下是行业常用的 30 种（按情绪分组）：

**紧张 / 压力**
1. 搓大拇指指节
2. 把笔在指间转（转一圈停半秒再转一圈）
3. 反复咽口水但不明显
4. 脚尖轻点地面
5. 把杯子里的水晃一下不喝
6. 嘴唇抿成一条线然后松开

**愤怒 / 克制**
7. 下颌骨一紧（明显的咀嚼肌鼓起）
8. 肩膀往下压半寸（不是往上耸）
9. 食指在桌上敲一下就停
10. 眼睛眨得比平常慢
11. 鼻翼轻微扩张
12. 说话前舌头抵一下上颚

**回避 / 羞耻**
13. 整理袖口（即使袖口是好的）
14. 低头看鞋尖
15. 把头发别到耳后（女性高频）
16. 手放在桌下摸膝盖
17. 笑的时候嘴动了但眼睛没动
18. 把对方递过来的东西放得比正常远半寸

**悲伤 / 哀痛**
19. 深呼吸但吸得太快（像抽气）
20. 看一个点看超过 3 秒
21. 不自觉地抚摸一件物品（项链 / 戒指 / 手表带）
22. 说话声音比平常小 20%
23. 点头的次数比必要的多
24. 手指收拢成半握拳不松开

**专注 / 决断**
25. 身体前倾半寸
26. 呼吸变深变慢（肩胛骨放下）
27. 目光聚焦在对方的眉心（不是眼睛）
28. 手掌贴桌面（不是握拳）
29. 说话节奏放慢但音量不变
30. 整理一下坐姿后开口

### 身体标记的独有性

**重要**：每个角色的身体标记**必须独有**。

```
❌ 错误（两个角色共享身体标记）：
  林墨搓大拇指指节。
  周文渊也搓大拇指指节。

✅ 正确（各自独有）：
  林墨搓大拇指指节。
  周文渊点一支烟但不吸，让它自己烧。
```

责编在 Phase 4 必须对照 SOUL.yaml 检查身体标记是否混用。

## 六、秘密、恐惧、渴望——人物的第三维

### 三个字段（写入 SOUL.yaml）

除了表层的身份（name/age/job），每个 S/A 级角色必须定义：

- **secret**：这个人从不告诉任何人的事（读者在卷内不能知道）
- **fear**：这个人最怕发生的事
- **longing**：这个人嘴上不承认但每天都在想的东西

### 样例

```yaml
# 林墨的第三维
secret: "他在病院第二年主动断了一次药——不是想死，是想看看清醒着的自己是什么感觉。那一周他听见了父亲的声音。他没告诉任何人。"
fear: "他怕自己这三年其实已经疯了，所有人都在陪他演戏。"
longing: "他想听一次有人说'你父亲是个好人'——但他知道这句话永远不会来。"
```

**这三条定义了角色在沉默时的样子。**

### 写作中如何使用第三维

**三维字段永远不能被角色说出口**。它们通过以下方式**泄漏**给读者：

1. **身体的背叛**——角色说 A 但身体在做 B
2. **场景的选择**——角色去某个地方/选某个物品的非理性理由
3. **语言的停顿**——某个词角色不敢接下去
4. **对他人的过度反应**——对方无心的一句话触动了角色的 secret/fear

### 破防戏的本质

所谓"破防戏" = **第三维突然压过第一维**的瞬间。

角色的表层（身份+表层欲望）维持不住了，secret/fear/longing 从地底涌出。

关于破防戏的具体写法（R1-R5 五铁律），见 `craft/prose.md`。

## 七、SOUL v4.0 与本章的映射

SOUL v4.0 的三层结构完全对应本章理论：

| SOUL 字段 | 对应本章 | 用途 |
|---|---|---|
| `identity.*` | 第一层·身份 | Uta Hagen Q1 "我是谁"的表层答案 |
| `psychology.ocean` | OCEAN 大五人格 | 约束角色的默认反应模式（无意识驱力的基底） |
| `psychology.trauma` | 创伤链 | 第三节所述创伤三型+五段式因果链 |
| `psychology.motivation` | 动机三层 | Want / Need / Drive 的根字段 |
| `psychology.secret` | 第三维·secret | 卷内保密，通过身体泄漏 |
| `performance.voice` | 语言指纹 | 详见 dialogue.md |
| `performance.tics` | 身体标记 | 本章第五节的 30 种选 1-2 |
| `performance.body` | 身体默认姿态 | 静止时的身体（坐姿/站姿/呼吸） |

### 校验清单

编剧在 Phase 2 拿到角色档案后，应快速过一遍：

- [ ] 创伤链是否完整 5 段（原始事件 → 身体印记 → 避免行为 → 替代反应 → 二次伤害）
- [ ] 三层动机是否明确 Want/Need/Drive
- [ ] 身体标记是否与其他角色不重复
- [ ] secret/fear/longing 是否够具体（"具体"指能被某个事件触发）

少任意一条 → 回 drama-world 补全角色档案再演绎。

## 八、火影案例——鸣人 / 佐助 / 卡卡西的三维分析

本节用火影疾风传三位核心人物做示范，让抽象理论落地。

### 鸣人（主角）

```
创伤三型：丧失型 + 羞辱型（童年被村子视为怪物）
创伤链：
  原始事件：出生即失去父母，被村子当九尾容器隔离
  身体印记：听到"怪物"/"妖狐"二字瞳孔会先扩再缩（非愤怒，是早年应激）
  避免行为：从不主动谈父母
  替代反应：用"我要当火影"这个口号吸引所有注意力
  二次伤害：越吵闹越孤独，因为吵闹是他唯一会的求救方式

动机三层：
  want：当火影（表层）
  need：被村子当成"家人"而不是"容器"
  drive：对"被无视"的本能抗拒——一旦被冷落会立刻提高音量

身体标记：挠后脑勺（尴尬时）+ 拳头在胸前砸一下（决心时）
第三维：
  secret：他十二岁之前偷偷数过自己对着镜子笑了多少次——因为他需要练习"看起来不难过"
  fear：最怕的不是死，是"醒来发现村子里所有人又回到把他当怪物的那个样子"
  longing：渴望有人叫他"儿子"
```

### 佐助（反角化主角）

```
创伤三型：丧失型 + 背叛型（全族被哥哥灭了）
创伤链：
  原始事件：七岁那晚回家看见全家尸体，哥哥告诉他"你还不值得我杀"
  身体印记：闻到特定花香（灭族之夜院子里开的那种花）会瞬间僵住
  避免行为：拒绝一切温情表达
  替代反应：把所有关系转化为"对我有用/没用"的工具化评估
  二次伤害：越工具化越孤独，越孤独越需要工具化——陷入循环

动机三层：
  want：杀哥哥（表层）
  need：证明"被家人选择"不是偶然（他是被留下的那个）
  drive：一切关系必须在对方伤害自己之前先斩断

身体标记：手背在身后握剑柄（思考时）+ 从侧面看人（不正面对视）
第三维：
  secret：他偷偷留着哥哥小时候给他的一块零食包装纸
  fear：怕杀了哥哥之后发现自己变得一无所有
  longing：想被鸣人"抓住"——哪怕一次他也好
```

### 卡卡西（导师）

```
创伤三型：丧失型 × 3（父亲自杀 + 队友带土死 + 队友琳死在自己手上）
创伤链：
  原始事件：琳主动扑向自己的雷切，死在他手上
  身体印记：雨天不撑伞（那天下雨，他撑着伞走了十分钟才意识到伞下面只有自己）
  避免行为：永远迟到（让自己始终处于"在路上"的状态，不真正抵达）
  替代反应：面具戴到极致——假装轻浮
  二次伤害：装得太久，连自己都不确定还有没有"面具下的那个人"

动机三层：
  want：保护三代目给他的小队
  need：证明自己有资格活着
  drive：在任何亲密关系靠近时本能地撤退一步

身体标记：看书不翻页（假装看其实在想事）+ 单手插兜（另一只手要随时能抓写轮眼）
第三维：
  secret：他每年琳的忌日会去同一个拉面摊——不吃，只坐半小时
  fear：怕徒弟们把他当"英雄"（他知道自己不配）
  longing：想听一次父亲说"对不起"
```

### 使用方式

Director 在 Phase 1 选角时，应对照本样本检查候选角色的 SOUL.yaml：

- 如果一个角色的 trauma / motivation / secret 字段**抽象程度比卡卡西还低**（比如只写"经历过战争"而没有具体场景锚点），退回 drama-world 补档。
- 标准：**所有字段必须能被摄像机/气味/声音触发**。

## 九、常见陷阱

### 陷阱 1：用"情感形容词"代替身体

```
❌ 他感到悲伤。
✅ 他端起茶杯，手腕停了一下才送到嘴边——茶已经凉了。
```

### 陷阱 2：动机单层化

```
❌ 林墨想见父亲最后一面。（只有表层）
✅ 林墨想见父亲最后一面（want）
    ——但他真正想要的是父亲的脸能告诉他"你值得"（need）
    ——而他的身体在下意识地不让自己哭（drive：不能在陌生人面前软弱）
```

### 陷阱 3：身体标记混用

两个角色都"搓大拇指指节" / 都"点烟不抽" —— **角色失去辨识度**。

责编发现混用时的处理：
- 保留主角的身体标记不变
- 给配角重新分配一个（从 30 种库里选）
- 在 SOUL.yaml 里记录"避免冲突"字段

### 陷阱 4：Agent 被作者替写心理

spawn Agent 时 prompt 里出现以下关键词 = 作者越权：

```
❌ 你感到 / 你觉得 / 你意识到 / 你心里涌起
✅ 你的身体是什么状态 / 你现在想做什么动作 / 你的手/脚/呼吸/视线
```

表演指导发现这种越权 prompt 时应立即改写。

### 陷阱 5：创伤过载

**不是所有角色都需要 S 级强度的创伤链**。B/C 级角色的创伤可以简化为一句话锚定：

```yaml
# B 级角色样例（够用了）
trauma_lite: "他妹妹三年前失踪——案子没破。他不提，但每年四月会请一周假。"
```

关键是"能被触发"——不是"写得深"。

## 附录：表演指导的 Spawn Prompt 模板

```
你是表演指导。你加载了 characterology.md 的全部内容。

本集演绎前，你要做三件事：

1. 对每个出场角色发送「9 问简化版」消息：
   "进场前对自己说：
   - 我要什么？（场景目标，具体）
   - 我怎么得到？（策略）
   - 阻碍？（外部+内部）
   - 身体？（此刻的手/呼吸/重心）"

2. 对世界管家发送「创伤触发建议」：
   "本场戏可以在第 X 轮注入 [某个物品/声音/气味]，
   这会触发 [某角色] 的创伤链第 Y 段（身体印记）。"

3. 在演绎过程中监控低质信号：
   - 角色使用了"感到/觉得/心里"类心理词 → 介入改写
   - 角色的身体标记与他人混用 → 介入修正
   - 角色的反应没有连到他的创伤/动机 → 介入提醒

你不直接写内容。你只给角色 Agent 发"激活 prompt"。
```

---

> 人物学的终点不是"让角色立起来"，而是"让角色在作者不注意的时候自己走路"。
> 当你发现 Agent 开始做出连你都没预料到的动作——恭喜，灵魂已经注入了。
