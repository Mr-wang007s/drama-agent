# 对话学 · Dialogue Craft

> 对话是角色之间的权力博弈。
> 本文件是**表演指导**引导 Agent 说话、**文学顾问**润色台词、**责编**审语言指纹的共用手册。
> 核心命题：**好的对话不是"交换信息"，是"相互试探"**。每一句台词都在做"我对你说"之外的事。

---

## 一、对话的本质

### 对话的两个维度

每一句台词同时在做两件事：

```
表层：字面意思（说了什么）
深层：潜台词（通过说，做了什么）
```

**"做了什么" 比 "说了什么" 重要 100 倍。**

### 样例对比

**A 类（只有表层）：**

```
林墨：你父亲最近还好吗？
周伯：他最近不太好，血压有点高。
林墨：让他多休息。
周伯：会的。
```

这是交换信息。**不是戏**。

**B 类（有深层）：**

```
林墨：你父亲最近还好吗？       ← 表层：关心 / 深层：我知道你父亲的秘密
周伯：他最近不太好，血压有点高。 ← 表层：回答 / 深层：你别来试探
林墨：让他多休息。             ← 表层：关心 / 深层：我知道你在防
周伯：会的。                   ← 表层：应承 / 深层：对话结束
```

同样的字，不同的戏。**B 类有潜台词，A 类没有。**

## 二、潜台词 7 层

### 7 层模型

每句台词的潜台词可以解析为 7 个可能的层：

| 层 | 功能 | 示例 |
|---|---|---|
| L1 · 字面 | 单纯信息交换 | "几点了？" → "三点。" |
| L2 · 情绪 | 表达情绪 | "几点了？"（语气不耐烦） |
| L3 · 关系 | 确认/试探关系 | "几点了？"（我们还要聊多久） |
| L4 · 权力 | 谁在控制对话 | "几点了？"（我可以随时终止） |
| L5 · 欲望 | 表达 Want | "几点了？"（我想快点离开） |
| L6 · 恐惧 | 表达 Fear | "几点了？"（我怕错过那班车） |
| L7 · 秘密 | 无意间泄漏 secret | "几点了？"（她每天三点打来电话） |

### 一句话可以同时有 3 层以上

```
"几点了？"
  L1：他想知道时间
  L3：他在暗示对话太久
  L4：他开始收回对话主动权
  L6：他怕错过某个时机
```

**好的对话每句至少有 2 层**。3 层以上 = 精彩对话。

### 潜台词的创作方法

写台词前，先问自己三个问题：

1. **这个角色在这一刻真正想要什么？**（连到 Want/Need）
2. **他为什么不直接说？**（这就是潜台词的起点）
3. **他选择说这句话，是在做什么？**（不是表达，是"做"）

### 实操样本

**原台词（无潜台词）：**

```
林墨：我想看我父亲的遗物。
周伯：遗物都在保险柜里。
林墨：我能看吗？
周伯：可以，但要等几天。
```

**改写（加潜台词）：**

```
林墨：我在想，父亲的茶杯还在不在。          ← 不说"遗物"，说一个具体小物件（试探）
周伯：你要找什么？                          ← 反问（转被动为主动）
林墨：一件小事。                            ← 回避具体（不让对方准备）
周伯：东西都在保险柜。                      ← 泛答（告诉他"别想单独要一件"）
林墨：我没说要拿。                          ← 反击（把"保险柜"这件事翻篇）
周伯：那你想说什么？                        ← 逼近（权力翻转）
  （此处林墨应该沉默 2 秒，然后转话题）
```

差别：每句都在**做**一件事（试探、反问、回避、泛答、反击、逼近），而不仅仅是**说**一件事。

## 三、说错话艺术——角色的"自我暴露"

### 什么是"说错话"

**角色说完一句话，立刻在心里后悔**——但嘴上不纠正。

这个 0.5-2 秒的后悔，是戏剧最浓缩的瞬间之一。

### 为什么有用

说错话暴露了角色的：

- 真实情绪（压抑不住）
- 真实关系（亲密度超出角色"官方人设"）
- 真实恐惧（触到了 fear）
- 真实渴望（longing 短暂压过理性）

**这是读者看到"真实的角色"的唯一窗口。**

### 样例

```
周伯：你这几年在病院，吃了不少苦吧。
林墨：没什么，比我爸那时候好多了。
（短暂的停顿）
林墨：——我是说，比从前的条件好多了。

↑ "我爸那时候好多了" 这句话
   - 泄漏了他一直在把自己的遭遇和父亲比（secret）
   - 暴露了他对父亲的复杂情感（既怨又想亲近）
   - 然后他立刻改口 → 说明他知道自己刚才说漏了
```

这个场景里，林墨的真实状态在 1.5 秒内对读者显形，然后又被收回去。

### 编剧/表演指导实操

**每场戏至少 1 次"说错话"**（K 教练 5 tips 之一）。

beat-sheet v3 中应标注：

```yaml
scene_4:
  slip_of_tongue:
    character: 林墨
    wrong_line: "比我爸那时候好多了"
    correction: "我是说，比从前的条件好多了"
    reveals: "他一直在和父亲比较自己的遭遇"
```

### 说错话的三种形式

**形式 1：词语型**
- 说出了一个本不该用的词（太亲密/太暴露/太专业）
- 样例："我也这么想" → "老师也这么想"（暴露了他内心把对方当权威）

**形式 2：语气型**
- 语气泄漏了压抑的情绪
- 样例：一句客套话里的微微颤抖 / 一句陈述句末尾的上扬

**形式 3：沉默型**
- 在该说话时没说 / 在该停顿时继续了
- 样例：被问到父亲时，犹豫了 2 秒才回答 → 这 2 秒是戏

## 四、沉默四用——沉默比说话更有力

### 沉默的四种功能

**用法 1：拒绝沉默**
- 角色选择不回答
- 表达："我不想和你谈这个"

**用法 2：选择沉默**
- 角色有话在嘴边但咽下
- 表达："我说了你也不信 / 我不能让你知道"

**用法 3：情感沉默**
- 角色被情绪击中无法说话
- 表达：崩溃前的静止 / 震惊的呆住

**用法 4：权力沉默**
- 强者用沉默施压
- 表达："我在等你先说 / 你的话不值得我回应"

### 沉默的写法

**不是**"他沉默了"——这是偷懒。

**是**用动作/环境填充沉默：

```
❌ 林墨沉默了。
✅ 林墨没说话。他把茶杯转了半圈，茶水撞到杯壁发出一声轻响。
✅ 林墨低头看鞋尖。窗外的雨打在空调外机上，声音一阵一阵的。
✅ 林墨拿出烟盒，摸了一下又放回去。没点。
```

**沉默 ≥ 对话的 20%**（K 教练 5 tips 之一）

单场戏 2000 字对话 → 至少 400 字是角色选择不说的时刻。

### 沉默的位置

沉默不能随便插——插在三个位置最有力：

**位置 1：被触到创伤时**
- 前：情绪/权力/信息的正常推进
- 中：触发物出现（一个词/一个物品/一个声音）
- 后：1-3 秒沉默
- 功能：读者看见"创伤被激活了"

**位置 2：转折点之后**
- 场景中间的转折点（60-75%）出现后
- 让其中一方沉默 2-5 秒
- 功能：让读者消化"刚才发生了什么"

**位置 3：场景结尾**
- 最后一句话说完
- 某人沉默不接
- 另一方的反应（不是接话）
- 功能：制造余韵

## 五、语言指纹 5 要素

### 模型

每个角色必须有独特的**语言指纹**，由 5 个维度组成：

| 维度 | 说明 | 差异化方法 |
|---|---|---|
| **句长** | 平均句子长度 | 短句型 / 中句型 / 长句型 |
| **口头禅** | 反复出现的词组 | 2-3 个专属口头禅 |
| **标点习惯** | 逗号/句号/破折号偏好 | 不同角色不同节奏 |
| **比喻来源域** | 打比方时从哪里取材 | 职业/童年/创伤/爱好 |
| **沉默模式** | 什么时候选择不说 | 被触创伤 / 权力展示 / 思考 |

### 示例：林墨 vs 周文渊 的语言指纹

```yaml
# 林墨（主角·心理封闭型）
voice_fingerprint:
  sentence_length:
    avg: 8-15 字
    style: "短句为主，偶尔一个长复合句（情绪爆发时）"
  
  verbal_tics:
    - "嗯。"（应声而非回答）
    - "我不知道。"（回避）
    - "就这样。"（结束话题）
  
  punctuation:
    - 句号用得多（明确切分）
    - 逗号用得少（不拖延）
    - 破折号几乎不用
  
  metaphor_domain:
    - 医疗（病院三年）："像做检查一样问我" / "这事得先观察"
    - 身体：从身体感觉出发打比方
  
  silence_pattern:
    - 被提及"父亲"时沉默 2 秒
    - 被问到情绪时用动作代替回答

# 周文渊（反角·修辞型）
voice_fingerprint:
  sentence_length:
    avg: 15-30 字
    style: "长句为主，有起承转合"
  
  verbal_tics:
    - "这件事..."（起句习惯）
    - "说到底..."（引入转折）
    - "你要明白..."（居高临下）
  
  punctuation:
    - 逗号用得多（思考型停顿）
    - 破折号偶用（深度展开）
    - 句号来得晚
  
  metaphor_domain:
    - 文学/历史："像春秋战国那样..." 
    - 父辈的权威感
  
  silence_pattern:
    - 对方情绪失控时沉默（以静制动）
    - 自己说完后停 3 秒（等对方消化）
```

**两个角色的语言从头到尾不会被读者搞混。**

### 对比测试

随便抽一段对话，把说话人标签遮住——读者能猜出是谁说的吗？

- 能猜出 ≥ 80% → 语言指纹合格
- 猜不出 / 猜错 → 语言指纹失败，重写

### SOUL.yaml 中的 voice 字段

每个 S/A 级角色 SOUL.yaml 必须配置：

```yaml
performance:
  voice:
    sentence_length: "短句为主"
    length_range: "8-15 字"
    tics: ["嗯。", "我不知道。", "就这样。"]
    unique_markers: ["从不用反问句", "从不称呼对方的名字"]  # 独有特征
    punctuation_style: "句号多，逗号少"
    metaphor_domain: ["医疗", "身体感觉"]
    silence_triggers: ["父亲相关话题", "被问情绪时"]
```

## 六、Jaccard 替代法——让角色从根本上不一样

### 什么是对话 Jaccard

两个角色说话的**高频词重合度**。

- Jaccard = 0 → 两个角色用词完全不同
- Jaccard = 1 → 两个角色用词完全一样
- 底线：**两个 S 级角色之间的 Jaccard < 0.25**

### 旧版的机械检测（已废弃）

原项目有 `dialogue-jaccard.js` 脚本做量化检测。本次重构**删除脚本**，改为**定性判断**——责编在审稿时直接判断：

### 责编定性判断法

**方法**：读两个角色各自的一段对话（≥10 句），问自己：

1. **他们的句长明显不同吗？**
2. **他们的口头禅/习惯用语没有重叠吗？**
3. **他们打比方时的"来源域"不同吗？**
4. **他们的标点节奏不同吗？**
5. **他们选择沉默的时机不同吗？**

4/5 项通过 → 语言指纹合格
3/5 以下通过 → 混用严重，必须重写一方

### 差异化的 7 个技巧

**技巧 1：给每个角色分配一个"从不用"的词类**
- 林墨：从不用反问句
- 周伯：从不用感叹词

**技巧 2：给每个角色分配一个"专属"句式**
- 林墨：主语常省略（"去了。""看了。"）
- 周伯：主语明确（"我昨天去了那里。""你应该看看。"）

**技巧 3：比喻来源域严格分工**
- 医生角色用医疗比喻
- 教师角色用教育比喻
- 禁止两个角色共享来源域

**技巧 4：口头禅要"奇怪"**
- 普通的"那个"、"然后"、"其实"——不具辨识度
- 奇怪的"我先不说" / "反正吧" / "等会儿"——有辨识度

**技巧 5：语气词分配**
- 林墨从不用"啊/呢/呀"
- 周伯从不用"嗯/哦/诶"

**技巧 6：独有称呼保护**
- 林墨称周伯"周伯"——其他人不许用（别人都叫"周先生/周老"）
- 周伯称林墨"墨"——其他人不许用（别人都叫"林先生/林医生"）
- 责编发现混用立即修正

**技巧 7：一字拒绝**
- 每个角色有一个"绝对不会说的字"
- 林墨绝对不说"好"（回避承诺）
- 周伯绝对不说"对不起"（身份骄傲）

## 七、对话节奏——短短长 vs 长长短

### 两个基础节奏模型

**模型 A：短短长（爆发型）**

```
"你来了。"
"嗯。"
"你父亲的遗物里，有一封信我一直没敢让你看。"
```

**效果**：前两句累积紧张，第三句释放信息。适合**信息揭示**场景。

**模型 B：长长短（收束型）**

```
"我知道你这几年在病院做了什么，也知道你为什么今天来找我。我不怪你
——你父亲走的那年，换作是我，我也会变成你现在这样。"
"我不是来找答案的。"
"那你来干什么？"
"——我不知道。"
```

**效果**：前两句铺开背景/情感，最后一句锋利收束。适合**情感爆发**场景。

### 节奏失败的症状

**症状 1：全短句型**
```
"你来了。"
"嗯。"
"坐。"
"嗯。"
"喝茶？"
"嗯。"
```
→ 对话没有推进感，像 AI 模拟人类说话。

**症状 2：全长句型**
```
"我一直觉得你这几年过得很不容易，毕竟你父亲走得那么突然，而且你
自己又因为病情的关系住进了病院，这三年我虽然不常来看你，但我一
直很挂念你的状态。"
"我其实也挺想你的，每次想到你一个人在北京我心里都不好受，但是
我又不知道该怎么跟你联系，因为我怕打扰你的治疗。"
```
→ 对话没有呼吸感，像写报告。

### 单场戏节奏配比

一场 40-60 段对话的场景，推荐分布：

- 短句（≤10 字）占 40%
- 中句（10-25 字）占 45%
- 长句（≥25 字）占 15%

### 关键瞬间的节奏

**揭示信息时**：短 + 短 + **长** → 炸点在长句
**情感爆发时**：长 + 长 + **短** → 炸点在短句
**沉默打破时**：**短** → 第一句必须短，重新启动对话
**场景收束时**：**短** → 最后一句短，干净利落

## 八、"后悔就是戏"——角色的 1 秒自省

### 核心原则

**角色说完一句话后，如果他立刻在心里后悔 → 这 1 秒就是戏。**

后悔不需要纠正——后悔本身就是高潮。

### 样本

```
周伯：你父亲最后一天，一直叫你的名字。
林墨：他已经叫了三年了。
（2 秒沉默。两人都意识到林墨刚才说漏了——他一直听见父亲的声音）
周伯没说话。他端起茶杯。茶是冷的。
林墨起身走向窗边，背对着他。
"我是说，你们说他一直叫。"
周伯终于开口："我明白你的意思。"
```

这段 40 字里有戏：

1. 林墨说漏了 secret（他在病院能听见父亲声音）
2. 两人都意识到了
3. 谁都没纠正——但空气变了
4. 林墨的补救（"我是说..."）显得更虚
5. 周伯的"我明白你的意思"（双关——我明白你在补救 / 我明白你没疯）

**最高级的对话艺术。**

### K 教练五 tips 补录

（原 pro-advisory-notes.md 的 K 教练建议，保留并整合）：

1. **每 3 轮必须注入 1 个"计划外微事件"**——打断角色的舒适区（由表演指导/世界管家执行）
2. **Agent 的第一句回复禁止直接回应上一句**——必须有 0.5-2 秒的"身体缓冲"
3. **每场戏至少 1 次"说错话"**——角色说完立刻后悔的瞬间
4. **沉默 ≥ 对话的 20%**——用行动/环境描写填充沉默
5. **每个角色每场戏有 1 个"私人议程"**——他真正想要什么（不一定说出口）

## 九、对话的常见陷阱

### 陷阱 1：对话只是"交换信息"

```
❌ 
"你今天去哪里了？"
"我去了医院。"
"为什么？"
"看望一个朋友。"
"这个朋友怎么样？"
"还好。"
```

→ 这是问答，不是对话。修法：让每句话同时**在做**别的事。

### 陷阱 2：角色都说"人话"

每个角色都说"自然的普通人话" = 所有角色说一样的话。

角色是**有人格的个体**，说话必须**偏离中性**。让角色有口音/方言/职业腔/年代感。

### 陷阱 3：对话过顺

```
❌
A："你觉得这事怎么样？"
B："我觉得应该这样处理。"
A："你说的对。"
B："那我们就这么办。"
```

→ 无冲突的对话 = 假对话。修法：每两句必须有一次"微小的不同意"。

### 陷阱 4：解释型对话

角色之间互相解释世界观、背景、规则。**这是给读者看的，不是角色之间的对话**。

修法：
- 让信息通过**动作**传递（物件、环境、行为）
- 让信息通过**侧面**传递（第三人的反应、报纸标题）
- 让信息通过**冲突**传递（两人争论时互相抛出）

### 陷阱 5：同质化反应

所有角色被同一件事触发时反应相同。

```
❌
A：听到了父亲去世的消息。"不可能！"
B：听到了父亲去世的消息。"不可能！"
C：听到了父亲去世的消息。"不可能！"
```

→ 每个角色的反应**必须连到他的创伤/动机**。

```
✅
A（丧失型创伤）：身体僵了。没说话。
B（羞辱型创伤）：立刻站起来走到窗边。"你告诉我的时间，是下午？"
C（背叛型创伤）："是你告诉我的。那你早知道了。"
```

### 陷阱 6：独白化对话

```
❌
林墨：我想告诉你，这三年我经常想你，我知道你一直在外面等我，我也知道你对我来说意味着什么，所以我今天来，是想让你知道我的真实感受，我希望我们可以重新开始，因为没有你的生活对我来说是空的。
```

→ 这是独白，不是对话。现实中没有人这样说话。

修法：把独白拆成 **5-8 段**，中间插入 B 的反应 / 动作 / 沉默：

```
✅
林墨：这三年我经常想你。
（周伯看了他一眼，没说话）
林墨：我知道你在外面等我。
周伯：我没等过你。
（林墨停下了）
林墨：你今天来见我是因为什么？
周伯：因为你父亲。
```

## 十、对话学检查清单

责编在 Phase 4 审稿时逐条检查：

### 潜台词层
- [ ] 每句对话是否在"做"某件事（不仅是"说"）
- [ ] 至少 30% 的台词有 ≥2 层潜台词
- [ ] 关键台词（揭示/爆发）有 ≥3 层潜台词

### 角色差异化层
- [ ] 随机抽 10 句对话盖住说话人标签，能猜出 ≥ 80%
- [ ] 两个 S 级角色的语言指纹 5 要素中 4 要素不同
- [ ] 口头禅/独有称呼没有混用

### 节奏层
- [ ] 短/中/长句比例大致 40/45/15
- [ ] 揭示/爆发瞬间的节奏对位合理（短短长 / 长长短）
- [ ] 沉默占对话时间 ≥ 20%
- [ ] 没有独白化对话（单句 ≥ 100 字的独白）

### 艺术层
- [ ] 每场戏至少 1 次"说错话"
- [ ] 每场戏至少 2 次"沉默"
- [ ] 每个出场角色有明确"私人议程"
- [ ] 没有"问答式对话"（连续 5 句都是 Q→A）

### 红线
- [ ] 没有解释型对话（角色互相解释世界观）
- [ ] 没有同质化反应（不同创伤角色反应一样）
- [ ] 没有"过顺"对话（没有微小的不同意）

**任意一项标红 → 进入责编修订流程。**

## 十一、火影案例——带土 vs 鸣人最终对话拆解

```yaml
scene: "带土面具裂开那一瞬间"

# 原始对话
dialogue_snippet: |
  带土："你以为你能改变什么？"
  鸣人："我不是来改变你的。"
  带土：（停了一下）"那你来干什么？"
  鸣人："我来告诉你——琳她如果还在，不会希望你这样。"
  带土："你不认识她。"
  鸣人："你也不认识我。"

# 潜台词分析
subtext_analysis:
  line_1:
    character: 带土
    surface: "质疑鸣人的改变能力"
    subtext_layers:
      - L3 关系: "拒绝让鸣人靠近自己"
      - L4 权力: "我比你看得透"
      - L6 恐惧: "我怕被你说服（说服就意味着承认自己错了）"
  
  line_2:
    character: 鸣人
    surface: "表明来意"
    subtext_layers:
      - L5 欲望: "我不是来谈判的"
      - L4 权力: "反转——我不要你的认可"
  
  line_3_pause:
    action: "带土停顿 1 秒"
    meaning: "他的预期被打破——他准备好了应付'改变我'的鸣人，但没准备好'不来改变我'的鸣人"
  
  line_4:
    character: 鸣人
    surface: "用琳的话施压"
    subtext_layers:
      - L5 欲望: "我要戳到你最软的地方"
      - L7 秘密: "我知道你的秘密是琳"
  
  line_5:
    character: 带土
    surface: "反击鸣人不了解琳"
    subtext_layers:
      - L6 恐惧: "如果我承认琳会这么想，我就白活了"
      - L3 关系: "拒绝建立共同语境"
  
  line_6:
    character: 鸣人
    surface: "反击"
    subtext_layers:
      - L4 权力: "镜像你的武器（你说我不了解她→我说你不了解我）"
      - L5 欲望: "不要让这场对话结束"

# 语言指纹
fingerprint_check:
  带土:
    - 句短 (7-10字)
    - 反问句（"你以为..."）
    - 拒绝承认对方（"你不认识她"）
    - 比喻域：抽象哲学
  
  鸣人:
    - 句短 (5-12字)
    - 陈述句（"我不是来..."）
    - 承认对方（"你也不认识我"）
    - 比喻域：身体/情感

# 节奏
rhythm:
  pattern: "短 + 短 + 停顿 + 中 + 短 + 短"
  climax: "line_4（用琳的名字）+ line_5（反击）"
  resolution: "line_6 的镜像反击让对话在对等中结束"
```

**这就是 3 层以上潜台词的精彩对话。**

## 附录 B：Jaccard 替代——责编的"语言指纹盲测"

```
责编在 Phase 4 执行：

1. 从 novel.md 随机抽 10 句对话
2. 遮住每句前的说话人标签（"林墨说"、"周伯说"...）
3. 请文学顾问独立猜测说话人
4. 记录命中率：
   - ≥ 80%：语言指纹合格
   - 60-80%：部分角色混用，标出需要改写的角色
   - < 60%：语言指纹失败，整体重写对话
5. 输出报告写入 editor-review.md 的"语言指纹"节
```

---

> 对话不是让角色说话。对话是让角色在说话的时候，暴露他们不想暴露的东西。
> 好的对话的标志：**角色说完后，他自己都不知道为什么要这么说。**
