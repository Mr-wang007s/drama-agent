# 样本与反模式对照集（samples-and-antipatterns）

> 这是 Director 的"好戏/坏戏"肉眼校准册。
> 读十本创作方法论不如看三组"Before / After"对照。
> 本册从真实流水线失败和成功片段里提炼，**可增量扩展**——每次 Phase 6 系列复盘时追加。

---

## 使用方式

- **Phase 1 规划前速读**：了解本故事已有的好戏样板 / 反模式样板
- **Phase 2 卡住时翻查**：某一段写不下去时查"我是不是踩了 AP-X"
- **Phase 3 编译自检时对照**：交付 novel.md 前读一遍 ✅ 样本确认自己是在正轨

## ✅ 成功样本（可复用范式）

### SP-1 · 物理施压替代心理描写

**出处**：jiu-ge EP01 场景 3（母亲探视）

**片段**：
```
她伸手想摸他左耳那枚青铜铃铛耳钉。

她的手伸到他耳边停住了，停了两秒，又收回去，收回去之前在空中停了一下，像是忘了自己原本要干什么。
```

**为什么有效**：
- 没有一句内心描写（"她开始忘记他了"这种）
- 完全靠**手的轨迹**说话（伸出 → 停 → 收回 → 停 → 放下）
- 读者自己拼出"她在忘"
- 符合许教授"身体诗学"方法论

**可复用范式**：需要呈现"不对劲的心理"时，**优先让身体做这件事**——手停在半空、坐姿僵了半秒、目光在物品上停 0.3 秒。

---

### SP-2 · 私语消失 = 褪色的第三方证据

**出处**：jiu-ge EP01 场景 3（母亲叫"糕"）

**片段**：
```
"糕，"她说。

我顿了一下。

糕糕。以前三十年是糕糕两个字。今天少了一个。
```

**为什么有效**：
- 变化极小（两个字变一个字）
- 变化极具体（"糕糕"→"糕"，读者能看见缺口）
- 变化的叙述极克制（"少了一个"四个字）
- 这是 **"显性变化 + 隐性含义"** 的典范——读者看到物理变化（少字），自己想出含义（她在忘）

**可复用范式**：关键信息通过**关系专属私语的微变化**呈现，不直接说。母亲说"儿子"变"孩子"，老友的称呼从"老王"变"王老师"，都是这类。

---

### SP-3 · 身体先决定，意识后确认

**出处**：jiu-ge EP01 场景 4（陈队长录像）

**片段**：
```
我的右手食指在桌下也敲了一下，两下，三下，四下。

停。

我没发觉。
```

**为什么有效**：
- 主角的**身体已经做出了选择**（敲到第四下停 = 和录像里的陈队长同频 = 他已经"看见"了）
- 但主角**意识还没追上来**（我没发觉）
- 这个"身体-意识"落差就是戏的全部张力

**可复用范式**：主角做了一个**他自己没意识到的动作**，而读者和场上某个角色（周伯/张教授）看见了。身体的诚实在主角的防御之前。

---

### SP-4 · 沉默代替对白的关键决定

**出处**：jiu-ge EP01 场景 5（服务区影子）

**片段**：
```
我的影子在地上，长，偏东北。他的影子也在地上，也长，也偏东北，但是淡了一度。我盯着那块地看了一秒就收回来。

我没说。
```

**为什么有效**：
- 林墨看见了一个重大异常（沈师兄的影子淡一度 = 褪色证据）
- 按普通写法他会问
- 按这个写法他"没说"
- 沉默本身是他的选择（"今天不命名"）——这个选择比任何对白都有力

**可复用范式**：场景里最重要的发现**不一定要说出来**。让主角做一个"他看见了但选择不说"的决定——这是比发问更高级的戏。

---

### SP-5 · 群戏中旁观者的关键目击

**出处**：jiu-ge EP01 场景 4（张教授的目击）

**片段**：
```
录像里陈队长敲第四下的时候，他食指也动了一下。第二下。第三下。第四下，停。

张教授的视线也落下去了。他和我对了一眼，谁都没出声。
```

**为什么有效**：
- 张教授是群戏里的第三人，几乎不说话
- 但他**看见了**林墨的食指在同步敲
- 他和周伯的"对视 + 都没说"本身是本场最重要的证词
- 读者从"第三方目击"获得了"这件事是真的，不是林墨的幻觉"的确认

**可复用范式**：群戏中让最不说话的角色**目击关键事件**，并用他们的**沉默对视**作为证词。比任何旁白都可信。

---

### SP-6 · 身体已忘 = 最深的褪色证据

**出处**：jiu-ge EP01 场景 4 结尾（空口袋）

**片段**：
```
我的身体忘了口袋这个东西了。三年。两年零十一个月。病号服没有口袋。我的手每天插不进任何地方。今天它插进去了，它以为里头有什么，结果什么都没有。
```

**为什么有效**：
- 一个极具体的身体经验（把手伸进口袋）
- 一个极小的发现（是空的）
- 一个巨大的隐喻（身体已经忘了这世上有"口袋"）
- 但作者**完全没说"这是一个隐喻"**——让读者自己读到

**可复用范式**：身体的失忆是最深的失忆。用**最微小的日常动作**（把手伸进口袋 / 按错电梯楼层 / 伸手想拿手机发现手机不在这个衣服口袋）呈现角色的生活被改变。

## ❌ 反模式（踩过的坑）

### AP-1 · 第一人称独白拼图

**出处**：jiu-ge EP01 v2 重写（未落盘，被用户识破）

**片段**（想象版）：
```
林墨

6:40。我睁眼。我走到镜子前。我看他三秒。

李医生

我 08:15 就坐下了。我的左手按在右上角。我按了十二年。
```

**为什么失败**：
- 用角色名作为第一人称独白的小标题
- 叙述者视角是第一人称"我"
- 多个角色各占一段独白
- 不是一个统一的叙述者在看戏，是 N 个人在自述

**根因**：Team 里每个 Agent 输出的是"我的视角全场景"，Director 编译时直接拼接，没做第三人称熔合。

**正确做法**：见 `compile-from-interactions.md` 的"按时间熔合，不按角色分段"原则。

---

### AP-2 · 全知旁白包装所有角色行为

**出处**：jiu-ge EP01 v1 首次生成（`d4c7448` commit）

**片段**：
```
他没伸手去碰。他先把床头柜上的白色小药片拿起来放进嘴里，端起玻璃杯，把昨晚留的半杯凉白开喝掉。然后他坐下来，用食指压住纸的一角，慢慢把它滑过来。
```

**为什么（偏）失败**：
- 第三人称是对的
- 但是**一个无形叙述者**在按节拍记录所有动作
- "他 X。他 Y。他 Z。" 句式垄断
- 角色没有"自己活"——是被叙述者摆布

**根因**：Director 采用了"全知视角 + 动作罗列"——没有让角色通过他**真正重要的动作**说话，而是平铺直叙所有动作。

**正确做法**：保留少数**具有情感重量的动作**（"他对镜子里的自己笑了一下"——这是"点名确认"的关键动作），删除大量流水账动作（"他把水喝掉"这种只是填空）。密度要收紧。

---

### AP-3 · 碎片化独占段（"他停了。" 独占段连环出现）

**出处**：jiu-ge EP01 v1（被 check-ai-taste Warning 抓到 45 处）

**片段**：
```
他停了。

他左耳那枚青铜铃铛耳钉很轻地响了一声。

他把那张纸翻过去，扣在桌上。

他没有起立。

他去拿袜子。
```

**为什么失败**：
- 每一句都独占一段
- 节奏全是断点，没有呼吸
- 张力被碎片化稀释

**根因**：Director 把"留白"理解为"每句独占一段"——其实留白是稀缺资源。真正的留白 = **在密度正常的段落之后，出现一个独占段**作为重拍。

**正确做法**：5-8 句连续叙事 + 1 句独占爆点 = 良好节奏。**独占段的爆炸力来自前文的密度**——没有密度就没有爆炸。

---

### AP-4 · 内心暗涌直白说出

**出处**：jiu-ge EP01 v1 修订前（被 Critic 标为"外在主导过强"相反面）

**片段**（想象版）：
```
林墨感到一阵深深的悲伤。他意识到母亲已经开始忘记他了。他的心沉了下去。
```

**为什么失败**：
- 三句话全是**直白的心理陈述**
- 没有身体、没有环境、没有物品
- "深深的悲伤"是 AI 味情绪词
- "心沉了下去"是 C5.6 黑名单句式
- 读者被告知情绪，但**感受不到**

**正确做法**：见 SP-1（物理施压替代心理描写）和 SP-2（私语消失）。

---

### AP-5 · 多角色同时说话（群戏失控）

**出处**：尚未在实战发生，但属于"如果全员 spawn 都设置相同 prompt 会发生"的预期失败

**片段**（想象版）：
```
周伯说："你去吧。"
沈砚之说："墨弟我来接你。"
张教授说："那我陪林老师走这一趟。"
林墨说："我去。"
周伯又说："路上小心。"
沈砚之又说："上车。"
```

**为什么失败**：
- 四个人一段一句连环说
- 没有身体动作间隔
- 读者跟不上视角
- 群戏失控

**正确做法**：群戏采用"1 主 + 1 副 + N 旁观"结构，每 200 字最多激活 1-2 个角色的对白，其余角色用"目光 / 点头 / 微动作"参与。见 `scene-staging.md` 形态 C 的调度策略。

---

### AP-6 · 环境描写开头长段堆砌

**出处**：Director 容易犯的通病

**片段**（想象版）：
```
面谈室在二层东南角，是全院日照最早的一间。八点二十分这时候太阳还没越过对面那栋楼的屋脊，室内是漫反射的灰白光。桌上那份复核病历摆得比平常偏右一寸半。夹子那一边冲着窗户。病历封面边缘有一处被反复按出的浅压痕。

李医生 08:15 就坐下了。
```

**为什么失败**：
- 开头 100 字全是环境
- 角色迟迟不出场
- 读者没有进入的锚点

**正确做法**：见 `compile-from-interactions.md` 技法 4——环境描写**拆散成呼吸**，散布到整个场景里。开场用**时间 + 地点 + 角色动作**快速入戏。

## 增量更新协议

Phase 6 系列复盘时产出的**优秀片段**和**失败片段**增量追加到本文件：

- 位置：对应分类（✅ 或 ❌）的末尾
- 编号：继续往下（SP-7, SP-8, ... AP-7, AP-8, ...）
- 格式：**出处 + 片段 + 为什么有效/失败 + 可复用范式（或正确做法）**
- 每个样本 ≤ 200 字

> 本文件是活文档，越用越厚，但不应改写老样本（它们是历史证据）。只做追加。
