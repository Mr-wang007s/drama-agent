## 一、世界观：《九歌》— 东方克苏鲁

### 核心设定（电梯演讲）

> **《山海经》《楚辞》里的神祇全是真实存在的"认知污染体"——寒武纪之前的先住民。青铜器是古人用血肉炼成的"精神防毒面具"，而主角是三千年来第一个天生免疫的人。**

### 世界观四层结构

```
┌─────────────────────────────────────────┐
│ 显性层：考古圈、民俗学界、体制内研究    │  ← 前10章活动范围
├─────────────────────────────────────────┤
│ 隐性层一：九歌司（屈原秘密传承，官方）  │  ← 10-30章
│ 用诗词歌赋作为封印语言                  │
├─────────────────────────────────────────┤
│ 隐性层二：九座"神墟"埋在中国大地        │  ← 30-80章
│ 每座对应《九歌》一篇，封印一位"旧神"    │
├─────────────────────────────────────────┤
│ 核心层：旧神=寒武纪先住民               │  ← 80章后
│ 它们的"存在"本身会污染人类认知，见即疯  │
└─────────────────────────────────────────┘
```

### 力量体系："心斋九境"（取自庄子）

| 境 | 名称 | 能力 | 突破方式 |
|---|------|------|---------|
| 一 | 观想 | 看见纹饰中的细节 | 入门 |
| 二 | 坐忘 | 听见死物的低语 | 吟诵《诗经》一篇 |
| 三 | 心斋 | 隔绝部分精神污染 | 直面弱小旧神而不疯 |
| 四 | 朝彻 | 用诗词制造短暂的"规则区域" | 完成一次仪式性献祭 |
| 五 | 见独 | 与旧神对话不被覆写 | 直面一位完整旧神 |
| 六 | 无古今 | 穿越文物进入"过去的某一刻" | 承受三千年记忆注入 |
| 七 | 不死不生 | 短暂脱离物理规则 | 临死前的顿悟 |
| 八 | 撄宁 | 以言语重定义现实局部 | 亲手封印一位旧神 |
| 九 | 物化 | 成为新的"神" | 未知 |

**核心规则**：每境突破都需吟诵古辞 + 直面一位旧神而不疯。能力不是打架，而是**"定义现实"**——用语言重塑物理规则。

### 多方势力（五方对抗）

1. **九歌司**（官方正统）：主张"封"。挂靠社科院，用诗词封印。**保守、体面、有底线**
2. **方士会**（民间野修）：主张"炼"。把旧神炼成丹药和金手指。**贪婪、实用主义**
3. **招魂堂**（南派巫蛊）：主张"通"。与旧神共生，换取超凡能力。**神秘、疯狂**
4. **旧神教**（被污染者）：主张"迎"。召唤旧神降临。**反派**
5. **海外同行**：日本阴阳寮、欧洲秘术会、梵蒂冈圣座观测团**各有诉求**

### 金手指 + 代价（核心爽感引擎）

**林墨的金手指**：精神病院的"误诊"其实是正确诊断——他天生携带**免疫基因**，能直视旧神而不疯。这是三千年一遇的变异。

**代价（核心痛点）**：免疫的本质是"不被污染=不被承认"，他正在从这个世界**褪色**。
- 档案消失
- 熟人遗忘
- 镜子照不出影
- 最终连自己的母亲都想不起他

**用得越多越接近"不存在"。**

### 三条硬规则（防崩塌）

1. **诗词必须精准对应情境**（吟《国殇》必须在战场，吟《湘夫人》必须在水边）
2. **每句诗一生只能用一次**（强迫作者设计场景而非堆砌）
3. **旧神具象化有红线**：完整真身要留到 80 万字后，前期只出"残影"

## 二、角色群像

### 主角：林墨 · 做梦的考古队长

> **"他每次在梦里'去过'的地方，第二天就能挖出东西。"**

| 维度 | 设定 |
|------|------|
| **外在标签** | 黑眼圈重到像打了阴影，却出奇地爱笑；随身带一本写满胡话的梦境笔记，字迹时而工整时而癫狂像两个人写的；左耳戴一只青铜铃铛耳钉（据说能"叫醒"他） |
| **内在驱动** | 七岁那年他在梦里"救过"一个穿红衣的小女孩，醒来却发现那是二十年前失踪的堂姐。从此他分不清自己是"活着的林墨"还是"别人梦里的林墨"。**他不是在找文物，他是在找"到底是谁在做这个梦"** |
| **OCEAN** | O85 / C70 / E40 / A55 / N65 |
| **Ghost** | 被误诊关精神病院三年——全世界都说你疯了，你怎么证明自己没疯？ |
| **Wound** | 对"被相信"有深入骨髓的饥渴，对"相信别人"极度谨慎 |
| **Lie** | 只有无可辩驳的证据才能保护我 |
| **Shield** | 用极度的黑色幽默和自嘲把自己包裹起来 |
| **金手指** | 梦境预知 + 与"梦中亡者"对话 + 免疫旧神污染 |
| **代价** | 每睡一次，现实中会有一个认识他的人"忘记他"——他正在"褪色" |
| **口头禅** | "我昨晚梦见你了。"（对陌生人说）／ "等等，你是真的吗？" |
| **上头点** | 存在主义级别的孤独 + 越强大越孤独的宿命感 |

### 核心对手：沈砚之 · 考古界的天之骄子

> **温润如玉的师兄，同时是林墨命运的"救命恩人 + 毁灭者"**

| 维度 | 设定 |
|------|------|
| **身份** | 九歌司最年轻的"执笔"（掌门候选）。国家级考古首席专家。沈家嫡长子 |
| **关系** | 林墨的亲师兄 + 未婚妻沈清鸢的亲哥哥 + 当年把林墨从第一座神墟里背出来的救命恩人 |
| **魅力点** | 永远三件套西装、对林墨说话永远是"墨弟，你要保重"。**他不是坏人，是"为了守护人类不得不封锁真相"** |
| **关键行为** | 亲手销毁了林父的遗物，因为"有些东西公开了，人类会疯" |
| **核心对抗** | **真相 vs 秩序**。林墨要揭开，沈砚之要埋葬。两人都对，必须有一个输 |
| **隐藏设定** | 他自己也在褪色——但他早已接受 |

### 二号对手：陆九 · "饕餮"

> **黑市盗墓第一人，林墨儿时玩伴，"不敢成为的另一个自己"**

| 维度 | 设定 |
|------|------|
| **身份** | 黑市排名第一的盗墓贼，代号"饕餮"。方士会的座上宾 |
| **外表** | 三十岁，吊儿郎当的酒鬼痞子，笑起来有颗小虎牙 |
| **魅力点** | 嘴贱、不讲武德、会在林墨最狼狈时递烟；盗墓只盗有"名字"的器物（他在替某个死人收集遗物） |
| **隐藏身份** | **林墨七岁那年在梦里救下的"红衣小女孩"——原本是男孩，为躲仇家被林家送走男扮女装** |
| **关系逆转** | 林墨忘了他，他记得林墨说过的每一句话。盗走的"有名字的器物"里有一件刻着"林墨" |
| **核心戏剧** | "你忘了我，没关系，我替你记着。" |

### 终极对手：先生 · 活了太久的文明守夜人

> **自称"先生"，看起来三十岁，实际活了上千年**

| 维度 | 设定 |
|------|------|
| **表面身份** | 神秘古董商人，品味极致 |
| **真实身份** | 上一轮文明派来"观察"这个文明的记录员 |
| **行为模式** | 不杀人，他"收走"人——把人变成自己记忆博物馆里的藏品 |
| **魅力点** | 极致优雅、语速缓慢、会在凶案现场给林墨泡一杯宋代点茶 |
| **对林墨** | 复杂的怜爱："小家伙，你走的每一步，我两千年前都走过。" |
| **终极反转** | **他就是无数循环之后、最终"成功"活下来的林墨自己** |
| **结局** | 年轻林墨选择一条先生从未走过的路，先生在他面前化为尘埃，含笑说："这次，你赢了。" |

### 女主一号：沈清鸢 · 退婚的豪门千金

> **五年前当众退婚林墨的那个女人，现为国际文物追索律师**

| 维度 | 设定 |
|------|------|
| **外在** | 冷脸，西装，高跟鞋，商界人称"沈阎王" |
| **独立动机** | 当年退婚不是嫌弃，是哥哥沈砚之拿父母性命威胁她——她以为自己在救林墨。五年来她暗中调查，**比林墨更接近真相** |
| **关系张力** | 表面冰冷互撕，私下办公室抽屉里锁着林墨所有论文的打印稿。重逢第一面扇了林墨一巴掌，第二面塞给他一个 U 盘 |
| **磕点** | "我退婚，是因为我爱你"的延迟爆点；她比他更狠更强更有资源，是强强联手而非拯救；她哥是最大反派，注定手足相残 |
| **经典台词** | "林墨，这次换我娶你，你敢不敢？" |

### 女主二号：阿依古丽 · 神秘族群最后的祭司

> **西域某失落部族末代"听墓人"（女性世袭）**

| 维度 | 设定 |
|------|------|
| **外在** | 十九岁，看起来像大学生，实际"活"了三代人的记忆 |
| **独立动机** | 族人全部在三十年前一场"沉默瘟疫"中集体坐化，她要找出元凶 |
| **与主角关系** | 她算出林墨是"开门的钥匙"，可能要在关键时刻**用掉**林墨 |
| **张力** | 她对林墨极好极温柔，林墨越爱她越不敢确认她的真心。**"我爱你，但我也要杀你"** |
| **经典场景** | 她在他熟睡时拿刀抵在他心口，手抖了一夜，最终转身离开 |

### 三个让人意外的关系反转

1. **林墨最敬重的导师周老 = 林父的凶手**（放在 2/3 处）
2. **盗墓贼陆九 = 林墨失散的堂哥/那个红衣小孩**（放在卷二卷末）
3. **反派"先生" = 未来的林墨**（全书最后一层反转）

## 三、商业定位

### 开篇 500 字四要素（第一章硬标准）

| 位置 | 任务 | 示例 |
|------|------|------|
| **0-50 字** | 视觉奇观钩（不正常画面） | "青铜面具的眼睛在流泪。不是水，是黑色的、带甲骨文的液体。" |
| **50-200 字** | 主角登场 + 专业差异 | 众人慌乱，只有林墨蹲下用镊子夹起液体："这不是墓，是药瓶的盖子。" |
| **200-400 字** | 金手指首次显形 | 林墨掏出爷爷留的纸条："见甲骨泪，退三步，背诵《诗经·黍离》首章。"背到第七句，面具闭上了眼 |
| **400-500 字** | 章末钩子 + 主线暗线 | 面具背后用朱砂写着活的字，还在渗血：**"林墨，你爷爷死前，欠我们第二条命。"** 而他爷爷三十年前就失踪在这座墓里 |

### 五大爽点模板

| 模板 | 频率 | 触发条件 | 核心结构 |
|------|------|---------|---------|
| **①知识碾压** | 40% | 权威嘲讽主角外行 | 主角说出三个连专家都不知道的细节 |
| **②规则揭秘** | 25% | 全队陷入诡异无法解释 | 主角突然安静→说出古代仪式规则→给出破解方法（留30%代价） |
| **③底牌升级** | 15% | 主角被逼入绝境 | 付出代价→亮出3章前的伏笔→反杀→轻描淡写一句收尾 |
| **④反派破防** | 15% | 反派得意长篇阐述 | 主角："你说完了吗？"→点破致命漏洞→反派四段情绪崩溃 |
| **⑤救场神迹** | 5% | 重要配角遇险 | 主角不在场→之前铺的物件/规则发动→他早就算到了 |

### 章末钩子三模板

```
疑问钩（70%）：具体名词级问题，不含糊
  ✅ "爷爷留的7把钥匙，为什么其中一把是他自己牙齿的形状？"

反转钩（20%）：本章刚解决的事，章末反转
  ✅ "他们下去的时候，只有六个人。"

危机钩（10%，卷末必用）：重新定义读者对主线的理解
  ✅ "照片里角落站着一个穿考古队制服的男人，是十分钟前死在他面前的那具尸体。拍摄时间：1998 年 6 月 12 日。"
```

### 差异化核心竞争力（营销 slogan）

> **"古墓不是目的地，古墓是规则。"**

**三点支撑**：
1. 从"探险"升级到"博弈"—古今智力对抗
2. 从"奇观"升级到"系统"—规则可积累可传承
3. 从"遇鬼"升级到"懂鬼"—主角主动利用规则反杀

### 目标读者三画像

| 群体 | 占比/付费比 | 关键要求 |
|------|------------|---------|
| **考据控老读者** | 30% / 50% | 真实考据 + 学术人设 + 不能有硬伤 |
| **爽文党番茄系** | 55% / 30% | 每章必爽 + 钩子密集 + 不受挫超过3章 |
| **女频悬疑新读者** | 15% / 20% | 智性浪漫 + 女主独立 + 不工具化 |

## 六、引擎适配建议

基于当前 DramaAgent 引擎的能力：

### Skill 扩展需求

1. **drama-novel 增加"硬核悬疑"模式**：
   - 章末钩子强制检查
   - 爽点模板使用统计
   - 考据细节真实性 lint

2. **drama-critic 增加维度**：
   - 追读率预测（根据钩子强度）
   - 爽点密度检测
   - "AI 味"检查（已有）

3. **新增 drama-continuity**（剧情连贯性）：
   - 伏笔追踪（埋设章 → 揭晓章）
   - 势力进度表
