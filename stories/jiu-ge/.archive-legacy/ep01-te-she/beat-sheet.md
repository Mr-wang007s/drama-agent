## 核心一句话

一个在精神病院住了三年的人，被一纸特赦拽回"他已经学会不去看"的那个世界——他接受了特赦，因为他母亲今天早上叫他"糕"，少了一个字。

## 场景节拍表

### 场景 1 · 病院清晨

| 维度 | 内容 |
|------|------|
| **冲突** | 林墨的秩序被打破：今天的桌上多了一张纸 |
| **爽点** | 读者第一次看见这个"疯子博士"的真实状态——他清醒得可怕 |
| **情绪** | 3 → 3.5（几乎不动） |
| **钩子** | 释放 H-A1（殷墟录像预告） |

- **字数预算**：1100 字 / 28-32 段
- **出场角色**：林墨（独幕）+ world-manager（病院环境 + 杂工）
- **创伤链触发目标**：`lie`（只有无可辩驳的证据才能保护我）
- **私人议程**：
  - 林墨：想按平常把药吃了然后睡回笼觉——他今天本来有 8:15 到 9:30 的放风
- **意象激活**：`青铜铃铛耳钉`（引入阶段——今天林墨戴着它起床）
- **信息差方向**：对等（读者和林墨同时看见那张纸）

**Beat 细节**：
1. 林墨 6:40 醒。照镜子——这是他每天第一件事（SOUL.quirks）。镜子里他还在。他对镜子里的自己笑了一下。这不是高兴的笑，是点名确认。
2. 他走到书桌前。桌上多了一张纸。不是病历。是社科院的公函抬头。他没伸手去碰。他先坐下来，把药放嘴里，把水喝掉。然后他才用食指把那张纸压住一角，慢慢滑过来。
3. 他读完。他敲桌——先两下慢的，第三下快。他停了。他的耳钉轻轻响了一下。他把纸翻过去扣在桌上。他没有起立。他去拿袜子。
4. 窗外保安老王今天没跟他打招呼。不是漏了，是没看他。林墨不生气——他记下来。这是第三十七笔他自己悄悄记的"被忘账"。他把袜子穿好。
5. 他在日记本上写下今天的日期和三个字："有人来。"就这三个字。然后合上。他去洗漱。

慢镜头 ≥2：①照镜子的"点名确认"；②食指把公函滑过来的 0.5 秒

### 场景 2 · 告别李医生

| 维度 | 内容 |
|------|------|
| **冲突** | 医生和病人同时知道"这不是治好了，是被调用了"，但两人都没说破 |
| **爽点** | 李医生最后一次表达"我相信过你没疯"——读者替他说不出口的那句话接上 |
| **情绪** | 3.5 → 4（一丝被看见的温度） |
| **钩子** | 释放 H-B3（敲桌节奏 7Hz 被医生点名一次——为后续"林墨听见青铜低语"做身体痕迹） |

- **字数预算**：1400 字 / 35-40 段
- **出场角色**：林墨 + 李医生（a_li-doctor，需创建，见附录）
- **创伤链触发目标**：`ghost`（被误诊的三年）
- **私人议程**：
  - 林墨：他想不靠李医生就走完今天——他怕李医生再对他好一次他会哭
  - 李医生：最后一次以医生身份把应该说的说出来——"林墨你当年说的青铜在说话，我这三年一直不确定那只是幻听。"但他今天不会把这句话讲完——他只讲半句
- **意象激活**：无
- **信息差方向**：对等

**Beat 细节**：
1. 面谈室。李医生桌上一份复核病历摊开。他没看，他看林墨。林墨坐下了。
2. 李医生问："你昨晚睡得怎么样。"林墨说："跟前天一样。"李医生点头。他没再问下一句。他沉默了六秒。
3. 他说："林医生——"他停了。他换了称呼，他说："林墨，你这里可能还会再来一个人找你说话。不是我。那个人说话的方式你可能会不舒服。"
4. 林墨说："好。"他只说一个字。他的手指在桌面底下敲了一下——7Hz 的那个节奏（SOUL.quirks）。李医生看见了。李医生没说。
5. 李医生把复核病历合上。他说："我这里最后一份给你的东西——你的出院证明。还有，"他从抽屉里拿出一个很普通的白色小卡片，"这是我私人电话。"他没把话说完——"万一……"他自己咽回去了。
6. 林墨接过卡片。他把卡片翻过去——背面空白。他把它夹进日记本。他说："谢谢李医生。"他站起来。他走到门口，回头，他说："那个青铜柜子——"他停了。他说："没事。"他走出去。
7. 李医生在林墨走出门的瞬间低头写病历。他写的第一行是："病人状态稳定。可出院。"他写完这一行，他没写下一行。他合上病历。他把笔放下。他看了一眼那扇门。

慢镜头 ≥2：①李医生合病历的那一下；②林墨走到门口回头"那个青铜柜子"说到一半咽回去

**C1 读者替接话**：林墨那句"那个青铜柜子——"读者会自动接上"它还在说话吗"。林墨没说。
**C2 沉默比发言主动**：李医生"万一"没说完；林墨"那个青铜柜子"没说完。两次。

### 场景 3 · 告别母亲

| 维度 | 内容 |
|------|------|
| **冲突** | 林墨发现母亲今天叫他"糕"不是"糕糕"——他假装没听见 |
| **爽点** | 读者比林墨晚一秒反应过来："糕糕"少一个字意味着什么 |
| **情绪** | 4 → 4（表面不动）/ 内心 4 → 7（巨大暗涌） |
| **钩子** | 释放 H-B1（褪色第一次对母亲生效）；回指前史"母亲每周三探视" |

- **字数预算**：1600 字 / 42-48 段
- **出场角色**：林墨 + 林母（b_lin-mu-mother）
- **创伤链触发目标**：`fear`（最后连母亲都忘记他，而他还没有找到任何他存在过的证据）
- **私人议程**：
  - 林墨：把母亲送走，不让母亲察觉他发现她叫错了
  - 林母：今天来接儿子回家——但她潜意识里已经不太确定"儿子还在不在"（她自己不知道自己在不确定）
- **意象激活**：`青铜铃铛耳钉`（挑战阶段——母亲想摸它但手停在半空；耳钉在她眼里"有点陌生"）
- **信息差方向**：读者 > 主角（林墨后知后觉，读者先反应过来——"糕糕"少字了）

**Beat 细节**：
1. 母亲来得比探视时间早半小时。保安老王今天不挡她。她拎一个保温饭盒。她坐在会客室的椅子上等。
2. 林墨走进来。她站起来。她看他的脸。她愣了半秒——半秒就是半秒，她自己察觉不到。她说："糕，过来。"
3. 林墨走过去。他没纠正。他坐下来。他打开饭盒——是他小时候她做的栗子糕。四块。她一向做五块。
4. 她说："今天我做了栗子糕。"她看他吃。他咬了一口。糯。他点头。她高兴了。
5. 她说："你今天怎么，今天……"她自己停下来。她自己不知道她想说什么。她笑了笑："今天天气好。"林墨说："嗯。"
6. 她伸手想摸他左耳那枚耳钉。她的手伸到他耳边停住了。她又把手放下。她说："这个……这个挺好看的。"她像第一次看见它。
7. 林墨没动。他的脖子硬了一下。他用左手把保温饭盒合起来——合的时候力气用大了半分，盖子扣响了。他说："妈，我今天要跟周伯伯出一趟门。中午不回来吃饭。"
8. 她说："好。糕——"她又叫了一次。她又停了："你，你记得早回。"林墨说："嗯。"
9. 她走出会客室的时候没回头。她走到门口把保温饭盒忘在椅子上。林墨看着那个饭盒。他没去追她。他坐下来。他把饭盒打开。他又咬了一口栗子糕。他咬得很慢。他咬了六下才咽。
10. 他把饭盒盖上。他把它抱在怀里。他站起来。他去还保温饭盒。他没追她——她已经走了。他把饭盒交给前台。他说："我妈忘拿了。麻烦你帮我给她。"他没写名字在便签上。他知道她不记得姓林的人是谁，便签写了也是白写。

慢镜头 ≥2：①母亲手伸到耳钉边停住的 0.5 秒；②林墨咬栗子糕咬得很慢那一下
**C1 读者替接话**：母亲没把"你今天怎么"说完——读者自动替她接"你今天怎么看着有点陌生"。她没说。
**C2 沉默比发言主动**：林墨全程没纠正"糕"。
**R1-R5 破防戏**：本场景不是破防戏（林墨 budget 是 2，留给 EP08 + EP12）。这里是**预崩塌**——读者感受到崩塌在远处走来但还没到。

### 场景 4 · 周文渊的录像

| 维度 | 内容 |
|------|------|
| **冲突** | 周文渊要让林墨"看见"——但林墨不能表现出他已经看见了 |
| **爽点** | 陈队长录像里敲桌节奏到第四下停——读者猛然想起 SOUL 里那个"7Hz 敲节奏" |
| **情绪** | 4 → 5（身体背叛了他的冷静） |
| **钩子** | 释放 H-A1（殷墟 M5-3A）、H-A2（林父二十年前）、H-B2（红塔山烟盒，本集不送，本集只"让林墨看见周伯桌上那盒烟"） |

- **字数预算**：1800 字 / 48-55 段
- **出场角色**：林墨 + 周文渊 + 张教授（b_zhang-prof，社科院代表）+ 录像里的陈队长（c_chen-duizhang，非实际在场）
- **创伤链触发目标**：`wound`（他分不清自己是"活着的林墨"还是"别人梦里的林墨"）
- **私人议程**：
  - 周文渊：让林墨"自己决定"去殷墟。他不想用命令——他要的是林墨未来二十年都不能说"是周伯逼我的"
  - 林墨：他想拒绝。他的理性在算"拒绝的后果"。他的身体已经在敲桌（7Hz）。他的身体先答应了
  - 张教授：走完流程，少说话。他今天的职责是把签字页递过去
- **意象激活**：`7Hz 敲桌节奏`（引入阶段——本集第一次把"身体习惯"和"青铜低语"建立因果）；`红塔山空烟盒`（引入阶段——本集林墨只看见，不接）
- **信息差方向**：读者 > 主角（读者已经从场景 2/3 攒了两处褪色证据；林墨还在压抑）

**Beat 细节**：
1. 办公室。周伯泡茶——六安瓜片，这是林墨从小喝的那种。张教授坐在沙发一角。周伯的办公桌上摆一盒红塔山空烟盒（开封）——林墨的目光在上面停了 0.3 秒（他认得这个牌子，他父亲抽这个）。周伯看见他的目光停了。周伯没说这盒烟是怎么回事。他让林墨坐下。
2. 周伯说："我放一段录像给你看。"他把笔记本转过来。张教授没看林墨，他看窗外。
3. 录像——殷墟 M5-3A 沟边。三月五号凌晨。陈队长蹲在沟边。他在念念有词。他在敲桌——不是桌，是他自己膝盖。七下一组。一组，两组，三组。到第四组他敲到第四下停。他抬头。他笑了。他的眼神不对。录像结束。
4. 周伯说："陈队长的队一共十六个人。现在十三个在医院，两个失踪，陈队长在殡仪馆。死因是'心源性猝死'——但他死的时候是站着死的，站在工地边上。法医三次过不去。"
5. 林墨的右手在桌面下敲了一下。两下。到第四下他停了。他自己没发现。张教授转过头看了林墨一眼，张教授的目光在林墨右手边停了两秒。张教授没说。
6. 周伯说："我不是来救你出院的。我是来问你：你愿不愿意去看一眼。只看一眼。不做判断。看完你回来，你愿不愿意继续当'林医生'都可以。"
7. 林墨说："陈队长敲到第四下停。"他只说了这一句。他没说他也刚才敲到第四下停。周伯笑了。周伯说："你看见了。"林墨没回答。
8. 林墨说："我去。"他只说了两个字。他站起来。
9. 周伯说："今天下午走。张教授跟你办手续。"周伯推过来一份特赦令，比今早桌上那张多一页——"社科院特聘顾问"聘书。张教授递笔。林墨签字。他签完名字他没收笔——他把笔放回桌上。他看了一眼那盒红塔山空烟盒。周伯没说。林墨也没问。
10. 林墨走出办公室的时候张教授跟上来。张教授在走廊里压低声音说："林墨，下午三点服务区见。那里还会有一个人。"林墨说："我知道。"他其实不知道。他赌张教授会告诉他。张教授说："沈砚之。"林墨的耳钉响了一下。林墨说："嗯。"

慢镜头 ≥2：①录像里陈队长敲到第四下停的那 0.5 秒；②林墨签完字把笔放回去看那盒红塔山的 0.5 秒
**C1 读者替接话**：林墨说"陈队长敲到第四下停"——读者自动替他接"我刚才也敲到第四下停"。他没说。
**C2 沉默比发言主动**：林墨全程没问那盒红塔山是怎么回事。

### 场景 5 · 服务区交接

| 维度 | 内容 |
|------|------|
| **冲突** | 沈砚之从周文渊手里接过林墨——两人都对林墨伪装"我只是你的师兄/导师"，林墨也伪装"我只是学生" |
| **爽点** | 林墨发现沈砚之的影子在夕阳下比自己的淡一点——他没说 |
| **情绪** | 5 → 5（不动）/ 警觉度从 5 → 7 |
| **钩子** | 释放 H-A3（沈砚之影子异常——卷一长线），H-B4（沈砚之鞋面一尘不染在殷墟不合物理——EP07 回收） |

- **字数预算**：1300 字 / 34-38 段
- **出场角色**：林墨 + 周文渊 + 沈砚之（第一次实际出场）+ world-manager（服务区环境）
- **创伤链触发目标**：`shield`（用黑色幽默和自嘲把真心埋得很深）
- **私人议程**：
  - 沈砚之：不让周伯看出他已经把林墨视为"替代自己褪色进度的人"——他来接人不是接学生，是接"继承者"
  - 周文渊：把林墨"正式"交给沈砚之——但他在心里记下沈砚之今天早到了十二分钟
  - 林墨：他想和沈师兄说些什么轻松的——他上一次见沈师兄是三年前
- **意象激活**：`沈砚之的影子`（引入阶段——夕阳下比林墨淡一度）
- **信息差方向**：读者 > 主角（读者从 state.json 里的 active_secrets 隐约知道沈砚之自己也在褪色；林墨不知道他师兄淡影的原因）

**Beat 细节**：
1. 京港澳高速服务区。下午四点十二分。车停在加油区边上。周伯没下车。张教授下车了。林墨下车。风大。
2. 沈砚之已经在那里。他站在一辆黑色别克旁边。他穿深色大衣。他的大衣没被风吹起来——哪怕风很大。林墨的大衣被吹起来了。
3. 沈砚之走过来。他说："墨弟。"他叫"墨弟"——这个称呼只有他能叫。林墨说："师兄。"他说出"师兄"两个字的时候他发现自己鼻子酸了一下。他用冷静压下去。
4. 沈砚之看林墨。他说："你瘦了。"林墨说："病院伙食其实不错，主要是我自己不争气。"——这是他的黑色幽默（SOUL.shield）。沈砚之笑了一下——笑得很短。沈砚之说："走吧。今晚住安阳。明天早上去殷墟。"
5. 周伯此时下车了。周伯和沈砚之握手。握手时间是正常的 1.2 秒。两人都没多说一句话。周伯对林墨说："到地方了给我发条短信。"林墨说："好。"周伯上车。周伯的车走了。
6. 夕阳从服务区屋檐下的缝里斜切下来。林墨这时候看见了——沈砚之的影子在地上比自己的淡一度。不是光的问题，光是平均的。林墨没说。他只用了 0.3 秒把这个发现塞进日记本（他脑子里的那一本）。他把它记在新一页上，题目是"今天不命名"。
7. 沈砚之说："上车。"林墨上车。副驾。沈砚之开车。车开出服务区。林墨从后视镜里再看了一眼那个夕阳——影子问题这会儿已经看不见了。
8. 车上广播正在放新闻。安阳方向在下毛毛雨。沈砚之打开雨刷。雨刷第一下摆过挡风玻璃的时候林墨的耳钉又响了一下。沈砚之听见了。沈砚之没说。林墨也没说。车往安阳开。
