## 核心一句话

> 林墨回京面见李医生，发现"签字盗用"指向一个系统性操作——而复核附件中夹着一页父亲笔迹的文件，让他三年来第一次失控。

## 场景节拍表

### 场景 1 · 高铁独处

| 维度 | 内容 |
|------|------|
| **冲突** | 林墨被困在"知道了什么但还不能行动"的等待里——高铁两小时是折磨 |
| **爽点** | 林墨整理三本账的体系感（读者看到主角的缜密，产生"他在布局"的期待） |
| **情绪** | 平静(4)→微紧(5) |
| **钩子** | 回收 H-15（李医生电话后续：林墨决定如何联系他）；暗推 H-04（方士会：李师傅"远亲"回味） |

- **字数预算**：1200 字 / 30-35 段
- **出场角色**：林墨（独角戏）
- **创伤链触发目标**：trauma.lie（"只有证据才可靠"——他在整理证据链）
- **私人议程**：
  - 林墨：把殷墟的收获和李医生电话整理成一条逻辑链，确定回京后第一步
- **意象激活**：I-07 账本手写字（三本笔记本摊开在折叠桌上）
- **信息差方向**：对等（林墨和读者知道的一样多）

**Beat 细节**：
1. 高铁窗外风景移动 → 林墨摸左耳（习惯动作但耳钉不在）→ 他把手放下，不再想它
2. 摊开三本账在折叠桌上：褪色账本/伪字本/使用账本 → 他在使用账本第一页补一行备注 → "第一笔的代价：视野扩收一次，闭眼五分钟"
3. 在心里过一遍李医生58秒电话的信息量 → 列出三个"不判断"：谁告诉李医生的/为什么是今天/复核记录的出处 → 手指无意识在桌面敲7Hz节奏
4. 到站广播 → 收账本 → 窗外楼群出现 → 北京的天灰得比安阳重一个色号

### 场景 2 · 到家

| 维度 | 内容 |
|------|------|
| **冲突** | 林墨取回耳钉想正常戴上，但手不稳——身体在告诉他今天不是"正常日" |
| **爽点** | 翻沈家族谱摘录的第一句话就是他父亲名字（读者高潮："原来他父亲也是这个系统里的人"） |
| **情绪** | 微紧(5)→紧张(6) |
| **钩子** | 推进 H-01（父亲真相：摘录第一页提到"林世铭"三个字）；推进 I-06（耳钉） |

- **字数预算**：1800 字 / 45-50 段
- **出场角色**：林墨、母亲（电话）、陆九（暗线·窗外一瞥）
- **创伤链触发目标**：trauma.ghost（七岁梦境 → 父亲的东西总会把他拉回去）
- **私人议程**：
  - 林墨：回书桌前的那个"确定的位置"重新稳住自己，然后联系李医生
  - 母亲（电话）：确认儿子回来了（但她今天叫他的方式……）
  - 陆九（暗线）：在对面楼下出现 3 秒（是巧合还是跟踪？林墨从窗口余光一瞥）
- **意象激活**：I-06 耳钉（取回 → 戴上的瞬间手抖）+ I-08 殷墟的土（鞋底带回来的泥）
- **信息差方向**：读者>主角（读者已知陆九可能是林墨堂哥，但林墨不知）

**Beat 细节**：
1. 进门 → 鞋底的殷墟泥在玄关留一个浅印 → 他没脱鞋直接去书房
2. 书桌。耳钉在白纸旁。"醒着"两字还在。他拿起耳钉 → 手指碰到金属的凉 → 他往左耳垂送 → 手不稳，试了两次才扣上 → 他的左耳重新有重量，但不是二十年前的那个重量
3. 他把沈师兄给的摘录从内兜取出。打开牛皮纸封面。第一页。竖排手抄。第三行出现一个名字：林世铭。→ 他的手指在那三个字上停了五秒 → 翻下一页 → 电话响（母亲）
4. 母亲电话。声音正常。"你回来了？吃晚饭不？" → 她今天没叫他"小墨"，也没叫"墨墨"，叫的是"你" → 他说"吃了妈，你早点睡" → 挂了 → 他不让自己想刚才那个"你"意味着什么
5. 他打开窗通风 → 对面楼下有一个人在抽烟 → 烟头的火星在暮色里是一个亮点 → 那个人的身形……→ 他关窗 → 不判断

### 场景 3 · 见李医生

| 维度 | 内容 |
|------|------|
| **冲突** | 李医生告诉林墨的信息比他准备好接受的多三倍——不是一次签字盗用，是一整套操作 |
| **爽点** | 信息量爆炸：病历三年、签字盗用、药物调整记录异常——每一条都在把真相的碎片拼上去（读者的侦探欲被激活） |
| **情绪** | 紧张(6)→高度紧张(8) |
| **钩子** | 深推 H-01（病历关联附件中有一份文件日期 2006/9/2——父亲"殉职"前一天）；深推 H-15（系统性操作的规模）；释放新 A 级钩子 H-16（"还有第二个人的签字也被盗用了——不是医生"） |

- **字数预算**：2300 字 / 55-65 段
- **出场角色**：林墨、李医生
- **创伤链触发目标**：trauma.wound（"我分不清什么是真的"——现在三年的诊断本身可能不是真的）
- **私人议程**：
  - 林墨：拿到证据，验证"谁在操控我这三年"
  - 李医生：把他知道的全告诉林墨然后抽身（他怕了——他意识到自己可能被利用了三年）
- **意象激活**：I-04 病院走廊灯（李医生描述复核现场时林墨脑中的画面闪回）
- **信息差方向**：主角>读者（林墨在当场做判断，读者跟不上他的推理速度——制造好奇）

**Beat 细节**：
1. 精卫中心附近一家咖啡店 → 角落位 → 李医生到了已经在喝第二杯 → 他的手在桌上（林墨注意到李医生的手比三年前更瘦，指关节突出）
2. 李医生把一个文件袋推过来 → 里面是三份复印件 → 第一份：他自己三年前的入院诊断书（上面有"签字：李XX"——李医生确认不是他签的）→ 第二份：半年复核记录（同样假签字）→ 第三份：关联文件清单（李医生说"这份清单是我今天早上才看到的"）
3. 林墨的手指在翻第三份文件 → 文件清单最后一栏有一行手写备注 → 字迹不同于前两份的打印体 → 他的手指停了 → "线索三明治"：前后两行是无关的行政编号 → 中间这行的日期是 2006/9/2 → 林墨的眼球不动但瞳孔在收
4. 李医生："还有一件事。签字被盗用的不止我一个人。第二个人的签字……不是医生。" → 他看林墨 → 他没说那个人是谁 → 他看了看门口 → "我能告诉你的到这里为止。"
5. 李医生起身 → 他犹豫了一下 → 把那三份复印件留在桌上 → "你留着。我没给过你。" → 他走了 → 林墨独坐 → 咖啡凉了

### 场景 4 · 回家独处（breakdown）

| 维度 | 内容 |
|------|------|
| **冲突** | 林墨看到第三份文件关联页的一处笔迹——父亲的笔迹，他认得——但内容是一句他无法接受的话 |
| **爽点** | 情绪爆破：三年精神病院的"真相碎片"第一次拼到足以让主角失控的程度（读者终于看到林墨崩溃——他不是机器） |
| **情绪** | 高度紧张(8)→崩溃(10)→清醒回收(7) |
| **钩子** | H-01 实质性推进（父亲笔迹证实"殉职前一天他还在写东西"→ 他不是当场死的）；H-16 悬念维持（第二个人是谁？） |

- **字数预算**：2200 字 / 55-60 段
- **出场角色**：林墨（独角戏 + 母亲/邻居作为"无关者"间接存在）
- **创伤链触发目标**：trauma.ghost + wound 双触发（父亲 + "我被关了三年可能从头到尾是被安排的"）
- **私人议程**：
  - 林墨：他只想安静地看完文件——但身体不允许
- **意象激活**：I-06 耳钉（breakdown 时他摸耳钉但摸不到自己在摸——解离感）+ I-04 病院走廊灯（闪回画面）
- **信息差方向**：对等→读者>主角（崩溃时读者比林墨清醒）

**Beat 细节**（R1-R5 逐条兑现）：
1. 【R1 前200字过度平静】回到家。关门。他把文件袋放书桌上。他去厨房倒了一杯水。喝了半杯。把杯子放水池边。去洗手间洗手洗脸。回到书房。坐下。他做每一件事都很慢。像一个下班回家的普通人。
2. 他把第三份文件从袋子里抽出来 → 翻到关联页 → 那行手写备注旁边有一个小字 → 他之前没看清 → 他把台灯拉近 → 五个字。
3. 【R2 触发物≤5字】五个字是："墨，别怕。——爸"
4. 【R3 第一个动作不是情绪动作】他把文件放下。他站起来。他去厨房。他重新倒了一杯水。他端着水回来。水洒了一点在地上。他没有看到水洒了。他把杯子放在书桌上。杯子放偏了桌沿一指。他坐下来。
5. 然后他的手开始抖。不是剧烈的抖。是那种从手腕内侧开始的、像是骨头在振动的微颤。
6. 【R4 无关者目击】隔壁传来邻居老太太关门的声音。走廊有脚步经过。有人在门外站了两秒——大概是看到他家没开灯 → 脚步走远了。
7. 崩溃核心（150字以内）：他的左手摸耳钉。耳钉在。但他的手指摸不到金属的凉。他知道是解离。三年精神病院他学过这个词。他的身体在保护他但他不需要保护他需要——他需要把那五个字再看一遍。他低头看。纸上的字没变。爸的字。爸的笔锋。二十年前教他写名字用的那种笔锋。爸在他被关进精神病院之前的前一天……还在写字。
8. 【R5 清醒≥2×崩溃（崩溃150字→清醒≥300字）】他坐在椅子里。呼吸。数到七。再数到七。七是他的数字。精神病院教的。一组七秒。手腕的振动在第三组七秒之后减弱。他把手掌平放在桌面上。桌面凉。和他父亲那张旧书桌一样凉。他的掌心压住灰——不，这张桌没有灰，这是他自己的桌。他今天坐在自己的桌前。耳钉在左耳。账本在内兜。那五个字在面前。他把文件翻回去盖上。他站起来。他打开窗。夜风进来。他深吸一口……不。他没有深吸。他只是把脸对着风。风凉。北京三月的夜风还带一点冬天的尾巴。他回到桌前坐下。他打开使用账本。第二笔：三月十七号夜，李医生给的文件，父亲笔迹确认。代价：解离一次，约90秒。他合上账本。他不哭。他三年没哭过。今天也不是第一次。他只是坐着。耳钉在左耳垂上，重量回来了。

## 破防戏判定

- **破防场景位于**：场景 4
- **R1 前 200 字过度平静**：✅ — 倒水、洗手洗脸、回书桌，每个动作慢且日常
- **R2 触发物 ≤5 字**：✅ — "墨，别怕。——爸"
- **R3 第一个动作不是情绪动作**：✅ — 去厨房重新倒水（杯子放偏桌沿一指）
- **R4 无关者目击**：✅ — 邻居老太太门外停两秒看到他家没开灯
- **R5 清醒 ≥ 2× 崩溃**：✅ — 崩溃约 150 字，清醒回收约 350 字
