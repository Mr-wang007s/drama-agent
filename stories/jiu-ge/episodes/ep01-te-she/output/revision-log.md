### R5 · 场景 4 空口袋段落（+140 字）

**After**（新增）：
```
林墨走到门口。他伸手把手放进大衣口袋。

他左边口袋里什么都没有。他右边口袋里什么都没有。

这件事他今天才发觉。他在病院三年，病号服没有口袋。他今天早上穿上这件三年前的灰色大衣出门，一路走下来他一次都没把手放进口袋——他的身体已经忘了口袋这个东西。

现在他把手放进去。空的。

他把手抽出来。他在门口站了一秒。他对自己说了一句话，这句话他没出声。他说："好。"

他推门出去。
```

一石四鸟解决：
- 老陈"反转点缺位"→ "他对自己说了一句话"是反转点身体锚
- Q 老师"线索三明治"→ "病号服没有口袋"这条信息作为下层面包夹住空口袋这个关键线索
- 许教授"身体诗学单一"→ 脚 + 手 + 声道三个部位都激活
- K 教练"shield 兑现"→ "好"是林墨标志性的单字回应

### R6 · 场景 4 红塔山 0.3 秒扩写

**Before**：
```
他的目光在那盒烟上停了 0.3 秒。

他认识这个牌子。他父亲抽这个。家里他小时候的床头柜第一层抽屉里常年有一盒红塔山，他父亲说"不抽只放着心里踏实"。他父亲出事以后他母亲把那一盒扔了。

他今天看见这盒烟在周伯桌上。
```

**After**：
```
他的目光在那盒烟上停了 0.3 秒。

然后他把目光收回来。他没让自己在这件事情上多留一秒。他向前走了两步，他在沙发边站定。他有一个他自己才知道的规矩——凡是会让他的手出现任何不必要动作的东西，他今天都不让自己看第二眼。

他没看第二眼。

但他左耳那枚青铜铃铛耳钉在他进门那一刻也响了一下。他听见了。他没动。
```

按 Q 老师建议**不解释**林墨想到什么——读者自己拼凑"父亲/红塔山/耳钉响"三方关联。主角 > 读者信息差建立。

### R8 · 场景 5 沈砚之 silence 回合

**Before**：
```
沈砚之说："你瘦了。"

林墨说："病院伙食其实不错，主要是我自己不争气。"

这是他的老毛病。他一紧张就自嘲。沈砚之知道这个老毛病。

沈砚之笑了一下。笑得很短。眼睛没笑。
```

**After**：
```
沈砚之说："你瘦了。"

林墨说："病院伙食其实不错，主要是我自己不争气。"

这是他的老毛病。他一紧张就自嘲。沈砚之知道这个老毛病。

沈砚之没有立刻接话。他等了一拍。他让这句自嘲在两个人中间空了一秒。然后他才笑了一下。笑得很短。眼睛没笑。这一拍的沉默是沈砚之的规矩，林墨三年前就知道这个规矩，今天他还记得。
```

兑现 sheng-yanzhi SOUL 的"沉默本身就是施压"+ 小悦"沈砚之出场偏短"的共识。
