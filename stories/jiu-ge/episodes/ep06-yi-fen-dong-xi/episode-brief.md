## 本集定位

**卷一末集**。卷一的戏剧任务：把林墨从病院送出山 → 经历四场逐级升温的博弈（工地-夜 / 天亮 / 回京 / 栗子糕）→ 最后把他**推**到一个不得不选择的节点。

本集的核心动作：周文渊兑现 EP05 电话里那句"带你看一份东西"。

**关键约束（对齐 SOUL canon）**：

`s_zhou-wenyuan` SOUL 明文：
- `arc.season_arc`：**"卷一：温厚导师（读者完全被骗）/ 卷二卷末：身份反转（大震撼）"**

**一句话主题**：周伯让林墨看一份东西，林墨看完之后第一次主动把青铜铃铛耳钉从左耳取下来——不是为了摘掉，是为了让它醒着的时候离身体远一点。

## 核心冲突

### 外部冲突

周文渊带林墨进考古所三年前没进过的一个房间（所内禁书库的某个分区 / 或一个只对所长开钥匙的档案室），拿出一份东西给林墨看。这份东西是**父亲 20 年前未寄出的一页田野笔记残页**。

残页的内容：父亲最后一次殷墟现场的记录。记录里有：
- 一段林墨从未听过的、对 M5-3A 那个位置的早期描述
- 一处日期异常（日期晚于官方"殉职日"一天）
- 父亲的签名——这个签名的笔迹与林墨左书房抽屉里父亲**其他**笔记上的签名有细微差别（笔锋停顿的角度 / 收笔的弧度）
- 一行用铅笔淡淡划过的字，被橡皮擦过又没擦干净，林墨不用光照看不见

周文渊当面对林墨说的话：
- "这是你父亲给我的。"
- "他走前一个礼拜。"
- "我压了二十年。"
- "你现在看，你能看懂。"

周文渊这一套话术的特征：**每一句都可能是真的，每一句都可能不全**。林墨识别不出谎与半真的边界——因为周伯说话从不带情绪抑扬，他的"真"与"伪"用同一个温度说出来。

### 内部冲突

林墨在看这份东西的时候面对一个新命题：**周文渊给他的每一样"父亲的东西"都精确地踩在他的感知阈值上**。红塔山烟盒（EP04）让林墨闻到父亲气味；钢笔（EP05）让林墨摸到父亲指痕；这份残页让林墨**读到父亲的字**。

这形成一个递进：气味 → 触感 → 文字。三步都在打林墨对父亲的身体回忆。

林墨的怀疑：**周伯是在用父亲的东西训练我**。训练什么？训练我去看某个东西？训练我去"想起"什么？

但他同时不能排除另一种可能：**周伯只是在偿还一个二十年的债**。

这两种解释在今晚并存。他无法证伪任何一个。

### 创伤链兑现

- **ghost**（精神病院三年）：EP06 设定里周文渊拿残页时的手法（慢慢铺开 / 用镊子 / 戴白手套）会让林墨瞬间想起**病院李医生第一次给他展示病历时的同一套动作**。李医生的专业动作 = 周文渊的专业动作 = 让一个病人"自己发现真相"的标准医用程式。林墨在看残页的同时看见自己作为"被看对象"的历史。
- **wound**（存在感可疑）：这份残页如果是真的，证明父亲在"死前"写过林墨现在才能看懂的东西——这意味着父亲**预知**到林墨长大后会读到它。但如果父亲预知，他就知道林墨会活下来。为什么要留一页而不是留信？预知了却不说话，这本身就是对"我存在"的否认。
- **lie**：林墨第一次在周伯面前**主动使用 EP04 学到的"歧义操纵"技巧反向操纵周伯**——他用一句模棱两可的观察让周伯暴露一个细微表情。
- **shield**：黑色幽默在 S2 全程失灵（对方是周伯，不是母亲——但周伯的"如父"性质让 shield 也收不住）。

## 关键目标

### 情节目标

1. **周文渊 S 级第二次大戏**。比 EP04 更深一层：EP04 是"四问一礼"的外显结构，EP06 是"一件东西 + 一个房间 + 一段话"的压缩式。
2. **父亲遗物三回归闭环**：烟盒 / 钢笔 / 残页。三件物件对应**嗅觉 / 触觉 / 视觉**三个感官通道。
3. **读者的怀疑达到最高点**：通过周伯本集的四个小动作（拿钥匙开门时手抖、解说残页时突然沉默三秒、林墨提"母亲今早的电话"时周伯的眼神变化、临别拍林墨肩膀的力道比 EP01 轻了一点）让读者在心里已经"确认"周伯有问题，但林墨作为主视角不能在本集说破。
4. **卷一钩子**：本集末林墨收到一条来自沈砚之的短信（或一个实物信物），意思是"明早来殷墟"。林墨要在凌晨做一个决定：**去 or 不去**。他不告诉周文渊。
5. **青铜铃铛耳钉首次被主动取下**：不是摘掉。是放在书桌上让它"醒着"。耳钉从贴身符号变成外置指示器——林墨和耳钉的关系第一次出现间距。这是卷一末的终极身体仪式。

### 心理目标

- 林墨在本集里第一次考虑**主动离开周文渊的保护圈**。不是明面断裂，是走一步看一步。
- 他意识到他昨天对母亲说的那个"保"字需要升级到另一个层面：他必须开始"保"父亲（的真相）——但保父亲意味着某种程度上的"弃"周伯。
- 卷一末林墨的心理定格：**"我不再是周伯给我画的那张地图上的一个点。我开始看地图本身。"**

### 视角合规

本集是**林墨主视角**，不能越界。周文渊的 secret（方士会高阶 / 活埋林父 / 孙女 / 监测林母）**严禁**泄露。林墨可以**推理**到边缘，但不能在内心独白里明确命名。

陆九身份（堂哥）不能泄露。如果陆九在本集出场，也只是**林墨仍无法确认**的身份。

沈砚之如果短信/电话，内容必须与他 EP01 服务区登场时的身份一致——给青铜残片的那位师兄。

## 人物出场

### 林墨（视角 · S）
- 情绪：本集从 weary-calm 开场，中段（S2 后半）进入 EP04 以来第一次的"冷怒"——他意识到周伯在用父亲当诱饵
- 硬边界：不在周伯面前动任何笔、不在周伯面前说"父亲"以外超过三个字的评论、离所前不对周伯表态

### 周文渊（正面 · S）
- 情绪：warm-paternal（按 SOUL 维持）
- 今晚他做的：开一个他三年前不曾让林墨进的房间 + 拿出一页他压了二十年的纸 + 说"你现在看，你能看懂"
- 今晚他的四处破绽（读者可见 / 林墨感知但不点破）：
  - 开钥匙的手背青筋跳（一次）
  - 解说残页到某处突然沉默三秒
  - 林墨提"母亲今早给您打过电话"时周伯的眼神往下走了半格
  - 临别拍肩力道比 EP01 的那一下轻

### 沈砚之（S · 背景）
- 不正面出场。一条短信，简短至极。
- 内容必须与 EP01 的青铜残片铺垫一致

### 陆九（S · 暗线）
- 可选微型照面：林墨回家途中在小区门口的花坛边看见一个烟头（陆九只抽祭祖烟，烟头的品牌林墨辨认不出但气味让他停了两秒）
- 不接触、不对话
