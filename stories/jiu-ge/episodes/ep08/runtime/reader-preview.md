## 弃文风险

1. **Sc2 陈"我吃"翻转**：EP07 她说"吃到最后一粒就是放弃"的逻辑极硬·EP08 让李学工摇头她就吃了·这个翻转太快。陈 SOUL 不会这样被一个李学工摇头就改主意。读者会觉得"作者在推进剧情"
2. **Sc5 沟边四条"归"线同场**：李学工"归来""兮"+陈吃药+骨片盒分家·Sc5 1100 字塞 3 个爆点·容易稀释
3. **Sc6 信息量过载**：沈给九片结构 + 骨片磨者方向 + 沈清鸢 + "那天没按住的那个人" + "你爸知道你也迟早知道"·5 个 ★★ 在一场·读者消化不过来

## 给编剧的话

**Sc2 陈"我吃"那一下请重写**。陈 agent 如果进 writers-room 一定会反对——EP07 她说"我还在"是有逻辑锚的·EP08 让她在李学工摇头后吃药等于把她的逻辑推翻。**建议**：陈在 Sc5 不吃那半粒·她说"留给需要的人"·然后她在 Sc5 结尾把半粒交还林墨——陈的第三天没有"她吃"这个动作·有的是"她选择不吃·把选择权还给林墨"。

**Sc6 信息量请分到 EP09**。九片结构 + 骨片磨者方向 + 沈清鸢——三件事一集给·EP09 读者就没动力了。**建议**：EP08 只给"沈清鸢二十年前保管"+"九片之二《湘夫人》"这两条。磨者方向"那天没按住的那个人"**留 EP09**。

**沈"停四秒"请让沈 agent 守**。这是沈自己的身体签名·EP06 用过一次·EP08 应该变成"停两秒"或"没停"（新的克制方式）。

**铃铛请让它响一次不响一次**·读者对铃铛机制的阅读已经太熟·需要新鲜感。
