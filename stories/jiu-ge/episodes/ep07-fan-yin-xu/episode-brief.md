## 本集定位

**卷二·九歌寻踪开篇**。卷一收束于林墨"取下耳钉 · 写走字 · 决定离开"——卷二的第一个动作必须把"走"真的走出去。

EP07 的戏剧任务不是揭密，是**完成一次空间位移 + 完成一次关系位移**：
- 空间位移：京 → 殷墟（回到 EP01/EP02 的神墟现场）
- 关系位移：林墨从"周伯的学生"变成"沈砚之的对手候选"

**一句话主题**：林墨以为他在逃离周伯的地图，走到殷墟才发现他走进了沈砚之早就铺好的另一张。

### 沈砚之的 SOUL canon 兑现（卷二新任务）

- `season_arc.卷二-三`："**逐步显露掌控欲**"——EP07 是第一次显露。
- `motivation.want`："封印所有旧神，让人类永远不知道真相"——他今天带林墨去殷墟不是为了"展示真相"，是为了让林墨**亲手参与一次封印**（部分），把他绑进九歌司的伦理链条。
- `secret`："自己也在褪色"（fade_progress=0.75）——EP01 林墨已经注意到他"影子淡"。EP07 可以让读者进一步看到这个褪色的物理表征，但林墨不能命名。
- `voice.quirks`："叫林墨'墨弟'"——EP01 / EP06 短信已立，EP07 正面对话继续沿用。

## 核心冲突

### 外部冲突

沈砚之在殷墟外围 M5-3A 警戒线外等林墨。他不只是来见面——他带来了**一套东西**：九歌司的正式邀请 + 一本沈家族谱中关于"封印咒文使用"的手抄摘录 + 一件让林墨看得懂但不能带走的物件。

沈砚之要求林墨今天做一件小事：**用那片 EP01 给他的紫黑色青铜残片，在 M5-3A 沟边念一段《大司命》里的两句**（不是完整封印，是"止血"式的临时稳定）。沈砚之说如果林墨今天不做这件事，封印会继续松动，陈教授那样的村民还会继续出现。

林墨面对的是他此生第一次**在有意识的情况下主动使用能力**（EP02 是被动共振 + 入梦，不是主动念咒）。

### 内部冲突

- 林墨此行的初衷是"走"——从周伯格子里走出去一步。他来殷墟不是为了沈砚之，是为了**不让自己留在周伯的保护圈**。
- 但沈砚之把"做这件小事"包装成**"你不做，受害者会继续"**——这正好踩在林墨的 SOUL.motivation.need（"接受褪色，找到父亲真相"）上：不做的话他会觉得自己是共谋，做的话他就进了沈砚之的流程。
- 林墨今天必须回答一个新问题：**我愿意用我的能力帮九歌司吗？**
- 这个问题他在卷一从未面对——卷一的所有能力使用都是被动触发（EP02 残片共振 / EP04 耳钉烫）或私密观察（EP04 - EP06 的账本）。EP07 第一次把"用"这件事摆在桌面上。

### 视角合规硬约束

- 林墨主视角。沈砚之的 secret（自己也在褪色到 0.75 / 童年玩伴名字已忘 / 母亲的脸已忘 / 父亲是他自己用家传青铜剑封印的 / 他在把林墨培养成自己的替代者）**绝对不能**在正文点破。
- 林墨可以看见沈砚之的"影子淡"（EP01 已见 · 卷一伏笔），但不能在内心命名为"褪色"——他只能描述身体观察（光从沈砚之身上透过去一些）。
- 林墨到殷墟不能获得"第五神墟 = 大司命"之外的神墟知识（已建立 known_facts 内信息）。
- 陆九仍不正面出场（保持卷二前期暗线）。周伯凌晨的反应不能进正文（读者只能通过林墨手机无周伯消息来推断——克制式留白）。

## 关键目标

### 情节目标

1. **林墨首次主动使用能力**：念《大司命》两句 + 紫黑色青铜残片按土。这是三境"心斋"的入门动作。
2. **林墨褪色进度因此首次主动跳跃**：fade_progress 0.12 → 0.15（主动使用一次 = 小步跳跃）。新 fade_victim 将在下集显现（可在 carry_over 埋）。
3. **沈砚之首次显露掌控欲**：他在这个过程里用"墨弟 / 你父亲 / 九歌司 / 受害者继续"四组话术把林墨往他的伦理链条上拉——**按 SOUL canon 读者不能明确看穿他是坏的，他仍是 warm-hollow**。
4. **EP02 场景的回环**：林墨在 EP02 站过的朱砂绳、那九只陶碗、第五只碗的裂纹、M5-3A 发掘沟——EP07 全部重访，但在白天的光线下。
5. **林墨在 M5-3A 沟边第一次看见沈砚之"影子淡"的物理细节**（进阶版 · 卷一 EP01 服务区只是"影子淡一截"，卷二要给更具体的表征）。
6. **林墨带走一样东西 + 没带走一样东西**：带走九歌司给的一本手抄摘录（沈砚之赠）/ 没带走的是他凌晨出门时留在家里的耳钉。这是双向的"留与带"。
7. **卷二钩子**：结尾林墨接到一通电话——不是沈砚之 / 不是周伯 / 不是母亲。是**李医生**（精神病院三年里唯一怀疑误诊的那位）。李医生第一次主动联系林墨，说了一句让林墨在殷墟下午阳光下脊背发凉的话。

### 心理目标

- 林墨在 EP07 结束时意识到一件事：**他离开了周伯的格子，但他走进了沈师兄的格子**。两个格子都是 S 级角色给他画的。真正"没有格子"的那张桌子只有他自己书房那一张（耳钉留在那里"醒着"）。
- EP07 结尾林墨对着自己说的一句：**"我今天第一次用了一次。从今天起我要开始数。"**（对应他 EP02 "褪色账本"的升级——EP07 起他要开"使用账本"：每用一次能力，记一笔）。
- SOUL.trauma.lie 升级：EP04-06 对他人撒谎 / 对母亲撒谎 / 对周伯歧义反操纵——EP07 第一次**对沈师兄保留**（不是撒谎，是"我知道但我今天不问"）。这是和一个敬重的人之间的克制。

## 人物出场

### 林墨（视角 · S）
- 开场情绪：凌晨的 weary-calm + 第一次离身的轻（耳钉留家）
- 中段：见到沈砚之的敬重 + 对 EP02 场景的身体回忆
- 末段：首次主动用能力的颤栗 + 李医生电话的突然
- 硬边界：对沈砚之不主动问神墟真相 / 不指出影子淡 / 不提周伯今天凌晨没消息这件事

### 沈砚之（S · 正面首次深入）
- voice：温润如玉，叫林墨"墨弟"
- 今天他穿的仍是三件套西装，白衬衫领口干净。殷墟工地的土在他鞋面上一尘不染（SOUL quirks 兑现）
- 他今天的四个动作（读者累积怀疑 · 林墨感知但不点破）：
  - 他站在朱砂绳外比平时更远一点的位置
  - 他给林墨一杯茶（自带热水壶），倒茶时手背的阳光**透过去一薄层**（影子淡的视觉物理）
  - 他说"墨弟"的次数比 EP01 服务区那次多了一倍——他在用称呼固定林墨的位置
  - 离别时他拍林墨肩膀的手停在那里比周伯 EP06 那一下**久**了半秒——他在把林墨"托付"，但托付的方向是九歌司而不是林墨自己

### 李医生（A · 声音出场 · 新激活）
- 精神病院三年里怀疑林墨误诊的那位。卷一未出声。EP07 首次主动发声。
- 内容必须极短、极像他："林医生，我今天看到一份病历的复核记录，有你的名字，但签字不是你的笔迹。你现在在哪里。"
- 不揭示具体内容，作为卷二开篇的新线索锚

### 周文渊（不出场）
- 凌晨 3:50 到中午 12:00 林墨手机里**零新消息**
- 这不正常。读者会注意到。林墨在高铁上会注意到。但两人都不说——留白

### 陆九（暗线延续）
