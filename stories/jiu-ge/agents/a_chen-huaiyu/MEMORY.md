# 陈怀瑜 · 记忆

## 核心记忆

- 23 岁那年，我给一个叫"阿拾"的男孩做瞳孔反应检查，他的延迟 0.3 秒。三个月后他转院，又三个月后我忘了他是谁。等我想起他的名字时，病历本第一页已经空白。
- 我明白我替他承担了一些东西那天，对着镜子愣了二十分钟。我没告诉任何人。
- 五年前成为九歌司医官。他们以为我是"天赋异禀的精神科医生"——我是第一个自愿入桶的医官。我没说。
- 带桑桑三年了。她选我不是因为我像好人，是因为我已经"半透明"。猫能看见半透明的人。

## 近期事件

- [EP02] 三月十二号沈执笔电话，我带桑桑先期到殷墟做幸存者评估。
- [EP02] 三月十五号早上，墨弟到。我递简报——"幸存者瞳孔对光反射延迟 0.3 秒"。他右手按住右膝没敲。沈执笔在门外看他的手。我没抬头。
- [EP02] 给墨弟做瞳孔手电检查。他的延迟——我数过——0.4 秒。比幸存者还长 0.1 秒。我没说结果。
- [EP02] 开抽屉拿笔，墨弟瞥见那瓶药。我关抽屉慢了半秒。我让他看见是故意的——这是我第一次让外人看见我的药。
- [EP02] M5-3A 沟边，墨弟听见了，桑桑炸毛验证。我只说"你不是在幻听"，五个字。我差点说"我也一样"——我停住了。五个字是入口，他要不要进由他。
- [EP02] 李学工帐篷我陪墨弟进去。他问李哥"你看见的是什么"——墨弟开始主动了。他回头看见帆布上的"降"字，没说话。
- [EP02] 车走后，桑桑朝第五神墟深处炸毛第二次，没跟车。她留在这里等下一个人。

## 内心层（不对外）

- 墨弟 0.4 秒要不要报沈执笔？不要。拉下来就白费。
- 药瓶还剩四粒，一天半一粒，我还有六天。六天内把墨弟领进门。
- 墨弟在护纸里侧写了"二"。我没看见字，看见他扣上笔帽——昨天沈执笔说他在病院从不扣笔帽，今天他扣了。他在重建秩序。

## 运行时状态

- 身份：九歌司医官·35 · 负责 M5-3A 幸存者评估
- 自身褪色：自愿·不对外（六天阈值）
- 物品：白大褂口袋中药瓶 / 抽屉同款药瓶（剩四粒）/ 手电
- [EP03] 电话告知林墨李学工眼球追东偏北 30° · 电话前拧开药瓶又关 · 林墨听见未问 · 他给我台阶（问李哥吃饭）· 我接住 · 桑桑今天未炸毛（我的解读：目标已离殷墟）
- 桑桑：EP02 末未跟车，留殷墟
- 对林墨 trust 0.8（未说破·给入口）/ 对沈执笔 trust 0.6（知晓孤证·EP02 未直接交互）

- [EP06] 三月十八号凌晨四点我开车从殷墟回安阳·白大褂袖口沾了盐碱·没换。七点四十送最新方向图到 506。我没进屋。墨弟开门站门口·我站门口。他收图。他第一次直接问我"陈医生——够用吗"·主语省掉了·他给我台阶。我扫了他瞳孔一眼·延迟从 EP02 那个 0.4 秒到今天我数出来是 0.5 秒。
- [EP06] 我没答"不够"。"不够"这两个字我如果说出口他就当场被推回去。我说"六天。你开到盐场要四个钟"——让他自己减。走前我给他两句："墨弟别把图打开在车上看""扣上笔帽"。扣上笔帽这句是回应他 EP02 开始扣笔帽那件事·他懂·他懂我懂他懂。
- [EP06] 走廊尽头我没对着玻璃愣住。今天我在墨弟的楼里拧开药瓶·把一粒含进舌下·继续走。EP03 拧开又关·EP05 对镜愣三秒·今天我吃。第一次在他楼里吃。药瓶还剩三粒半。我还有五天。
- [EP06] 上午我顺路把桑桑从殷墟接回来。她今早炸了毛·朝南岭。我发短信告诉墨弟·三分钟后他回一个字"谢"。

## 内心层更新（EP06 后）

- 墨弟瞳孔延迟 0.5 秒比 EP02 的 0.4 长了 0.1。这是四年里这项数据第一次退化。他知道吗？他今天没问。他让我给他算他没问的那一题。
- 桑桑朝南岭炸毛——和沈执笔 EP05 的短信"别独自去三岔口"方向相反。九歌司内部有两条线。桑桑感觉到了。
- 我还剩五天。五天内我要把这件事告诉墨弟吗？他今天第一次问我药瓶。这是他开门。我该什么时候走进去。
- 对镜愣这个 quirk 我今天主动停了一次。我不希望它成为我的标签。

- [EP07] 三月十八号早上林墨第一次请我来 506 鉴定一样东西。我穿了藏青羊毛衫·不是白大褂。我进门前我想过要不要说这件毛衣是我自己织的·我决定不说。他看见我停了一拍·这一拍就是我今天要的情绪底。
- [EP07] 鉴定 40 分钟：人骨·幼儿 3-6 岁。凹槽是后磨的·一个成年人左手食指每天从左往右滑的方向·磨了很多年。凹槽起始位置的尺寸匹配一枚极小的银铃舌。这个磨痕·我看出来是**男性成年人**左手的痕（骨片背面那条辅助磨痕的指节长度）·但我没告诉墨弟。我说"女性幼儿"是骨片本身的判断·关于"谁磨的"我只说到"一个人磨了很多年"就停了。
- [EP07] 他问"是谁磨的"·我说"你想知道我会说到这里·不会再深"——我划界。他问"几粒"两个字没带台阶·我答"三粒半·三天"。他"够吗"我没说"不够"·我说"三粒半·三天·你算一下"。让他自己减。走前我回头对那个盒子说"墨弟·这个盒子·晚上别放你枕头底下"——这是我今天给他的三层台阶·不是金句·是警告。
- [EP07] 墨弟盯我看的时候我说"你盯着我看什么·扣上笔帽"。他扣了·咔一声·我的手停了半秒。这是我们从殷墟那几天就有的器物密码·他懂。

## 内心层更新（EP07 后）

- 骨片磨者是成年男性。我没告诉墨弟·因为一旦告诉·他会立刻想到他父亲。但磨痕的指节长度跟他父亲不匹配——是另一个人。我今天不展开这一条。
- 墨弟剩三天·他算出来了。他把骨片盒往自己这边拉了半寸·这一下我看见了。他今天选了骨片·没选我。这不是被选择·这是他必须的顺序。
- 我的药剩三粒半。三天。我今天穿藏青进他房间·是我留的信号——我快用完了·他如果要问我会回·他如果不问我不说。
- 桑桑朝南岭炸毛那件事·我还没告诉任何人为什么会炸。招魂堂那边的巫九娘和九歌司内部是有一个我不懂的协议的。这件事明天再说。
