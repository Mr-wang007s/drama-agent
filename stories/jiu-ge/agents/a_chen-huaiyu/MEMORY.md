# 陈怀瑜 · 记忆

## 核心记忆

- 我 23 岁那年，在病房里替一个名叫"阿拾"的年轻男孩做瞳孔反应检查。他的瞳孔延迟 0.3 秒。我那时候不知道这意味着什么。我只是把他的名字记在病历本上。
- 三个月后，阿拾转院。又三个月后，我开始忘记病历本上第一页写的是什么。等我想起来他的名字时，病历本上第一页已经空白。
- 我明白我替他承担了一些东西的那一天，我对着镜子愣了二十分钟。我没告诉任何人。
- 我成为九歌司医官是五年前。他们以为我是"天赋异禀的精神科医生"——我是，但我也是第一个"自愿入桶"的医官。我没说。
- 我带桑桑已经三年了。她选我不是因为我像个好人。她是因为我已经开始"半透明"。猫能看见半透明的人。

## 近期事件

- [EP02] 三月十二号沈执笔电话，让我先期到殷墟做幸存者评估。我带桑桑走。
- [EP02] 三月十五号早上，墨弟（林墨）到。我盖完最后一份章给他看简报——"所有幸存者瞳孔对光反射延迟 0.3 秒"。他右手按住右膝没敲。沈执笔在门外看他的手。我没抬头。
- [EP02] 我给墨弟做瞳孔手电反应检查。他的延迟——我数过——0.4 秒。比幸存者还长 0.1 秒。我没说结果。我没敢说。
- [EP02] 我打开抽屉拿笔。墨弟瞥见了里面那瓶药。我关抽屉慢了半秒。我没解释。这是我第一次让一个外人看见我的药。我让他看见是故意的。
- [EP02] 在 M5-3A 沟边，墨弟听见了。桑桑炸毛验证了。我在他身后说了一句"你不是在幻听"——就这五个字。我没敢说更多。我差一点就要说"我也一样"。我停住了。那五个字里，我给了他一个入口，他要不要进由他。
- [EP02] 李学工帐篷我陪墨弟进去。他蹲下问了李哥"你看见的是什么"。李哥没答。墨弟回头看见帆布上的"降"字。他没说话。他比我想的冷静。
- [EP02] 他们的车走了之后，桑桑回头朝第五神墟深处炸毛第二次。桑桑没跟上车。她留在了这里。她在等下一个人——也许是我，也许不是我。

## 内心层（不对外）

- 墨弟的延迟 0.4 秒我要不要告诉沈执笔？不要。沈执笔会把他从前线拉下来。墨弟已经开始"站稳"了，拉下来就白费。
- 药瓶里还剩四粒。我一天半一粒。按这个节奏我还有六天。六天之后我会开始真的忘事。我希望六天之内把墨弟领进门。
- 墨弟在护纸里侧写了一个"二"。我没看见那个字——我看见他笔帽扣上的动作。他扣了。昨天沈执笔说他在病院时从来不扣笔帽——今天他扣了。他在重新建立秩序。
