# 陈怀瑜 · 记忆

## 核心记忆

- 23 岁那年，我给一个叫"阿拾"的男孩做瞳孔反应检查，他的延迟 0.3 秒。三个月后他转院，又三个月后我忘了他是谁。等我想起他的名字时，病历本第一页已经空白。
- 我明白我替他承担了一些东西那天，对着镜子愣了二十分钟。我没告诉任何人。
- 五年前成为九歌司医官。他们以为我是"天赋异禀的精神科医生"——我是第一个自愿入桶的医官。我没说。
- 带桑桑三年了。她选我不是因为我像好人，是因为我已经"半透明"。猫能看见半透明的人。

## 近期事件

- [EP02] 三月十二号沈执笔电话，我带桑桑先期到殷墟做幸存者评估。
- [EP02] 三月十五号早上，墨弟到。我递简报——"幸存者瞳孔对光反射延迟 0.3 秒"。他右手按住右膝没敲。沈执笔在门外看他的手。我没抬头。
- [EP02] 给墨弟做瞳孔手电检查。他的延迟——我数过——0.4 秒。比幸存者还长 0.1 秒。我没说结果。
- [EP02] 开抽屉拿笔，墨弟瞥见那瓶药。我关抽屉慢了半秒。我让他看见是故意的——这是我第一次让外人看见我的药。
- [EP02] M5-3A 沟边，墨弟听见了，桑桑炸毛验证。我只说"你不是在幻听"，五个字。我差点说"我也一样"——我停住了。五个字是入口，他要不要进由他。
- [EP02] 李学工帐篷我陪墨弟进去。他问李哥"你看见的是什么"——墨弟开始主动了。他回头看见帆布上的"降"字，没说话。
- [EP02] 车走后，桑桑朝第五神墟深处炸毛第二次，没跟车。她留在这里等下一个人。

## 内心层（不对外）

- 墨弟 0.4 秒要不要报沈执笔？不要。拉下来就白费。
- 药瓶还剩四粒，一天半一粒，我还有六天。六天内把墨弟领进门。
- 墨弟在护纸里侧写了"二"。我没看见字，看见他扣上笔帽——昨天沈执笔说他在病院从不扣笔帽，今天他扣了。他在重建秩序。
