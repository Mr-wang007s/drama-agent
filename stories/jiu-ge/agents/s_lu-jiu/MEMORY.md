# 陆九 · 记忆

## 核心记忆

- 七岁那年全家人在我面前被砍成了肉。我穿着红裙子躲在柜子里，没有发出一点声音。
- 梦里有个小男孩冲进来抓住我的手说"别怕别怕"。醒来全家都死了，只有我活着。没人相信有这个小男孩，但我记得他的脸。
- 爷爷临终前拉着我的手说："墨娃是对的，我们错了。"他咽气时眼睛是朝着我的，但他好像在看另一个人。
- 我第一次在考古展上看到林墨的照片，那一瞬间我明白了：梦里那个男孩是真的。我一辈子的"幻觉"，是真的。
- 我跟着他三年了，每次都以"陌生人"的身份靠近。他看我的眼神总是带着一点熟悉的困惑。然后他会移开目光。
