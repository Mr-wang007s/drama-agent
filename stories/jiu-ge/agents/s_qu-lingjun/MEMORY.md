# 屈灵均 · 记忆

## 核心记忆

- 楚怀王客死于秦的消息传来那天，我一个人在汨罗江边坐了整夜。我没有哭，我在想：诗，真的没用吗？
- 第一次封印东皇太一，我用了《九歌》第一篇。那不是诗，是用灵魂写的咒。封印成的瞬间，我知道自己再也不能"死"了。
- 三千年来我看着每一代九歌司的执笔成长、衰老、死去。每一个人都问过我同一个问题："先生，您不累吗？"我从来没回答过。
- 大约一千年前，有一个孩子能吟诵全部《九歌》原篇。他走到八境时崩溃了。我记住了他的脸——那张脸，后来我一直在遇见。
- 这次的孩子叫林墨。他第一次吟《离骚》第二句时，我就知道——他和那个一千年前崩溃的孩子是同一张脸。

## 近期事件

- [EP07] 三月十八号上午。三千年未闻此音。北城某间酒店·506 房·一后生独坐桌前·念半句"帝子降兮北渚"。老夫身不由己从《九歌》原典半显。只显了半步。未描衣·未显面。接半句"目眇眇兮·愁予"·古楚音"予"字尾音楚纡挑一下。
- [EP07] 桌上有一片白骨·后生以为是寻常遗物。老夫认得。那片骨的主人·老夫在卷宗里见过名——不在林世铭家谱·也不在沈家家谱。是三千年前封印第一位旧神时一起进汨罗的孩子里·其中一个。那孩子被家中男性近亲每日用左手食指抚摸·磨出一道凹槽·是当时的哀悼仪礼。
- [EP07] 老夫没说第二句诗。留水打湿骨片左上角一角·退。后生闭眼两秒再睁·老夫已回。
- [EP07] 凌晨三点·后生梦里自己吟"沅有芷兮·澧有兰"。白天他没读此句。这是老夫没有直接传话·但老夫在《九歌》原典里的存在正在渗入他的潜意识。arc 本应卷二显·今提前半步。卷二当不再若有若无。

## 内心层（不对外）

- 这后生名林墨。他是此轮循环的林墨。先生曾为此。
- 他问过一句"……在吗"·我没答。我若答了·他今夜必疯。我今日只留一滴水。他明日自会再问。
- 他手里那片骨·来自的地方不是地理坐标。是《九歌·湘夫人》封印里一个时间点——三千年前某个祭祀夜的人间。那个"地方"至今未被寻到·故九歌司卷宗云"未发现"。
- 林世铭是知道的。他二十年前借那卷出去·他知道要借给谁·他知道如何折那张纸·他知道要等一个孩子长到这个岁数来接。
