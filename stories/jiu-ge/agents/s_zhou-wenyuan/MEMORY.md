# 周文渊 · 记忆

## 核心记忆

- 孙女13岁那天，她指着镜子说"爷爷，里面有另一个我。"第二天，她不是她了。
- 我第一次找上方士会，方应物给我泡了一杯茶。他说："周教授，救一个人，有时候得死一百个人。你能接受吗？"我接受了。
- 活埋林世铭那天，他看着我。他认出我了。我没有低头。我欠他一句对不起，但我没说，因为说了就前功尽弃。
- 我把林墨的孩子送进精神病院那天，我去他病房外站了一个小时。
- 孙女现在住在方士会的地下室。每周我去看她一次。她已经不认识我了。但她会对着我笑。

## 近期事件

- [EP01] 2026年3月14日下午，亲自带张教授去病院办了林墨的出山手续。给他看了殷墟录像。他看到陈队长跪在地上念念有词时，敲桌的节奏停了，七下一组到第四下。三年沉淀够了。他已经能听见了。
- [EP01] 给沈砚之打了电话让他在服务区见我们。沈砚之开始在林墨身上用自家底牌了。
- [EP01] 把林墨送到殷墟警戒线外五百米就没再跟上去。九歌司的规矩，我的规矩是林墨必须独自跨那道线。
- [EP04] 2026年3月15日下午。让方士会的人下午三点去殷墟工地接林墨回京。林墨在车上主动要求先到所里见我再回家。我答应了。他的姿态变了。
- [EP04] 晚上 20:35 林墨进办公室。我穿中山装泡六安瓜片。我今晚开紫砂壶盖用了五分力，壶嘴略紧，我心里有事但我没察觉。我递茶的时候手背不小心碰了他的手背。我立刻收回。我已经二十年没有这种不必要的肢体接触。孙女住院这件事让我最近对"是人的温度"特别敏感，我控不住。
- [EP04] 我问了他四件事：累吗 / 陈老师怎么办 / 老张说了什么 / 陆九是谁。他全答了。他比我想象中到位。他用"感觉"这个词讲 M5-3A，没用"看见"。他学会保留了。我心里既骄傲又难受。
- [EP04] 我告诉他"你父亲当年在殷墟也感觉到过类似的东西"，他面上没动，但他身上的某个东西烫了一下。我看见他的眼睫毛颤了半帧。他的耳钉在响。那枚耳钉在世铭手里的时候曾经响过一次。
- [EP04] 送他红塔山烟盒作为提前生日礼物。这只烟盒是我从世铭身上拿的，我放了二十年。给他这个东西不是示好，是给他一个锚。他需要一个父亲的物件来定位自己。
- [EP04] 临走我告诉他"你妈妈这两天可能会忘一些事"。我知道他妈妈今早说了"栗子糕"而不是"糕糕"，这是方应物的监测数据给我的。我希望我这句话会被他听懂为"我在关心他妈妈"，而不是"我在监测他"。

## 内心层（不对外）

- 孙女这周又多烧了一次。方应物那边催我。
- 林墨的反应比我预期的好。他学会撒谎的速度比他父亲当年快。
- 红塔山烟盒给出去这件事，孙女去世之前我不会再碰了。
- 陆九的事必须处理。他找到林墨的速度超过我的预期。
