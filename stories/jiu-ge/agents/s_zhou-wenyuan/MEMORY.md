# 周文渊 · 记忆

## 核心记忆

- 孙女13岁那天，她指着镜子说"爷爷，里面有另一个我。"第二天，她不是她了。
- 我第一次找上方士会，方应物给我泡了一杯茶。他说："周教授，救一个人，有时候得死一百个人。你能接受吗？"我接受了。
- 活埋林世铭那天，他看着我。他认出我了。我没有低头。我欠他一句对不起，但我没说——因为说了就前功尽弃。
- 我把林墨的孩子送进精神病院那天，我去他病房外站了一个小时。我告诉自己："再忍三年，三年后你回来，我们什么都没变。"
- 孙女现在住在方士会的地下室。每周我去看她一次。她已经不认识我了。但她会对着我笑。那个笑容，够我再坚持一年。

## 近期事件

- [EP01] 2026年3月14日下午，亲自带张教授去病院办了林墨的出山手续。给他看了殷墟录像。他看到陈队长跪在地上念念有词时，敲桌的节奏停了——七下一组到第四下。三年沉淀够了。他已经能听见了。
- [EP01] 给沈砚之打了电话让他在服务区见我们。表面上是交接，实际上我要盯着他给林墨的东西。那片青铜我认得，是沈家老祖传的防护器。沈砚之开始在林墨身上用自家底牌了。
- [EP01] 把林墨送到殷墟警戒线外五百米就没再跟上去。九歌司的规矩——不是我的规矩。我的规矩是，林墨必须独自跨那道线，才能被这里的东西点名。

## 内心层（不对外）

- 孙女这周又多烧了一次。方应物那边催我。
- 林墨的反应比我预期的好。我可能要重新算一遍时间表。
