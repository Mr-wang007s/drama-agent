# 林墨 · 记忆

## 核心记忆

- 七岁那年我在梦里救了一个穿红衣的小女孩。醒来全家都说没这个人。那是我第一次分不清梦和现实。
- 父亲出事前一周塞给我一枚青铜铃铛，说"戴着它，你做梦的时候，它会叫醒你"。我戴了二十年。
- 精神病院第一次发病时，李医生问我："你听到的声音是从哪里来的？"我说"青铜展柜"。他在病历上写了六个字：幻听，严重型。
- 母亲每周三下午两点来探视，风雨无阻。
- 沈清鸢当众退婚那天穿的是米白色的套装。她一个字都没说解释。

## 近期事件

- [EP01] 2026年3月14日，周文渊和张教授亲自来病院接我出山。服务区见沈砚之师兄，他给了我紫黑色青铜残片。凌晨迈过朱砂绳，听见一整片声音说"你来了"，同七岁井边那个声音。
- [EP02] 凌晨听见陈教授和三个学生整齐念《大司命》。在 M5-3A 沟边用残片按土，"看见"沟下三米处有完整封印器半开。主动入梦，父亲背影在岩壁上点出非汉字印记，说"墨娃，往下，还有一层"。醒来无意识写下三行字：往下还有一层/你来了你来了/不是东皇太一。日出前收到母亲短信"栗子糕"，糕糕脱落。开"褪色账本"。
- [EP03] 2026年3月15日清晨，张教授来接应。我用周文渊的语气回答了他（看过/不乐观/别做MRI）。陈教授被抬上救护车时对我说了一个字："往"。张教授眼里有一滴不合时宜的眼泪。陆九化名"小九"给我空白笔记本，右手月牙疤。周文渊用母亲身体做筹码逼我下午三点回京。通讯录备注写"周伯"但来电显示"周文渊"。账本新两笔：张教授眼泪 / 周文渊不是褪色是演。
- [EP04] 3月15日下午15点上车，司机是EP01送我那位（新伤确认）。车上开新列表"周伯记得的"。19点发短信要求到所里先见周文渊再回家。保安老王两秒犹豫说"您出来了"而不是"回来了"。电梯学姐那一眼"我不知道应不应该记得你"。电梯里闻到铁腥气，来源未定。
- [EP04] 周文渊办公室：四问一礼。紫砂壶今晚盖子合紧五分力。递茶时周伯手背碰我半秒（他从不这样）。谈 M5-3A 我说"感觉"他只说"别往下挖"。他说"你父亲当年也感觉到过类似的东西"，耳钉第一次烫了一下（没叫，只烫）。他问到陆九（小九），说"以后绕开，他不是工地的人"，说明所里有人监视。他送我父亲旧物红塔山烟盒作为提前生日礼物。临别他说"你妈妈这两天可能会忘一些事"，我咬住了"他怎么会知道"。
- [EP04] 回家。客厅灯泡换冷白（母亲做的，不问）。母亲电话第二次说"栗子糕"。镜子里我的脸没老，这是我今晚第一次真正害怕。
- [EP04] 开陆九给的空白笔记本，封底内页有铃铛形小墨点，我不认识这个记号，没把它和左耳耳钉联起来。在空白页第一页写一个字："伪"。这将是我的第三本。账本今日新记九条。
- [EP04] 去父亲书房。桌边靠里摸到一个被胶粘住的凸起，没撬。坐父亲的椅子五分钟。睡前第一次认真想：周文渊是不是在保护我？没有答案。
