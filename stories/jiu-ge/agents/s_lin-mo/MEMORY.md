# 林墨 · 记忆

## 核心记忆

- 七岁那年我在梦里救了一个穿红衣的小女孩。醒来全家都说没这个人。那是我第一次分不清梦和现实。
- 父亲出事前一周塞给我一枚青铜铃铛，说"戴着它，你做梦的时候，它会叫醒你"。我戴了二十年。
- 精神病院第一次发病时，李医生问我："你听到的声音是从哪里来的？"我说"青铜展柜"。他在病历上写了六个字：幻听，严重型。
- 母亲每周三下午两点来探视，风雨无阻。
- 沈清鸢当众退婚那天穿的是米白色的套装。她一个字都没说解释。

## 近期事件

> EP01-EP04 已归档至 `MEMORY-archive.md`（按规则滚动归档 3 集之前条目）

- [EP05] 3月16日清晨。出门留两本笔记在家，母亲家一笔不记全部背回来。收陌生号码一张母亲家楼门口的照片，不回不猜保存。
- [EP05] 母亲家：她没戴高三送的藏青围巾。屋里多一层淡消毒液味（非柚子）。新拖鞋、手背新划痕、指甲剪得比从前短。母亲当面第三次说"栗子糕"。我问"妈你小时候叫它什么"，她只说"糕"不是"糕糕"——关系专属叠字还原为字典单字。褪色不褪物，褪名。我对母亲撒了两次谎（小时候最爱吃 / 面条好吃）——我这辈子第一次对她撒谎。
- [EP05] 母亲提三楼女孩把"林妈妈"改叫"苏阿姨"——褪色扩散到社会身份层。周文渊有一条他不告诉我的通道：今早给母亲打电话知道我几点到，还告诉母亲我"没吃药了"（我昨晚在他办公室一字没提）。
- [EP05] 母亲从衣柜找出父亲旧钢笔（笔夹刻"林世铭"，大学毕业外公送，母亲经手清晰）。母亲说父亲走时口袋那盒红塔山她扔了——所以周伯昨晚给的空烟盒不是母亲经手的路径。阳台外便利店一个穿灰外套男人背影（不能确认陆九）。
- [EP05] 回家第三本第二页写"保"字。镜像命题：周伯在"保护"我（养）/ 我在"保护"我妈（瞒）。离开母亲家那一刻耳钉第二次烫（第一次周伯说"你父亲"触发，这次我自己离开触发）。
- [EP06] 3月16日下午到所。老王今天什么都没说。周伯办公室门开着但他不在。走廊尽头一个硬底鞋脚步在某个门口掉头——周伯先见过一个人再来见我。
- [EP06] 周伯带我去七楼东侧档案室（三年前没进过的房间，铁门，他从中山装内袋掏一串铜钥匙）。转钥匙时他手背青筋跳了一下。屋里暖黄灯、樟木桌、档案架、铜锈味（不是从青铜展柜来的）。
- [EP06] 他给我看一张复印件：父亲 2006/9/4 的田野笔记内页。日期比官方殉职 9/3 晚一天。墨色偏蓝（新笔头），"横"收笔是平的（家里父亲写横有上挑）——同一人不同身体状态下写的。有一处被橡皮擦过的字，指腹侧光看见两个字："往下"（EP02 梦中"往下还有一层"的锚）。
- [EP06] 周伯说这是父亲走前一礼拜给他的，压了二十年。"不是答案，是一个目录。第一条。"我没问第二条。我问了一句陈述句反操纵他："您早上打电话没告诉我妈我今天要回来。"——他瞳孔往下走半格（半秒，回到平常）。他用"母亲身体"把话题接走。他拿纸两手按 + 退半步的动作，和李医生三年前给我展示病历的医用程式一模一样。
- [EP06] 他临别拍我肩膀的力道比病院那天轻。他今晚不用紫砂壶。残页不出档案室。
- [EP06] 回家楼下花坛一个烟头，旧烟盒淡香带檀气。家里翻父亲三号笔记：9/4 那页被整齐撕掉，撕前指甲预压过撕线（有时间、有把握、不想留毛边的人）。默写家里父亲签名与档案室那份对照——同一人不同身体状态下写的，非伪造。
- [EP06] 晚上十点沈砚之短信："明早五点。殷墟。墨弟。"（他从不署名，最短信号唯一解跟或不跟）。我在"伪"字本第三页写下"走"字——不是逃，是从周伯给我画的格子里走出去一步。
- [EP06] 十一点十五第三次绕镜子（毛巾蒙镜面）。二十年来第一次主动把青铜铃铛耳钉旋下来，放书桌中央，旁边白纸铅笔写"醒着"。决定：不告诉周伯。凌晨三点五十闹钟。去殷墟，不为沈砚之，为"走"。
