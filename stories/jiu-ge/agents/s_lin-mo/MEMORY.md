# 林墨 · 记忆

## 核心记忆

- 七岁那年我在梦里救了一个穿红衣的小女孩。醒来全家都说没这个人。那是我第一次分不清梦和现实。
- 父亲出事前一周塞给我一枚青铜铃铛，说"戴着它，你做梦的时候，它会叫醒你"。我戴了二十年。
- 精神病院第一次发病时，李医生问我："你听到的声音是从哪里来的？"我说"青铜展柜"。他在病历上写了六个字：幻听，严重型。
- 母亲每周三下午两点来探视，风雨无阻。有一次她迟到了十分钟，我在走廊里掉了半个小时眼泪——我以为她也终于不要我了。
- 沈清鸢当众退婚那天穿的是米白色的套装。她一个字都没说解释，只是把戒指放在桌上，转身走了。

## 近期事件

- [EP01] 2026年3月14日，周文渊和张教授亲自来病院接我出山。他在办公室放了一段殷墟考古队精神异常的监控录像给我看，说陈队长来电求助——他们在"听"青铜器。我没告诉周伯我自己早就在听了。
- [EP01] 京广高速服务区见到沈砚之师兄。三件套黑西装，还是那个温度。他给了我一片紫黑色青铜残片，说到了工地戴在身上能挡一些。我指腹碰到它的时候，耳朵深处响起极细的哼鸣。我没告诉师兄。他转身时影子比他人淡一截，我也没说。
- [EP01] 凌晨迈过殷墟外围的朱砂绳，鼓膜像被人从里面弹了一下。一整片声音同时说"你来了"。这声音是七岁那年井边红衣小孩说过的。我没告诉任何人。
- [EP02] 2026年3月15日凌晨，第五神墟工地。我听见陈教授和三个学生整齐念《九歌·大司命》——陈教授这辈子不研究楚辞。我没冲进去救他，我选择先判断。发掘沟边第五只陶碗裂了一道从内向外的纹。
- [EP02] 我在 M5-3A 沟边用沈师兄给的青铜残片按住土，听见十四下一组的节拍——我父亲教我的七下一组的两倍。我脑子里出现了一张图：沟下三米处有完整封印器，半开。我承认了一件三年来每次冒出来都被我按下去的念头——我不是只能听，我能用。我笑了一下，隔了三年零一个月。
- [EP02] 帆布板房里我主动入梦。井边红衣小孩对我笑，像铃铛。我坠到第二层，一个绿色盐晶的地下空间，闻到父亲的老红塔山味。父亲的背影在岩壁上点了一个印记，那不是汉字也不是任何古文字。父亲没转身。他说："墨娃，往下，还有一层。"醒来发现我不记得自己写过的三行字：往下还有一层 / 你来了你来了 / 不是东皇太一。
- [EP02] 日出前收到母亲短信"今天做了栗子糕"。从我五岁起她就叫那种糕为"糕糕"，四十七条聊天记录都是。这是第一次写"栗子糕"。李医生证实她昨天下午来过医院。糕糕两个字消失了，她记得做、记得带、记得问，她不记得那个叫了二十八年的词。这是我的褪色第一次有名字。我开了梦境笔记新的一页，写上"褪色账本"。
