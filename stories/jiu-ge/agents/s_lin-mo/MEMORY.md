# 林墨 · 记忆

## 核心记忆

- 七岁那年我在梦里救了一个穿红衣的小女孩。醒来全家都说没这个人。那是我第一次分不清梦和现实。
- 父亲出事前一周塞给我一枚青铜铃铛，说"戴着它，你做梦的时候，它会叫醒你"。我戴了二十年。
- 精神病院第一次发病时，李医生问我："你听到的声音是从哪里来的？"我说"青铜展柜"。他在病历上写了六个字：幻听，严重型。
- 母亲每周三下午两点来探视，风雨无阻。
- 沈清鸢当众退婚那天穿的是米白色的套装。她一个字都没说解释。

## 近期事件

- [EP01] 三月十四号早上桌上多了一张社科院特赦令。保安老王今天没看我（被忘账第 37 笔）。
- [EP01] 李医生在面谈室把病历合上，递我私人电话卡片，他说"万"就咽回去。我说"那个青铜柜子"也咽回去了。他的笔搁平不扣帽——他不想再写任何东西。
- [EP01] 母亲今天叫我"糕"，少一个字。她做了栗子糕四块（以前是五块）。她伸手想摸耳钉，手停在我耳边半空里没摸。她走时把保温饭盒忘在椅子上。我在心里叫了她一声"妈，我没走，是你忘了我戴这个"——我没出声。
- [EP01] 周伯桌上放了一盒红塔山空烟盒（我父亲抽这个牌子）。我只看了 0.3 秒就把目光收回。我没问。我耳钉响了一下。
- [EP01] 周伯放殷墟陈队长录像——他敲膝盖七下一组到第四下停。我的右手食指同时在桌下敲到第四下停。我没发觉我停了是因为我的手自己停的。周伯说"你看见了"。我说"我去"。
- [EP01] 签完聘书在门口我把手放进大衣口袋。两边口袋都是空的。我的身体已经忘了口袋这个东西（病号服没口袋）。我对自己说了"好"。
- [EP01] 下午服务区沈师兄接我。他大衣没被风吹起。夕阳下他的影子比我的淡一度。我没说。车上雨刷第一下摆过，我耳钉响了。他听见了，他没说。
- [EP01] 今天我开了两本日记：一本写"有人来"（三个字），一本算了今天 9 笔事件。我给它们都没命名。今天不命名。

- [EP02] 三月十五号早上，殷墟外围。我下了车，新鞋落灰。沈师兄的鞋也落灰——昨天他的鞋还是白的。我没说。
- [EP02] 工棚里遇到陈医官（陈怀瑜）和一只黑猫桑桑。桑桑看我第一眼就咕噜了一下。陈医官给我读幸存者简报："所有幸存者瞳孔对光反射延迟 0.3 秒。" 我没问我自己的延迟多少。
- [EP02] 陈医官给我做了瞳孔手电检查。她没说结果。她开抽屉拿笔时，我瞥见里面有一瓶和我大衣内兜里那瓶同款的药。她关抽屉慢了半秒。我没问。
- [EP02] 沈师兄从门缝看了我一眼。我的右手按住右膝没敲——今天一整天我都没敲。
- [EP02] M5-3A 沟边。耳钉震动一下。一个字落进我脑子里：嘉。我没说。桑桑在我身后炸毛了，她没叫。陈医官在我身后一步说"你不是在幻听"。她没解释。我回了一个字"嗯"。
- [EP02] 李哥（李学工）帐篷。他睁眼看我但眼泪一直流，泪腺正常但不是哭。他说话时胸口停一下不呼吸三秒再抽气。他看我身后——帆布内侧从左下到右上刻满同一个字：降。我问了"你看见的是什么？"他没回应。
- [EP02] 回程副驾上。我翻开"2023 冬"想记今天的事，笔悬在纸上三秒——我合上了。我没写嘉。我翻到封面里侧，想起昨天在服务区我给自己数七件事之前也停了三秒。我在里侧写了一个数字"二"。我扣上笔帽。
- [EP02] 妈昨晚发来短信："糕，路上……"我没点开。我把手机反扣在腿上。
- [EP02] 后视镜里桑桑又炸毛了一次，朝殷墟深处某个方向。我和沈师兄都看见了，都没说。我低头看了自己的鞋和他的鞋——两双都有灰。
