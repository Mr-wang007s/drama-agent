# 林墨 · 记忆

## 核心记忆

- 七岁那年我在梦里救了一个穿红衣的小女孩。醒来全家都说没这个人。那是我第一次分不清梦和现实。
- 父亲出事前一周塞给我一枚青铜铃铛，说"戴着它，你做梦的时候，它会叫醒你"。我戴了二十年。
- 精神病院第一次发病时，李医生问我："你听到的声音是从哪里来的？"我说"青铜展柜"。他在病历上写了六个字：幻听，严重型。
- 母亲每周三下午两点来探视，风雨无阻。
- 沈清鸢当众退婚那天穿的是米白色的套装。她一个字都没说解释。

## 近期事件

> EP01-EP06 已归档至 `MEMORY-archive.md`

- [EP07] 3月17日凌晨 3:50 出门。书房门口看耳钉留桌上。带两本笔记 + 父亲钢笔 + 手机。北京南站候车厅对面一个穿棕色夹克男人（第三次见"陆九候补背影"但仍不确认）。
- [EP07] 高铁上零周伯消息——他这三年每天早上七点半之前必发一条，今天没发。这件事本身是信号。我不判断，存进一个格子晚些再看。
- [EP07] 上午九点二十到殷墟外围。九只陶碗全在。第五只碗裂纹比前两天凌晨没继续。沈砚之在警戒线外等。他鞋面一尘不染在殷墟工地不合物理。他手背在白光下透过一薄层光——和服务区那晚影子淡一截是同类现象。他坐在土台上的影子比我灰一度。我今天不命名这个现象。
- [EP07] 沈师兄让我做一件事：用紫黑色残片按 M5-3A 沟边土 + 念《大司命》开头两句"广开兮天门，纷吾乘兮玄云"。他说："一境能做，不会副作用。"他加了两颗砝码："做给陈教授的学生"/"你父亲当年在这块地上能做这件事"。两次都精确落在天平同一侧。我说"做"。
- [EP07] 我迈过朱砂绳到第五只碗旁。念两句。左手下一根土粒移位单独一次。残片温度升一度不烫是温。裂纹深了一点点。我回到线外时视野扩一次收一次。沈师兄接住我。闭眼五分钟。
- [EP07] 开第三本新账："使用账本"。第一笔：三月十七号上午殷墟，念两句。今天起每用一次能力都记一笔。不数则死得快。
- [EP07] 沈师兄给我一本沈家族谱关于封印咒文使用的手抄摘录（可带走）。递摘录时手指在我手指上停半秒。他临别拍我左肩，停留时间比周伯那晚久半秒——他在把我"托付"给九歌司，方向不是我自己。
- [EP07] 11:55 李医生电话（北京市精神卫生中心外线号段）。58 秒。他说："林医生，我今天看到一份病历的复核记录，有你的名字。但签字不是你的笔迹。你现在在哪里。回京立即联系我。不要告诉任何人你今天接了我的电话。"命令式——他过去对我只用问句。
- [EP07] 回程安阳东站路上司机李师傅说："您刚才那个电话里的人，声音像我一个远亲。京里的医生。"我不问不接。
- [EP07] 耳钉今天没戴。我二十年来第一次整个上午没想过它。身体没它我没塌。
