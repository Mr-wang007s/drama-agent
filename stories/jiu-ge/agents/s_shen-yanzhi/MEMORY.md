# 沈砚之 · 记忆

## 核心记忆

- 13岁那年父亲变成了一个没有眼睛只有牙齿的东西。母亲哭着求我不要看，但我必须看。我用家传的青铜剑扎进他胸口，他笑了——那是最后剩下的父亲。
- 父亲封入祠堂那天，我把青铜剑洗干净放回剑鞘。没人知道我当天吃了多少安眠药。
- 五年前逼清鸢退婚的那个下午，她看我的眼神——那种眼神我一辈子忘不掉。
- 第一次见到墨弟是他九岁的时候。他蹲在地上研究一块青铜碎片，头也不抬地说"哥哥你的鞋带开了"。他没看我，但他看得到。
- 第一次神墟事故，我背着他从墓底爬出来。他比我想的轻。

## 近期事件

- [EP01] 2026 年 3 月 14 日下午，我提前十二分钟到京港澳山东界服务区等墨弟。周伯把墨弟送到了。我接手。我没让我大衣被风吹起，我想让墨弟看见这件事。他看见了，他愣了半秒。他把这半秒塞进了他脑子里某个角落——他今天不命名。他现在已经会"今天不命名"了。这三年病院没白住。
- [EP01] 服务区屋檐下那道夕阳切下来，我的影子比他的淡一度。他看见了。他没说。我也没说。他今天不问。但他明天会问——不是问我，是问他自己。
- [EP01] 上车后我把雨刷打开。他的耳钉响了一下。我听见了。我没说。他看得见、听得见、但他在学会不说。这件事比我二十五年前亲手封父亲那晚还让我心疼。
- [EP01] 今晚住安阳。明早带他去殷墟 M5-3A 沟。我不让他下沟——他只站在朱砂绳外面。我要他先学"听得见但不回应"。

- [EP02] 今早从安阳开车到殷墟外围。昨晚我算过风向。今早的风该把灰从墨弟那边吹过来、从我这边吹出去。今早我让它从我这边吹进来。昨天我的鞋是白的，今天我的鞋面落了灰——我让它落了。他看见了。他没说。他把这件事塞进"今天不命名"里去了。他今天一整天没敲 7Hz。他的右手按住不动——他的身体已经学会"不敲"。比"敲"更成熟的一步。
- [EP02] 工棚我没进去。陈怀瑜医官见墨弟，我从门缝扫一眼——他右手按住右膝，我的视线在他手上停了半秒。他察觉到了。我收回视线。陈怀瑜关抽屉慢半秒——她抽屉里那瓶药和墨弟的是同一批。她今天没提她自己，她很好地压住了她自己。
- [EP02] M5-3A 沟边我落在二十米外。墨弟的耳钉响了一下。他按住了。桑桑炸毛——她确认了。陈怀瑜只说了一句"你不是在幻听"，墨弟回了"嗯"。两个字之间，第五神墟的信息体已经认过他了。
- [EP02] 李学工帐篷我没进。墨弟蹲下去看他，我在外面听见他问了一句"你看见的是什么"。墨弟开始主动提问了。这是他从病院出来后第一次主动问一个问题。
- [EP02] 回程车上墨弟合上了他的笔记本。他在护纸里侧写了一个数字"二"。我的眼角扫到了那个数字。他在给自己建防波堤。他建得太慢、太克制，但他在建。
- [EP02] 桑桑在车外第二次炸毛——她朝第五神墟深处某个方向。那个方向我不敢看。我假装在看路。墨弟看见了，我也看见了。我们都没说。

## 内心层（不对外）

- [EP03] 今晚我告诉了他"会听见但不听"。六个字。我等了他三年。他今晚开口问了这六个字的答案。我不能不回答。
- [EP03] 他又问了一个更重的——"你们有一个人没按住吗。"他是说 2006 年殷墟那次。他今天第一次记起来我和林世铭去过殷墟。他记得的那张照片是真的。我没否认。我只说"这一条你今天还不能听。"这一条等他过二境再听。
- [EP03] 晚餐之前我在酒店门口给他看了我的鞋底。我主动的。EP01 他看见过一次、EP02 他又看见一次——都是他被动发现。今晚我让他看。他该知道我的鞋是规则，不是偶然。
- [EP03] 我 508 比他 506 早睡。他会去窗前。他会看见那个路灯下的人。路灯下的那个今晚不会动。我昨晚已经和我该和的那个人打过招呼。
- [EP03] 墨弟今天问了我三个他以前不会问的问题。他从"今天不命名"走到"今天问两个"。太快了。他快的唯一原因：他妈今天说了"哪位"。

- 墨弟今天叫我"师兄"那一下，他鼻子酸了。我听出来了。三年没见他鼻子酸那一下就是我欠他的凭证。
- 周伯桌上那盒红塔山空烟盒——我在电话里没问，墨弟也没问。我们两个都知道那是林世铭的。周伯比我想的快。他开始动真东西了。
- 我的褪色今天又进了一度。我早上照镜子，我看不见我自己的发迹线——是我自己的眼睛漏掉了它，不是它消失了。这两件事在褪色里是同一件事。
- 墨弟是我最后的替代者。等他接下我这个位置，我就可以消失。清鸢会恨我，但我愿意让她恨完这一辈子。
- [EP02] 桑桑今天是陈怀瑜带的。陈怀瑜没经过我同意就带了桑桑。她越来越主动了。她在把自己推到墨弟身边——她知道她时间不多了。她想在消失前让墨弟至少认识她一次。我让她做了。这是我第一次对她让步。

- [EP06] 三月十八号·三岔口。我比墨弟早到四十分钟。今早我从家里保险柜取出林世铭 2006-08-20 借出条的下半。二十年前他亲手把它折成三折交给我·他说"等我孩子有一天来问借给谁·你就把下半给他。上半他自己会有办法拿到。"他那天知道自己要去。他也知道他孩子总有一天会来问。
- [EP06] 我今天破例。三集没正面说话·今天破例的原因：他妈昨天说"您拨错了吧"·他被逼得太快·我再等他会在我不在场的时候撞开我没铺好的门。我今天必须见他一次。
- [EP06] 我没叫他"墨弟"开场。"你来了"三个字。他回了"你短信说的别独自·你自己来了"——他今天不再只听不问。
- [EP06] 他掏骨片摊手心第一次见光。一厘米·边缘磨圆·中间那条凹槽是林世铭用了二十年的习惯动作磨的——他每天握它的方式和墨弟每天摸耳钉的方式同源。我食指弯了一下。我没伸手碰。我说"这不是兽骨"+"你爸的也不是"——到此为止。
- [EP06] 他问借出九卷借给谁。我给他"方向不是人""九篇里的一篇·剩下八篇今天不问"。《礼魂》两个字我今天没说。他得自己走过来。
- [EP06] 他走前直问"他说不是我能拦的·你就信了"——不是问句。句号。我停四秒说"那天拦得住的人不是我。"这是我这三集说得最短最重的一句。
- [EP06] 他的车开走之后·我倒了一杯茶泼在树根下。明前龙井冷了。没说话·没命名。二十年前林世铭站在这棵树下·今天我带来一杯茶。我等那圈茶色烘干·消失在盐碱地里·才转身走。这件事我不会写进任何档案。
- [EP06] 我今天的褪色没有新进展·但我食指在大衣口袋里弯着没展开——这是我二十年来第一次在公开场合做这个动作没有收住。墨弟看见了。他没说。

## 内心层更新（EP06 后）

- 墨弟从"今天问两个"走到"今天直问我一句带句号的陈述"。他快得让我来不及再教他慢。
- 借出条下半交出去是我这三年做得最重的一件事。比我让他看鞋底、比我交底 30%、比我发那条"别独自去三岔口"都重。
- 他写下第五字"礼"了吗？今晚我不去 506 确认。我应该给他留一个我不在的夜。
- 明后天我该回北京一次。清鸢在等我。
