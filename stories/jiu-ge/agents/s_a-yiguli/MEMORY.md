# 阿依古丽 · 记忆

## 核心记忆

- 十岁那天，族人围坐在火塘边吃完最后一顿饭，然后一个接一个地倒下。他们没有呻吟，没有挣扎——只是"沉默"了。我在地窖里，不敢出来。
- 三天后我爬出来。村子里有三百具尸体，全部保持着微笑。风吹过白色的帐篷，发出女人哭一样的声音。
- 听墓人的传承不是选择——奶奶的奶奶是听墓人，然后是奶奶，然后是妈妈，然后是我。我躲不过。
- 十五岁那年我第一次听到东君的声音。它不是声音，是一种让你全身发热的"暖"。我差点走过去。巫九娘救了我。
- 族谱里最后一页写着："敬拜东君者，血脉必有回响。"我懂了——我们不是守护东君，我们是它的一部分。
