### 《九歌》· 世界观

- **题材**：考古悬疑 / 东方克苏鲁 / 规则怪谈
- **一句话故事**：一个天生免疫旧神的考古博士，每用一次能力就会让一个认识他的人忘记他。
- **核心主题**：真相 vs 秩序 / 记住 vs 被忘记 / 免疫的孤独

### 核心设定

#### 一、旧神本质

"旧神"不是神，是**寒武纪之前的先住民**——一种以高维信息形式存在的古老生命。它们的"存在"本身会污染人类的神经系统，**看见即疯**。

《山海经》《楚辞·九歌》里记载的那些神祇（东皇太一、云中君、湘君、河伯、山鬼……）全部是真实的旧神名册。上古祭祀仪式本质是"投喂"——用活人换取旧神不主动侵蚀人类。

#### 二、青铜防毒面具

三千年前，古人发现青铜器的特定几何结构可以"**承接**"旧神的信息体，将其封印在器物内部。每一件国宝级青铜器都是一座小型精神防护罩。

**禁忌**：出土必须举行"封印仪式"（即考古界偶尔流传的"开棺诗"），否则旧神会沿着解封的通道渗入现场人员的精神。

#### 三、九座神墟

中国大地下埋着九座"神墟"，每座对应《九歌》一篇，封印着一位完整的旧神：

| 神墟 | 对应篇章 | 所在地 | 状态 |
|------|---------|-------|------|
| 第一神墟 | 东皇太一 | 三星堆 | 封印稳固 |
| 第二神墟 | 云中君 | 良渚 | 轻度泄漏 |
| 第三神墟 | 湘君 | 洞庭湖底 | 裂缝中 |
| 第四神墟 | 湘夫人 | 九嶷山 | 未发现 |
| 第五神墟 | 大司命 | 殷墟 | 被考古队误触(EP01事件) |
| 第六神墟 | 少司命 | 二里头 | 封印中 |
| 第七神墟 | 东君 | 嘉峪关外 | 阿依古丽族人所在 |
| 第八神墟 | 河伯 | 兰州黄河底 | 已被炼化(方士会) |
| 第九神墟 | 山鬼 | 南岭深山 | 招魂堂守护 |

**终墟**（隐藏）：《九歌》还有两篇——《国殇》与《礼魂》。对应的神墟其实是"墓碑"，封着比九位旧神更古老的存在。

### 修行体系："心斋九境"

取自《庄子》，修行路径层层递进：

| 境 | 名称 | 能力范围 | 突破方式 |
|---|------|---------|---------|
| 一 | 观想 | 看见纹饰中的细节 | 入门 |
| 二 | 坐忘 | 听见死物的低语 | 吟诵《诗经》一篇 |
| 三 | 心斋 | 隔绝部分精神污染 | 直面弱小旧神而不疯 |
| 四 | 朝彻 | 用诗词制造短暂的规则区域 | 完成一次仪式性献祭 |
| 五 | 见独 | 与旧神对话不被覆写 | 直面一位完整旧神 |
| 六 | 无古今 | 穿越文物进入过去某一刻 | 承受三千年记忆注入 |
| 七 | 不死不生 | 短暂脱离物理规则 | 临死前的顿悟 |
| 八 | 撄宁 | 以言语重定义现实局部 | 亲手封印一位旧神 |
| 九 | 物化 | 成为新的"神" | 未知 |

**核心法则**：
- 每境突破需吟诵古辞 + 直面旧神而不疯
- **每句诗一生只能用一次**
- 吟诵必须"精准对应情境"（《国殇》必须在战场，《湘夫人》必须在水边）

### 五方势力

```
        九歌司（官方·封）
       /         \
      /           \
方士会（炼）——主角——招魂堂（通）
      \           /
       \         /
        旧神教（迎）
              |
      （幕后）海外同行
```

1. **九歌司**：官方正统。挂靠社科院，屈原秘密传承。主张"封"
2. **方士会**：民间野修。把旧神炼成丹药和金手指。主张"炼"
3. **招魂堂**：南派巫蛊。与旧神共生换取超凡能力。主张"通"
4. **旧神教**：被污染者组织。主张"迎"旧神降临
5. **海外同行**：日本阴阳寮、梵蒂冈圣座观测团、CIA蓝皮书计划各有诉求

### 金手指规则（林墨专属）

**能力**：
- 梦境预知（梦里去过的地方第二天能挖出东西）
- 与"梦中亡者"对话
- 天生免疫旧神污染（三千年一遇）

**代价——褪色**：
- 每睡一次/每用一次能力，现实中会有一个认识他的人"忘记他"
- 档案消失 → 熟人遗忘 → 镜子照不出影 → 母亲不记得他
- 用得越多越接近"不存在"

**褪色进度**：作为可视化的"倒计时"贯穿全书，每卷末更新。

### 时间规则

- 主要时间线：2026 年（当代）
- 闪回：林墨童年 + 灭族之夜（堂兄陆九相关）+ 20 年前林父失踪
- 超长回溯：屈原时代（三千年前封印之战）

### 叙事准则

1. **战斗即规则博弈**——不是拳脚对轰，是谁先掌握这次遭遇的规则
2. **闪回要克制**——只在关键选择时用，不泛滥
3. **褪色要具象**——每一次忘记都要有画面（母亲在精神病院叫错他的名字）
4. **旧神要神秘**——前 80 万字只出残影和信息，完整真身留到后期
5. **诗词有价**——每吟一句都要有代价，不能当快捷键
