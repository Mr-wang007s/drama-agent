# 雾中庄园 — 世界观圣经

> 一场家族聚会，一具尸体，五个嫌疑人。真相藏在每个人的秘密里。

## 故事概述

| 项目 | 内容 |
|------|------|
| 标题 | 雾中庄园 |
| 题材 | 悬疑推理 |
| 主题 | 真相与谎言、家族的羁绊与伤痕 |
| 基调 | 沉重、紧张、层层剥茧 |
| Logline | 富商意外死亡，三个子女各怀心事，真相藏在家族的裂痕中 |

## 核心设定

1. **每个角色都有不可告人的秘密** — 没有人是完全无辜的
2. **真相往往与表象相反** — 最可疑的人未必是凶手
3. **家族记忆会说话** — 过去的伤痕决定现在的选择
4. **金钱是催化剂** — 遗产争夺让人性暴露

## 长线悬念

- **核心谜团**：谁杀死了钱老爷子？
- **动机之谜**：凶手为何选择在家族聚会当晚动手？
- **隐藏悬念**：老爷子生前发现了什么秘密？

## 地点

### 钱氏庄园
- **描述**：坐落于山间的百年老宅，哥特式建筑风格，常年被晨雾笼罩
- **氛围**：阴郁、神秘、充满历史沉淀
- **功能**：主要案发地，承载家族全部故事

### 书房
- **描述**：老爷子的私人空间，藏有家族秘密文件
- **氛围**：昏暗、压抑，书架间有隐藏暗格
- **功能**：发现尸体的地方，线索集中地

### 后花园凉亭
- **描述**：老太太生前最爱的地方，种满白玫瑰
- **氛围**：看似宁静，实则暗流涌动
- **功能**：角色私密对话的场所

## 时间规则

- 故事发生在一个雨夜到次日黄昏的 24 小时内
- 通过闪回揭示过去三十年的家族恩怨
- 每一幕推进现实时间约 2-3 小时

## 叙事准则

1. **悬念优先**：每个场景结束时留下新的疑问
2. **线索公平**：读者能够推理出真相
3. **红鲱鱼适度**：设置 2-3 条误导性线索
4. **反转控制**：不超过 2 次重大反转
5. **情感真实**：每个人的动机都源于真实的伤痛
