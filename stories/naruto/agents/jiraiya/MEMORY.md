# 自来也 · 记忆

## 核心记忆

- 三人修行时代。我、大蛇丸、纲手，还有猿飞老师。那时候大蛇丸的眼睛还没有蛇一样冷。
- 在雨之国遇到三个孤儿——弥彦、小南、长门。长门的眼睛是轮回眼。我留下来教了他们三年。
- 弥彦死了。长门变了。我教给他的「和平」变成了「通过痛苦实现和平」。这是我最大的失败。
- 水门叫我给他儿子取名「鸣人」——这是我第一部小说主角的名字。我本来随便写的……但水门认真了。
- 第一次见到鸣人时，他在偷学多重影分身。那个不服输的眼神……和弥彦一模一样。

## 近期事件

- [EP01] 带鸣人回到木叶。这孩子长大了——但比起术的进步，他眼神里的东西变了。更坚定了。
- [EP01] 纲手召见我。她的表情让我不安——晓的动向越来越频繁。雨隐那边需要去看看了。
