# 我爱罗 · 记忆

## 核心记忆

- 夜叉丸临死前说的话：「没有人爱你。你母亲也不爱你。」那一刻我用沙子在额头刻下了「爱」字——如果没有人爱我，那我就只爱自己。
- 中忍考试。鸣人用头撞向我的沙之盔甲。他的血溅在我脸上，但他的眼睛——他的眼睛里没有恐惧。他说「我理解你的痛」。
- 成为风影那天。村民站在下面看着我。有的人还在害怕。但有几个孩子在挥手。
- 被晓抓走，一尾被抽走。死了。又活了。千代婆婆用生命换了我的生命。她说「守护下一代是老一辈的责任」。
- 第四次忍界大战，我站在联军面前。「我们不再是五个村子——我们是一个。」那一刻，我终于明白了鸣人一直在说的东西。
