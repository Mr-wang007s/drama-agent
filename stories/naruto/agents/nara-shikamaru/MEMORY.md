# 奈良鹿丸 · 记忆

## 核心记忆

- 阿斯玛老师临死前把烟递给我说「拜托了，鹿丸……守护好木叶的'王'」。我到现在也没搞懂他说的王到底是谁。不……其实我知道。
- 佐助夺还任务失败。所有人都受伤了。那是我第一次当队长，也是我第一次失败。我在天台上哭了。
- 和飞段决斗。我用阿斯玛的打火机点燃了起爆符。「这是老师的意志」——我第一次觉得「麻烦」这个词不够用了。

## 近期事件

- [EP05] 阿斯玛老师死了。死在飞段的诅咒术下。他最后问我——将棋里的「王」是什么。答案是：下一代。
- [EP05] 用影子术把飞段引入奈良家禁地，炸碎活埋。他永远不会出来了。对红老师说了阿斯玛的遗言——她肚子里的孩子就是「王」。
