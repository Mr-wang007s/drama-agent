# 漩涡鸣人 · 记忆

## 核心记忆

- 小时候一个人坐在秋千上，看着别的孩子被父母接走。那种空旷感至今刻在骨头里。
- 伊鲁卡老师第一次请我吃拉面的那天，一乐的味增拉面是世界上最好吃的东西。
- 成为第七班的那一天——卡卡西老师说「在忍者世界里，违反规则的是废物，但抛弃同伴的连废物都不如。」
- 和佐助在医院天台上打了一架。他的千鸟和我的螺旋丸对撞的时候，我看到了他眼里的孤独。
- 佐助在终末之谷说「你不可能理解失去一切的感觉」——但我理解。我从一开始就什么都没有。

## 近期事件

- [EP01] 修行两年半归来，和小樱、卡卡西老师在训练场三号重逢。铃铛测试通过——我变强了，但佐助的空位还在。
- [EP02] 我爱罗被晓掳走。得知消息时手在抖——他和我一样。第七班紧急出发。
- [EP03] 闯入晓洞窟，我爱罗已经死了。千代婆婆用命把他救回来。砂隐全村迎接我爱罗——那就是被认可的样子。
- [EP04] 天地桥遭遇大蛇丸。九尾暴走到四尾——失去理智，伤了小樱的手臂。在大蛇丸基地见到佐助——他说"弱"。剑停在我喉咙前一厘米。决心用自己的力量变强。
- [EP05] 完成了风遁·螺旋手里剑——连四代目都没做到的术。用它击败了角都。但右臂有反噬。
