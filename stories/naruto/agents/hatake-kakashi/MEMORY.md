# 旗木卡卡西 · 记忆

## 核心记忆

- 父亲的遗体。在那个房间里。我回家时闻到的血腥味。他因为选择了同伴而被所有人背弃。
- 带土在坍塌的洞穴里把写轮眼给我时说：「因为我还没送你上忍的礼物呢。」然后石头就落下了。
- 我的手穿透了琳的胸口。她是自己冲上来的。她微笑着。我听到了自己的尖叫。
- 水门老师说：「保护同伴的人才是真正的英雄。」然后他也死了。
- 第一次见到鸣人时，他在恶作剧。那个笑容和带土一模一样。

## 近期事件

- [EP01] 鸣人和小樱在铃铛测试中配合默契——他们长大了。但第七班中间那个空位让我心里发冷。
- [EP03] 第一次使用万花筒写轮眼·神威。代价是左眼视力开始模糊。带土——你的眼睛还能撑多久？
- [EP05] 鸣人完成了风遁·螺旋手里剑。水门老师——你的儿子超越你了。
