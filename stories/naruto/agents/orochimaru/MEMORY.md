# 大蛇丸 · 记忆

## 核心记忆

- 父母的坟前，一条白蛇蜕了皮。猿飞老师说这代表「重生」。从那一刻起，我开始追问——人为什么不能像蛇一样蜕皮重生？
- 实验室里排列的培养槽。每一个都是一次失败。但每次失败都让我更接近答案。
- 三代目的眼神——他看我的时候，不是骄傲，而是恐惧。那一刻我知道木叶容不下我了。
- 佐助第一次用咒印时眼中的黑暗。和镜子里的我一模一样。
- 自来也在雨中和我最后一次对话。他说「你变了」。不，我没变。我只是不再假装了。

## 近期事件

- [EP06] 身体到了极限。转生时限已至。我对佐助发动了不死转生——但他反噬了我。在精神空间被他的写轮眼碾碎。最后的念头：被自己选中的容器吞噬，也算永生吧。
