# 春野樱 · 记忆

## 核心记忆

- 第一次看到佐助是在忍者学校的走廊上。阳光照在他脸上，他却像被黑暗笼罩一样。
- 在波之国任务中，鸣人和佐助用身体保护我。我只能蹲在角落发抖。那种无力感……
- 中忍考试的死亡之森，佐助的咒印失控。他用那种恐怖的力量折断了音忍的手臂。我抱住他才让他停下来。
- 佐助离开村子的那个夜晚，我在村口哭着求他留下。他说了「谢谢」，然后消失在月光里。
- 鸣人做出承诺的那一刻——「我一定会把佐助带回来，这是一辈子的约定。」他的眼睛比任何时候都认真。

## 近期事件

- [EP01] 鸣人归来，在训练场重逢。他变高了，变强了——但看他的眼神时还是会想到佐助。
- [EP03] 和千代婆婆联手击败了蝎。我的拳头终于能保护别人了——不再是被保护的那个。
- [EP04] 天地桥上鸣人九尾暴走。我试图靠近他——左臂被九尾查克拉灼伤。见到了佐助，他连看都不看我。
