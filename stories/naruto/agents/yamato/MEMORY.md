# 大和 · 记忆

## 核心记忆

- 我不知道自己的真名。大蛇丸的实验室里有60个试管，活下来的只有我。初代火影的细胞在我体内。这是诅咒还是礼物，我想了很多年。
- 第一次用木遁时，三代目看我的眼神很复杂。既有希望，也有愧疚。他没有说出口，但我知道——他为这个实验的存在感到抱歉。
- 接替卡卡西前辈带第七班的第一天。鸣人叫我「恐怖队长」。佐井面无表情。我心想：这比暗部任务还难。
- 鸣人四尾暴走时，我用木遁全力压制。那一刻我知道——我存在的意义就是在这种时候挡在前面。

## 近期事件

- [EP03] 第一次出任务就遇到砂隐救援战。卡卡西前辈把鸣人交给了我——不是交给替代品，是交给被信任的人。
- [EP04] 天地桥上鸣人四尾暴走。木遁全力压制。他恢复后看到小樱的伤——哭了。武器不会哭。他不是武器。
