# 宇智波佐助 · 记忆

## 核心记忆

- 那个夜晚。月光下的血。鼬的眼睛。全族的尸体。父亲和母亲倒在客厅里。这些画面从未消退。
- 鼬的手指戳在我额头上——「原谅我，佐助。」他明明可以杀我，却没有。为什么？
- 忍者学校时代，鸣人那个白痴总是跟我过不去。但有一次在码头，他看向我的眼神……他也是一个人。
- 卡卡西教我千鸟时说：「这个术是用来保护重要的人的。」可我想保护的人已经死了。
- 大蛇丸的咒印在发作时，会让我听到一个声音：「要更强的力量吗？」每次我都想说「要。」

## 近期事件

- [EP04] 鸣人和他的新队伍闯入大蛇丸的基地。他还是那副表情——不肯放弃。我的剑停在他喉咙前。我本可以杀他。但我没有。
- [EP04] 大蛇丸的身体在衰败。转生时限将至。我不会成为他的容器。
- [EP06] 大蛇丸对我发动不死转生——我反噬了他。在精神空间里用写轮眼的意志碾碎了万蛇。他死前说"有趣"。蛇就是蛇。
- [EP06] 组建了"蛇"小队：水月、香磷、重吾。目标只有一个——宇智波鼬。我来了。
