# EP04 — 天地桥：四尾的咆哮

**时间线**：疾风传·天地桥篇（第七班前往天地桥 → 佐井暗线 → 九尾四尾暴走 → 与佐助重逢）
**参演角色**：鸣人(A)、小樱(A)、卡卡西(A)、佐助(A)、大蛇丸(A)、佐井(B)、大和(B)、药师兜(B)
**时长**：约三天（行程 + 天地桥接头 + 大蛇丸基地突入）
**技能**：novel（小说格式）

## Logline

第七班在天地桥遭遇大蛇丸——鸣人九尾暴走至四尾状态几乎失控；佐井背负团藏密令在暗线中觉醒；最终众人闯入大蛇丸基地与佐助重逢——但佐助的冰冷远超所有人的想象。

## 核心冲突

- 外：天地桥接头 → 大蛇丸现身 → 九尾暴走 → 大蛇丸基地潜入
- 内A：鸣人的九尾失控——力量与代价的临界点
- 内B：佐井从"根的工具"到"有感情的人"的觉醒
- 暗：团藏的双重密令——佐井的真正任务
- 底：佐助与鸣人重逢——"你还是这么弱，鸣人"
