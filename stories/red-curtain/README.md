# 红幕剧场 — 样板故事

> 这是 DramaAgent 的第一个完整样板故事，展示了引擎的核心能力。

## 故事简介

**题材**：都市悬疑 / 群像成长

**一句话故事**：一家濒临关闭的小剧场，成了三个年轻人追查旧案与自我重建的秘密入口。

**长期主题**：身份如何在表演、谎言与选择中被不断重写。

## 角色

| 角色 | 原型 | 说明 |
|------|------|------|
| **林七** | 舞台监督 / 不可靠见证者 | 保存了火场旧录像的守秘者 |
| **苏遥** | 回归主演 / 关系搅动者 | 五年后回归，带着不可告人的目的 |
| **高鸣** | 调查记者 / 外部压力源 | 受匿名委托而来的搅局者 |

## 已完成集数

- **EP01 红幕再开**：剧场重开，三人相遇
- **EP02 影价**：高鸣向林七提出交换条件
- **EP03 碎镜**：投影系统被入侵，旧日画面重现

## 如何使用这个样板

1. **学习 SOUL v4.0 格式**
   参考 `agents/` 下的角色定义（已升级为 v4.0 格式）

2. **理解世界状态管理**
   参考 `world/state.json` 的结构

3. **创建自己的故事**
   ```bash
   # 从 master 分支创建新故事分支
   git checkout master
   git checkout -b story/my-story

   # 使用预设初始化
   npm run drama -- init --preset mystery

   # 或从种子文件初始化
   npm run drama -- init --from my-story-seed.yaml
   ```

## 版本说明
